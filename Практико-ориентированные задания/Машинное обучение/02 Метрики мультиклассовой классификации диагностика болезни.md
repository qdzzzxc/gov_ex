# Задача 2. Метрики мультиклассовой классификации (диагностика болезни).

## Условие
Дана матрица результатов классификации в задачах диагностики болезни, где классы 1 и 2 представляют различные варианты вида диагностируемой болезни, а класс 0 — отсутствие болезни.

| Расчётный класс \ Эмпирический | 0 | 1 | 2 |
|---|---|---|---|
| 0 | 111 | 3 | 5 |
| 1 | 2 | 123 | 17 |
| 2 | 19 | 0 | 20 |

**Требуется:**
- рассчитать метрики **полноты (recall)** и **точности (precision)**, а также **F-меру** по каждому классу в отдельности;
- найти **средние** (macro) и **средневзвешенные** (weighted) значения метрик;
- обосновать выбор наиболее релевантной метрики, учитывая специфику задачи и несбалансированность классов;
- предложить основные подходы для совершенствования классификации.

## Решение

### Программа минимум

```python
import numpy as np
C = np.array([[111,3,5], [2,123,17], [19,0,20]])  # строки: прогноз, столбцы: факт
rows = []
for k in range(3):
    tp = C[k,k]; fp = C[k].sum() - tp; fn = C[:,k].sum() - tp
    p, r = tp/(tp+fp), tp/(tp+fn)                 # precision и recall
    rows.append([p, r, 2*p*r/(p+r)]); print(k, rows[-1])
print("macro:", np.mean(rows, axis=0))             # среднее по классам
print("weighted:", np.average(rows, axis=0, weights=C.sum(axis=0))) # по support
```

### Полное решение

### Идея
Для каждого класса $k$:
$$\mathrm{TP}_k = C_{kk},\ \mathrm{FP}_k = \sum_{j\ne k} C_{kj},\ \mathrm{FN}_k = \sum_{i \ne k} C_{ik}.$$
$$\mathrm{precision}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FP}_k},\quad \mathrm{recall}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FN}_k},\quad F_1 = \frac{2 P R}{P+R}.$$

В таблице условия строки = расчётный класс, столбцы = эмпирический.

### Расчёт по классам

| Класс | TP  | FP | FN | Precision | Recall | $F_1$  | Support |
|-------|-----|----|----|-----------|--------|--------|---------|
| 0     | 111 | 8  | 21 | 0,9328    | 0,8409 | 0,8845 | 132     |
| 1     | 123 | 19 | 3  | 0,8662    | 0,9762 | 0,9179 | 126     |
| 2     | 20  | 19 | 22 | 0,5128    | 0,4762 | 0,4938 | 42      |

Accuracy: $\mathrm{tr}(C) / n = (111+123+20)/300 = 254/300 \approx 0{,}847$.

### Сводные метрики

|              | Precision | Recall | $F_1$  |
|--------------|-----------|--------|--------|
| **macro**    | 0,7706    | 0,7644 | 0,7654 |
| **weighted** | 0,8460    | 0,8467 | 0,8438 |

- **macro** — простое среднее по классам, не учитывает разную поддержку.
- **weighted** — взвешено по support.

### Код

```python
import numpy as np
C = np.array([[111, 3, 5],
              [2, 123, 17],
              [19, 0, 20]])
for k in range(3):
    TP = C[k,k]
    FP = C[k,:].sum() - TP
    FN = C[:,k].sum() - TP
    p, r = TP/(TP+FP), TP/(TP+FN)
    f1 = 2*p*r/(p+r)
    print(f"class {k}: P={p:.3f}, R={r:.3f}, F1={f1:.3f}")
```

### Обоснование выбора метрики
**Контекст** — медицинская диагностика, классы несбалансированы (132 / 126 / 42).
- **Accuracy** ($\approx 0{,}85$) обманчиво высока: тратится «бюджет» на доминирующие классы, плохая работа по классу 2 ($F_1 = 0{,}49$) теряется в среднем.
- В медицине цена ошибок асимметрична: пропущенный больной (FN) опаснее ложной тревоги (FP), поэтому **recall важнее precision** — особенно для классов 1 и 2 (виды болезни).
- Для итоговой оценки на несбалансированной задаче — **macro-$F_1$** ($\approx 0{,}77$): даёт равный вес всем классам и не маскирует слабое распознавание редкого класса 2.
- Если стратегически важнее общая ошибка с поправкой на распространённость — **weighted-$F_1$** ($\approx 0{,}84$).

Главная метрика — **macro-$F_1$**; в дополнение приводят **recall класса 2** (как самого «больного»).

### Подходы к улучшению
1. **Балансировка данных** — oversample класса 2 (SMOTE), undersample класса 0; альтернатива — class weights в функции потерь.
2. **Изменение порога** для классов 1 и 2 — снизить порог принятия в эти классы, увеличив recall ценой precision.
3. **Признаки** — добавить новые информативные (анамнез, лабораторные показатели).
4. **Иерархическая классификация**: сначала «болен / здоров», затем «вид 1 / вид 2» — отдельная модель для дифдиагностики.
5. **Cost-sensitive learning** — явно задать матрицу штрафов, в которой пропуск пациента дорог.
6. **Калибровка вероятностей** (Platt, isotonic) и оценка по $\mathrm{ROC}$-$\mathrm{AUC}$ / $\mathrm{PR}$-$\mathrm{AUC}$ дополнительно к точечным метрикам.
