# Задача 1. Регрессия $Y$ от $X$ по социо-экономическим наблюдениям.

## Условие
Дан набор данных по социо-экономическим наблюдениям, состоящий из двух переменных:

| $Y$ | $X$ | $Y$ | $X$ |
|---|---|---|---|
| 622{,}9 | 4{,}9 | 876{,}8 | 4{,}7 |
| 658 | 5{,}2 | 900 | 4{,}6 |
| 700{,}4 | 5{,}5 | 951{,}4 | 5 |
| 740{,}6 | 5{,}6 | 1007{,}9 | 5{,}4 |
| 774{,}4 | 5{,}6 | 1004{,}8 | 4{,}2 |
| 816{,}2 | 5{,}3 | 1010{,}8 | 4{,}2 |
| 853{,}5 | 5 | 1056{,}2 | 4{,}6 |

**Требуется:** построить предиктивную модель, предсказывающую значение переменной $Y$ в зависимости от значения переменной $X$. Предложить и обосновать выбор одного из методов регрессии, метод оптимизации параметров исходя из размерности задачи. Визуализируйте метод регрессии на графике.

## Решение

### Программа минимум

```python
import numpy as np, matplotlib.pyplot as plt
X = np.array([4.9,5.2,5.5,5.6,5.6,5.3,5.0,4.7,4.6,5.0,5.4,4.2,4.2,4.6])
Y = np.array([622.9,658,700.4,740.6,774.4,816.2,853.5,876.8,900,951.4,1007.9,1004.8,1010.8,1056.2])
a1 = np.cov(X, Y, ddof=1)[0, 1] / np.var(X, ddof=1)  # МНК-наклон
a0 = Y.mean() - a1*X.mean()                          # свободный член
plt.scatter(X, Y); plt.plot(X, a0 + a1*X)             # точки и прямая
print(a0, a1)
```

### Полное решение

### Идея и обоснование выбора метода
- $n = 14$ — выборка очень маленькая, признак $X$ один. При таких размерностях сложные модели (нейросети, бустинг) переобучатся; ансамбли и регуляризация (Ridge/Lasso) тоже избыточны.
- Естественный выбор — **линейная регрессия** ($Y = a_0 + a_1 X$). Параметры — по **МНК в закрытой форме** (а не градиент): для одного признака решение известно аналитически:
$$\hat a_1 = \frac{\mathrm{cov}(X,Y)}{\mathrm{var}(X)},\quad \hat a_0 = \bar Y - \hat a_1 \bar X.$$
Никакой итерационной оптимизации при $n=14$ не нужно.

### Код

```python
import numpy as np, matplotlib.pyplot as plt

Y = np.array([622.9,658,700.4,740.6,774.4,816.2,853.5,
              876.8,900,951.4,1007.9,1004.8,1010.8,1056.2])
X = np.array([4.9,5.2,5.5,5.6,5.6,5.3,5.0,4.7,4.6,5.0,5.4,4.2,4.2,4.6])
n = len(Y)

a1 = np.cov(X, Y, ddof=1)[0,1] / np.var(X, ddof=1)
a0 = Y.mean() - a1 * X.mean()
yhat = a0 + a1*X
R2 = 1 - ((Y - yhat)**2).sum() / ((Y - Y.mean())**2).sum()
print(f"Y = {a0:.2f} + {a1:.2f}·X,  R² = {R2:.4f}")

xx = np.linspace(X.min()-0.2, X.max()+0.2, 100)
plt.scatter(X, Y); plt.plot(xx, a0 + a1*xx, 'r-')
plt.xlabel("X"); plt.ylabel("Y"); plt.title(f"Y = {a0:.0f} + {a1:.0f}·X,  R²={R2:.2f}")
plt.show()
```

### Прогон
```
n = 14
corr(X, Y) = −0.58

Y = 1715.83 − 172.60·X
R² = 0.34
```

### Ответ
$\hat Y = 1715{,}8 - 172{,}6 \cdot X$, коэффициент детерминации $R^2 \approx 0{,}34$.

### Замечания
- Связь **обратная** (коэффициент при $X$ отрицательный): чем выше $X$, тем меньше $Y$.
- $R^2 = 0{,}34$ — линейная модель объясняет лишь треть дисперсии $Y$. Это значит: признак $X$ сам по себе слабо предсказывает $Y$, нужна добавочная информация (другие признаки).
- Полиномиальная регрессия 2-й степени даёт $R^2 = 0{,}36$ — несущественное улучшение, но добавляются 2 параметра — переобучение. Линейная модель предпочтительна по бритве Оккама.
- Для оценки качества стоило бы добавить кросс-валидацию (например, leave-one-out), но при таком маленьком $n$ доверительный интервал по $R^2$ всё равно широкий.
