# Задача 1. Метод простой итерации.

## Условие
Используя метод простой итерации, решите для интервала $[-10;\ 10]$ уравнение $\ln(x) - x + 2 = 0$ с точностью до $0{,}001$.

## Решение

### Программа минимум

```python
import numpy as np
x, eps = 3.0, 1e-3                         # старт около корня, точность
for _ in range(100):
    xn = np.log(x) + 2                      # x = ln(x) + 2
    if abs(xn - x) < eps: break             # критерий остановки
    x = xn
print(xn)
```

### Полное решение

### Идея
Из $f(x) = \ln x - x + 2 = 0$ выражаем $x = \ln x + 2$, итерируем $x_{k+1} = g(x_k) = \ln x_k + 2$. Условие сходимости — $|g'(x)| = |1/x| < 1$, т.е. $x > 1$. Корень в окрестности $x \approx 3$.

ОДЗ: $x > 0$, поэтому фактически работаем на $(0;\,10]$.

### Код

```python
import numpy as np
g = lambda x: np.log(x) + 2

x, eps = 3.0, 1e-3
for k in range(100):
    x_new = g(x)
    if abs(x_new - x) < eps:
        x = x_new; break
    x = x_new
print(x, k+1)
```

### Прогон
```
x* ≈ 3.146037,  итераций: 6,  невязка f(x*) ≈ 1.1·10⁻⁴
```

### Ответ
$x^* \approx 3{,}146$.

### Замечания
- $|g'(3)| = 1/3 < 1$ — сжатие, теорема о неподвижной точке гарантирует сходимость.
- Выбор стартовой точки $x_0 = 3$ — внутри области сходимости. С $x_0 < 1$ итерации могут разойтись.
- Скорость сходимости — линейная с константой $1/3$.
