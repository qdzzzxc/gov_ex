# Задача 5. Псевдоградиентный поиск.

## Условие
Используя метод псевдоградиентного поиска, выполните 5 итераций поиска весов $a$ в уравнении $y = a_0 + a_1 x$ с шагом $\mathrm{d}a = 0{,}1$, если известный $y = \{5,\ 3,\ 8,\ 10,\ -4\}$, а $x = \{0,\ 2,\ 11,\ 6,\ -3\}$.
## Решение

### Идея
Минимизируется функционал MSE
$$L(a_0, a_1) = \frac{1}{n} \sum_{i=1}^n (y_i - a_0 - a_1 x_i)^2.$$
Градиент:
$$\frac{\partial L}{\partial a_0} = -\frac{2}{n}\sum (y_i - a_0 - a_1 x_i),\qquad \frac{\partial L}{\partial a_1} = -\frac{2}{n}\sum x_i(y_i - a_0 - a_1 x_i).$$

Псевдоградиентный поиск: вместо градиента используем его **знак**, шаг $\mathrm{d}a$ фиксированный:
$$a^{(k+1)}_j = a^{(k)}_j - \mathrm{d}a \cdot \mathrm{sign}\!\left(\frac{\partial L}{\partial a_j}\right).$$

Стартуем с $a^{(0)} = (0, 0)$, делаем 5 итераций.

|$k$|$a_0$|$a_1$|$\sum r$|$\sum x_i r_i$|$g_0$|$g_1$|$\mathrm{sign},g$|$a^{\text{нов}}$|
|---|---|---|---|---|---|---|---|---|
|1|0,0|0,0|22,0|166,0|$-8{,}80$|$-66{,}40$|$(-,-)$|$(0{,}1;\ 0{,}1)$|
|2|0,1|0,1|19,9|147,4|$-7{,}96$|$-58{,}96$|$(-,-)$|$(0{,}2;\ 0{,}2)$|
|3|0,2|0,2|17,8|128,8|$-7{,}12$|$-51{,}52$|$(-,-)$|$(0{,}3;\ 0{,}3)$|
|4|0,3|0,3|15,7|110,2|$-6{,}28$|$-44{,}08$|$(-,-)$|$(0{,}4;\ 0{,}4)$|
|5|0,4|0,4|13,6|91,6|$-5{,}44$|$-36{,}64$|$(-,-)$|$(0{,}5;\ 0{,}5)$|
**Финал:** $a_0 = 0{,}5$, $a_1 = 0{,}5$.
Проверка лосса в финальной точке: $\hat y = (0{,}5;\ 1{,}5;\ 6{,}0;\ 3{,}5;\ -1{,}0)$, $r = (4{,}5;\ 1{,}5;\ 2{,}0;\ 6{,}5;\ -3{,}0)$, $r^2 = (20{,}25;\ 2{,}25;\ 4{,}00;\ 42{,}25;\ 9{,}00)$, сумма $77{,}75$, среднее $L = 15{,}55$ ✓
### Код
```python
import numpy as np
y  = np.array([ 5,  3,  8, 10, -4], float)
xs = np.array([ 0,  2, 11,  6, -3], float)

def loss(a0, a1): return ((y - a0 - a1*xs)**2).mean()
def grad(a0, a1):
    r = y - a0 - a1*xs
    return -2*r.mean(), -2*(r*xs).mean()

a0, a1, da = 0.0, 0.0, 0.1
for k in range(1, 6):
    g0, g1 = grad(a0, a1)
    a0 -= da * np.sign(g0)
    a1 -= da * np.sign(g1)
    print(f"k={k}: a=({a0:+.2f},{a1:+.2f})  L={loss(a0,a1):.3f}")
```
### Ответ
После 5 итераций: $a_0 = 0{,}5$, $a_1 = 0{,}5$, $L \approx 15{,}55$.
