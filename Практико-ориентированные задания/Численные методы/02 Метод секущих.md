# Задача 2. Метод секущих.

## Условие
Используя метод секущих, решите для интервала $[-10;\ 10]$ уравнение $3\sin(x) + x - 2 = 0$ с точностью до $0{,}005$.

## Решение

### Программа минимум

```python
import numpy as np
f = lambda x: 3*np.sin(x) + x - 2
x0, x1, eps = 0.0, 1.0, 5e-3               # две точки, где есть корень
for _ in range(100):
    x2 = x1 - f(x1)*(x1-x0)/(f(x1)-f(x0))  # формула секущих
    if abs(x2 - x1) < eps: break
    x0, x1 = x1, x2
print(x2)
```

### Полное решение

### Идея
Метод секущих — итерация
$$x_{k+1} = x_k - f(x_k)\,\frac{x_k - x_{k-1}}{f(x_k) - f(x_{k-1})}.$$
Это аппроксимация Ньютона, в которой производная заменена конечной разностью. Сходимость — суперлинейная, порядок $\varphi = (1+\sqrt5)/2 \approx 1{,}618$.

Корень $f(x) = 3\sin x + x - 2$ ищем на промежутке, где $f$ меняет знак: $f(0) = -2$, $f(1) = 3\sin 1 + 1 - 2 \approx 1{,}52 > 0$ — корень в $(0;\,1)$.

### Код

```python
import numpy as np
f = lambda x: 3*np.sin(x) + x - 2

x0, x1, eps = 0.0, 1.0, 5e-3
for k in range(100):
    fx0, fx1 = f(x0), f(x1)
    x2 = x1 - fx1*(x1 - x0)/(fx1 - fx0)
    if abs(x2 - x1) < eps:
        x1 = x2; break
    x0, x1 = x1, x2
```

### Прогон
```
x* ≈ 0.517049,  итераций: 4,  невязка ≈ 4·10⁻⁷
```

### Ответ
$x^* \approx 0{,}517$.

### Замечания
- За 4 итерации точность улучшилась с $10^{-1}$ до $10^{-7}$ — суперлинейная сходимость в действии.
- Метод не требует вычисления производной (плюс по сравнению с Ньютоном), но требует двух стартовых точек.
- Опасность: если $f(x_k) \approx f(x_{k-1})$, делитель почти ноль — численная нестабильность.
