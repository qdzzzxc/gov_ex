# Задача 3. Метод дихотомии.

## Условие
Используя метод дихотомии, решите для интервала $[-10;\ 10]$ уравнение $\sin(x) + x^2 - 1 = 0$ с точностью до $0{,}0005$.

## Решение

### Программа минимум

```python
import numpy as np
f = lambda x: np.sin(x) + x*x - 1
a, b, eps = 0.0, 1.0, 5e-4                 # f(a)*f(b) < 0
while b - a > 2*eps:
    c = (a + b) / 2                         # середина отрезка
    if f(a)*f(c) <= 0: b = c                # оставляем половину со сменой знака
    else: a = c
print((a+b)/2)
```

### Полное решение

### Идея
Бисекция: дан отрезок $[a, b]$, на концах которого $f$ имеет разные знаки ⇒ по теореме Больцано-Коши корень внутри. На каждом шаге делим пополам и оставляем ту половину, где знак меняется.

$f(x) = \sin x + x^2 - 1$: $f(0) = -1 < 0$, $f(1) = \sin 1 \approx 0{,}84 > 0$ — корень в $(0; 1)$.

После $k$ шагов длина отрезка — $(b-a)/2^k$. Для точности $\varepsilon$ нужно $k \ge \log_2((b-a)/(2\varepsilon)) = \log_2(1/0{,}001) \approx 10$ шагов.

### Код

```python
import numpy as np
f = lambda x: np.sin(x) + x**2 - 1

a, b, eps = 0.0, 1.0, 5e-4
k = 0
while b - a > 2*eps:
    c = 0.5*(a + b)
    if f(a)*f(c) < 0: b = c
    else:             a = c
    k += 1
x = 0.5*(a + b)
```

### Прогон
```
x* ≈ 0.637207,  итераций: 10,  невязка ≈ 1·10⁻³
```

### Ответ
$x^* \approx 0{,}637$.

### Замечания
- Метод **гарантированно** сходится (если на концах разные знаки) и не требует производной.
- Скорость — линейная с коэффициентом $1/2$ (медленнее Ньютона/секущих).
- Стоп-критерий по длине отрезка: $b - a < 2\varepsilon$ гарантирует $|x^* - c| < \varepsilon$.
