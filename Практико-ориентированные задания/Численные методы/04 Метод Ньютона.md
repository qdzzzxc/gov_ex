# Задача 4. Метод Ньютона.

## Условие
Используя метод Ньютона, решите для интервала $[-10;\ 10]$ уравнение $2\ln(x) + 0{,}01 x^2 = 0$ с точностью до $0{,}002$.

## Решение

### Программа минимум

```python
import numpy as np
f = lambda x: 2*np.log(x) + 0.01*x*x
df = lambda x: 2/x + 0.02*x
x, eps = 0.5, 2e-3                         # старт в ОДЗ x > 0
for _ in range(100):
    xn = x - f(x)/df(x)                     # шаг Ньютона
    if abs(xn - x) < eps: break
    x = xn
print(xn)
```

### Полное решение

### Идея
Метод Ньютона: $x_{k+1} = x_k - f(x_k)/f'(x_k)$. Геометрически — пересечение касательной с осью $x$. Квадратичная сходимость: $|x_{k+1} - x^*| \le C |x_k - x^*|^2$.

$f(x) = 2\ln x + 0{,}01 x^2$, $f'(x) = 2/x + 0{,}02 x$. ОДЗ $x > 0$. На $(0; 10]$:
$f(0{,}5) = 2 \ln 0{,}5 + 0{,}0025 \approx -1{,}38 < 0$, $f(1) = 0 + 0{,}01 > 0$ — корень в $(0{,}5;\,1)$.

### Код

```python
import numpy as np
f  = lambda x: 2*np.log(x) + 0.01*x**2
df = lambda x: 2/x + 0.02*x

x, eps = 0.5, 2e-3
for k in range(100):
    x_new = x - f(x)/df(x)
    if abs(x_new - x) < eps:
        x = x_new; break
    x = x_new
```

### Прогон
```
x* ≈ 0.995061,  итераций: 4,  невязка ≈ 5·10⁻⁹
```

### Ответ
$x^* \approx 0{,}9951$ (близко к 1, но не строго равен — из-за слагаемого $0{,}01 x^2$).

### Замечания
- За 4 итерации невязка $|f(x)|$ упала до $\sim 10^{-9}$ — квадратичная сходимость на видном расстоянии от корня.
- Если стартовать с $x_0 \le 0$, метод неприменим (выход из ОДЗ).
- При $x \to 0^+$ $f' \to +\infty$, шаг $f/f'$ маленький — метод ведёт себя устойчиво.
