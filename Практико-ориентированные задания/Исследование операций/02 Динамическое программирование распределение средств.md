# Задача 2. Динамическое программирование: распределение средств между двумя предприятиями.

## Условие
Планируется работа двух предприятий на 3 года. Начальные ресурсы равны $s_0 = 3000$.

- Средства $x$, вложенные в первое предприятие в начале года, дают в конце года прибыль $f_1(x) = 0{,}7x$ и возвращаются в размере $\varphi_1(x) = 0{,}2x$.
- Средства $y$, вложенные во второе предприятие в начале года, дают в конце года прибыль $f_2(y) = 0{,}6y$ и возвращаются в размере $\varphi_2(y) = 0{,}4y$.

В конце года возвращённые средства заново перераспределяются между отраслями.

**Требуется:** методом динамического программирования найти оптимальный план распределения средств и максимальную прибыль за 3 года.

## Решение

### Идея (метод Беллмана, обратная индукция)
Состояние — текущий капитал $s$. Решение в начале года — $(x, y)$ с $x + y = s$, $x, y \ge 0$, где $x$ идёт на 1-е предприятие, $y$ — на 2-е.

За год получаем прибыль $\pi(x, y) = 0{,}7 x + 0{,}6 y$ и в конце года получаем «возврат» $R(x, y) = 0{,}2 x + 0{,}4 y$, который становится капиталом следующего года.

$F_k(s)$ — максимальная суммарная прибыль с года $k$ до года 3 при начальном капитале $s$:
$$F_k(s) = \max_{0 \le x \le s} \left\{ 0{,}7x + 0{,}6(s-x) + F_{k+1}\bigl(0{,}2x + 0{,}4(s-x)\bigr) \right\},\qquad F_4(s) \equiv 0.$$

Подставляя $y = s - x$:
$$F_k(s) = \max_{0 \le x \le s} \left\{ 0{,}6 s + 0{,}1 x + F_{k+1}(0{,}4 s - 0{,}2 x) \right\}.$$

### Год 3 (последний)
Возврат не нужен:
$$F_3(s) = \max_{x} \{0{,}6 s + 0{,}1 x\} = 0{,}7 s\quad (x^* = s,\ y^* = 0).$$

### Год 2
$$F_2(s) = \max_x\{0{,}6 s + 0{,}1 x + 0{,}7(0{,}4 s - 0{,}2 x)\} = \max_x\{0{,}88 s - 0{,}04 x\} = 0{,}88 s \quad (x^* = 0,\ y^* = s).$$

### Год 1
$$F_1(s) = \max_x\{0{,}6 s + 0{,}1 x + 0{,}88(0{,}4 s - 0{,}2 x)\} = \max_x\{0{,}952 s - 0{,}076 x\} = 0{,}952 s \quad (x^* = 0,\ y^* = s).$$

### Оптимальный план для $s_0 = 3000$

| Год | $s$  | $x$ (в 1-е) | $y$ (во 2-е) | Прибыль | Возврат |
|-----|------|-------------|--------------|---------|---------|
| 1   | 3000 | 0           | 3000         | 1800    | 1200    |
| 2   | 1200 | 0           | 1200         | 720     | 480     |
| 3   | 480  | 480         | 0            | 336     | —       |
| **Итого** |   |             |              | **2856** |        |

Совпадает с $F_1(3000) = 0{,}952 \cdot 3000 = 2856$.

### Ответ
**Максимальная суммарная прибыль за 3 года — $2856$.** План:
- годы 1, 2 — все средства во 2-е предприятие;
- год 3 — все средства в 1-е предприятие.

### Логика результата
- 1-е предприятие даёт большую разовую прибыль ($0{,}7$ против $0{,}6$), но возвращает мало ($0{,}2$ против $0{,}4$).
- На последнем году будущая «отдача» уже не нужна — берут более доходное 1-е.
- На предыдущих годах важнее «накопить» капитал для последующих, поэтому идут во 2-е (бо́льший возврат компенсирует меньшую прибыль).
- При длинном горизонте граница между «копить» и «снять» сдвигается ближе к концу.
