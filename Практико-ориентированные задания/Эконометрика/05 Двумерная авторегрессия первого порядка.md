# Задача 5. Двумерная авторегрессия первого порядка.

## Условие
Заданы следующие параметры модели двумерной авторегрессии первого порядка: $N = M = 500$, $M(x) = 0$, $D(x) = 1$, $r_i = 0{,}92$, $r_j = 0{,}84$.

Представьте программный код для генерации такого случайного поля и расчёта сечений автокорреляционной функции.

## Решение

### Программа минимум

```python
import numpy as np
N, M, ri, rj = 500, 500, .92, .84
s = np.sqrt((1-ri**2)*(1-rj**2))                 # D(x)=1
e = np.random.normal(0, s, (N, M)); x = np.zeros((N, M))
for i in range(N):
    for j in range(M):                            # двумерная AR(1)
        up = ri*x[i-1,j] if i else 0
        left = rj*x[i,j-1] if j else 0
        diag = ri*rj*x[i-1,j-1] if i and j else 0
        x[i,j] = up + left - diag + e[i,j]
print([np.corrcoef(x[:-k,:].ravel(), x[k:,:].ravel())[0,1] if k else 1 for k in range(6)])
```

### Полное решение

### Идея
Сепарабельная двумерная AR(1) (модель Хабиби):
$$x_{i,j} = r_i\, x_{i-1,j} + r_j\, x_{i,j-1} - r_i r_j\, x_{i-1,j-1} + \varepsilon_{i,j},\quad \varepsilon \sim \mathcal{N}(0, \sigma_\varepsilon^2).$$
АКФ распадается в произведение:
$$\rho(k, l) = r_i^{|k|}\, r_j^{|l|}.$$
Дисперсия:
$$\sigma_x^2 = \frac{\sigma_\varepsilon^2}{(1-r_i^2)(1-r_j^2)}.$$
Из $D(x) = 1$, $r_i = 0{,}92$, $r_j = 0{,}84$:
$$\sigma_\varepsilon^2 = (1-0{,}8464)(1-0{,}7056) \approx 0{,}0452,\quad \sigma_\varepsilon \approx 0{,}2126.$$

### Код

```python
import numpy as np

N, Mw, ri, rj = 500, 500, 0.92, 0.84
sigma_e = np.sqrt((1 - ri**2) * (1 - rj**2))

rng = np.random.default_rng(17)
eps = rng.normal(0, sigma_e, (N, Mw))
x = np.zeros((N, Mw))
x[0,0] = eps[0,0]
for i in range(1, N): x[i,0] = ri*x[i-1,0] + eps[i,0]
for j in range(1, Mw): x[0,j] = rj*x[0,j-1] + eps[0,j]
for i in range(1, N):
    for j in range(1, Mw):
        x[i,j] = ri*x[i-1,j] + rj*x[i,j-1] - ri*rj*x[i-1,j-1] + eps[i,j]

# сечения АКФ
def acf_axis(x, axis, max_k):
    out = []; xc = x - x.mean(); var = xc.var()
    for k in range(max_k+1):
        if k == 0: prod = (xc**2).mean()
        elif axis==0: prod = (xc[:-k] * xc[k:]).mean()
        else: prod = (xc[:, :-k] * xc[:, k:]).mean()
        out.append(prod/var)
    return out
```

Эмпирические сечения практически совпадают с теоретическими $r_i^{|k|}$ и $r_j^{|l|}$.

### Замечания
- Граничные строка/столбец инициализируются одномерным AR(1), чтобы избежать «холодного старта» с нулей.
- По сравнению с двумерной AR общего вида сепарабельная даёт быстрый рекуррентный шаг и аналитически известные АКФ.
