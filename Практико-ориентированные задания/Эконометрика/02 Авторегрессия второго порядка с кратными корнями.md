# Задача 2. Авторегрессия второго порядка с кратными корнями.

## Условие
Заданы следующие параметры модели авторегрессии второго порядка с кратными корнями характеристических уравнений: $N = 8000$, $M(x) = 1$, $D(\varepsilon) = 0{,}1$, $r = 0{,}75$.

Представьте программный код для генерации такого случайного процесса и расчёта его автокорреляционной функции на языке программирования по вашему выбору.

## Решение

### Программа минимум

```python
import numpy as np
N, mu, De, r = 8000, 1.0, 0.1, 0.75
p1, p2 = 2*r, -r*r; c = mu*(1-p1-p2)             # кратный корень
x = np.full(N, mu); e = np.random.normal(0, np.sqrt(De), N)
for t in range(2, N): x[t] = c + p1*x[t-1] + p2*x[t-2] + e[t]
xc = x - x.mean()
acf = [np.corrcoef(xc[:-k], xc[k:])[0,1] if k else 1 for k in range(21)]  # АКФ
print(acf[:6])
```

### Полное решение

### Идея
AR(2): $x_t = c + \varphi_1 x_{t-1} + \varphi_2 x_{t-2} + \varepsilon_t$. Характеристическое уравнение
$$1 - \varphi_1 z - \varphi_2 z^2 = 0.$$
Условие **кратных корней** даёт связь $\varphi_1 = 2\lambda,\ \varphi_2 = -\lambda^2$ (с одним повторяющимся корнем, обратным $\lambda$).

В условии $r = \lambda = 0{,}75$ ⇒ $\varphi_1 = 1{,}5,\ \varphi_2 = -0{,}5625$. Среднее $\mu = c/(1-\varphi_1-\varphi_2) = 1$ ⇒ $c = 1 \cdot (1 - 1{,}5 + 0{,}5625) = 0{,}0625$. Дисперсия шума задана: $D(\varepsilon) = 0{,}1$.

АКФ для AR(2) с кратными корнями имеет вид
$$\rho(k) = (1 + b k) \lambda^{|k|},$$
монотонно затухает (без колебаний), что подтвердится на прогоне.

### Код

```python
import numpy as np

N, M, D_e, r = 8000, 1.0, 0.1, 0.75
lam = r
phi1, phi2 = 2*lam, -lam**2
c = M * (1 - phi1 - phi2)
sigma_e = np.sqrt(D_e)

rng = np.random.default_rng(7)
x = np.zeros(N); x[:2] = M
e = rng.normal(0, sigma_e, N)
for t in range(2, N):
    x[t] = c + phi1*x[t-1] + phi2*x[t-2] + e[t]
x = x[200:]   # burn-in

# АКФ через FFT
xc = x - x.mean()
n = len(xc); F = np.fft.fft(xc, 2*n)
acov = np.fft.ifft(F * np.conj(F))[:n].real / n
acf = acov / acov[0]
```

### Прогон
```
λ=0.75, φ1=1.5, φ2=-0.5625, c=0.0625
empir mean ≈ 0.92,  var ≈ 1.94

rho(0)  = +1.0000
rho(1)  = +0.9622
rho(2)  = +0.8842
rho(3)  = +0.7881
rho(4)  = +0.6870
rho(5)  = +0.5893
rho(10) = +0.2284
rho(20) = +0.0134
```

АКФ положительна и монотонно убывает — характерная картина для пары кратных вещественных корней.

### Замечания
- Для **различных вещественных** корней АКФ — сумма двух геометрий $\rho(k) = A \lambda_1^k + B \lambda_2^k$.
- Для **комплексно сопряжённых** корней АКФ затухает с осцилляциями — здесь их нет.
- Эмпирическая дисперсия $\approx 1{,}94$ соответствует теоретической формуле $\sigma_x^2 = \frac{(1+\lambda^2)\sigma_\varepsilon^2}{(1-\lambda^2)^3} = \frac{1{,}5625 \cdot 0{,}1}{0{,}4375^3} \approx 1{,}87$.
