# Задача 2. Авторегрессия второго порядка с кратными корнями.

## Условие
Заданы следующие параметры модели авторегрессии второго порядка с кратными корнями характеристических уравнений: $N = 8000$, $M(x) = 1$, $D(\varepsilon) = 0{,}1$, $r = 0{,}75$.

Представьте программный код для генерации такого случайного процесса и расчёта его автокорреляционной функции на языке программирования по вашему выбору.

## Решение
### Идея
AR(2): $x_t = c + \varphi_1 x_{t-1} + \varphi_2 x_{t-2} + \varepsilon_t$. Характеристическое уравнение
$$1 - \varphi_1 z - \varphi_2 z^2 = 0.$$
Условие **кратных корней** даёт связь $\varphi_1 = 2\lambda,\ \varphi_2 = -\lambda^2$ (с одним повторяющимся корнем, обратным $\lambda$).

В условии $r = \lambda = 0{,}75$ ⇒ $\varphi_1 = 1{,}5,\ \varphi_2 = -0{,}5625$. Среднее $\mu = c/(1-\varphi_1-\varphi_2) = 1$ ⇒ $c = 1 \cdot (1 - 1{,}5 + 0{,}5625) = 0{,}0625$. Дисперсия шума задана: $D(\varepsilon) = 0{,}1$.
### Код

```python
import numpy as np

N, M, D_e, r = 8000, 1.0, 0.1, 0.75
lam = r
phi1, phi2 = 2*lam, -lam**2
c = M * (1 - phi1 - phi2)
sigma_e = np.sqrt(D_e)

rng = np.random.default_rng(7)
x = np.zeros(N); x[:2] = M
e = rng.normal(0, sigma_e, N)
for t in range(2, N):
    x[t] = c + phi1*x[t-1] + phi2*x[t-2] + e[t]
x = x[200:]   # burn-in

xc = x - x.mean()
var = np.mean(xc**2)

def acf_at(k):
	if k == 0: return 1.0
	return np.mean(xc[: -k] * xc[k:]) / var

acf = [acf_at(k) for k in range(50)]
```
АКФ положительна и монотонно убывает — характерная картина для пары кратных вещественных корней.
