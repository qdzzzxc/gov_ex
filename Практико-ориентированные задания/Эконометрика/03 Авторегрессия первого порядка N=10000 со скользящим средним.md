# Задача 3. Авторегрессия первого порядка со скользящим средним ($N = 10000$).

## Условие
Заданы следующие параметры модели авторегрессии первого порядка: $N = 10000$, $M(x) = -2$, $D(x) = 3$, $r = 0{,}9$.

Представьте программный код для генерации такого случайного процесса авторегрессии скользящего среднего в окне $M = 5$ и построения графика на экране на языке программирования по вашему выбору.

## Решение

### Программа минимум

```python
import numpy as np, matplotlib.pyplot as plt
N, mu, D, r, w = 10000, -2.0, 3.0, 0.9, 5
x = np.full(N, mu); e = np.random.normal(0, np.sqrt(D*(1-r*r)), N)
for t in range(1, N): x[t] = mu*(1-r) + r*x[t-1] + e[t]  # AR(1)
ma = np.convolve(x, np.ones(w)/w, "valid")               # скользящее среднее
plt.plot(x, alpha=.4); plt.plot(range(w-1, N), ma); plt.show()
```

### Полное решение

### Идея
Модель AR(1) (как в задаче 1), но с другими параметрами и пост-обработкой скользящим средним. Параметры:
- $\varphi = r = 0{,}9$, $\sigma_\varepsilon^2 = (1-0{,}81) \cdot 3 = 0{,}57$ ⇒ $\sigma_\varepsilon \approx 0{,}755$;
- $c = M(1-\varphi) = -2 \cdot 0{,}1 = -0{,}2$.

Скользящее среднее в окне $M = 5$: $\bar x_t = \tfrac{1}{5}\sum_{k=0}^{4} x_{t+k}$ — фильтр низких частот, режет высокочастотные шумовые колебания, не сдвигает уровень.

### Код

```python
import numpy as np
import matplotlib.pyplot as plt

N, M, D, r, win = 10000, -2.0, 3.0, 0.9, 5
phi = r
c = M * (1 - phi)
sigma_e = np.sqrt(D * (1 - phi**2))

rng = np.random.default_rng(11)
x = np.zeros(N); x[0] = M
e = rng.normal(0, sigma_e, N)
for t in range(1, N):
    x[t] = c + phi * x[t-1] + e[t]

ma = np.convolve(x, np.ones(win)/win, mode="valid")

plt.figure(figsize=(10,3))
plt.plot(x, lw=0.3, alpha=0.5, label="x_t")
plt.plot(np.arange(win-1, N), ma, lw=0.7, label=f"MA({win})")
plt.legend(); plt.tight_layout(); plt.show()
```

### Прогон
```
phi=0.9, c=-0.2, sigma_e=0.7550
x:        mean=-1.898, var=3.121
ma(M=5):  mean=-1.898, var=2.666
```

Сглаживание сохранило среднее (для стационарного процесса оно и должно сохраняться) и сократило дисперсию: для независимого шума MA уменьшил бы её в $5$ раз, но из-за высокой автокорреляции эффект мягче — лишь $\approx 0{,}85$.

### Замечания
- Для AR(1) теоретическая дисперсия после MA(M):
$$\mathrm{D}[\bar x_t] = \frac{\sigma_x^2}{M^2}\sum_{k,l=0}^{M-1} \varphi^{|k-l|}.$$
При $\varphi = 0{,}9$, $M=5$ получается $\approx 2{,}55$ — близко к эмпирике.
- Видно характерное «затушёвывание» высокочастотных колебаний и сохранение медленных трендов.
