# Задача 1. Авторегрессия первого порядка ($N = 10000$): построение графика.

## Условие
Заданы следующие параметры модели авторегрессии первого порядка: $N = 10000$, $M(x) = 0$, $D(x) = 1$, $r = 0{,}95$.

Представьте программный код для генерации такого случайного процесса и построения графика на экране на языке программирования по вашему выбору.

## Решение

### Программа минимум

```python
import numpy as np, matplotlib.pyplot as plt
N, r = 10000, 0.95
sigma = np.sqrt(1 - r*r)                         # D(x)=1 => D(eps)=1-r^2
x = np.zeros(N); e = np.random.normal(0, sigma, N)
for t in range(1, N): x[t] = r*x[t-1] + e[t]     # AR(1)
plt.plot(x); plt.show()
```

### Полное решение

### Идея
AR(1): $x_t = c + \varphi x_{t-1} + \varepsilon_t$, $\varepsilon_t \sim \mathcal{N}(0, \sigma_\varepsilon^2)$. Стационарность при $|\varphi| < 1$. Связи параметров:
$$\mathrm{E}[x_t] = \frac{c}{1-\varphi},\quad \mathrm{D}[x_t] = \frac{\sigma_\varepsilon^2}{1-\varphi^2},\quad \rho(1) = \varphi.$$

Из условия $r = \varphi = 0{,}95$, $M(x) = 0 \Rightarrow c = 0$, $D(x) = 1 \Rightarrow \sigma_\varepsilon^2 = 1 - 0{,}95^2 = 0{,}0975$, $\sigma_\varepsilon \approx 0{,}3122$.

### Код

```python
import numpy as np
import matplotlib.pyplot as plt

N, M, D, r = 10000, 0.0, 1.0, 0.95
phi = r
c = M * (1 - phi)
sigma_e = np.sqrt(D * (1 - phi**2))

rng = np.random.default_rng(42)
x = np.zeros(N); x[0] = M
e = rng.normal(0, sigma_e, N)
for t in range(1, N):
    x[t] = c + phi * x[t-1] + e[t]

plt.figure(figsize=(10,3))
plt.plot(x, lw=0.4)
plt.title(f"AR(1): N={N}, M={M}, D={D}, r={r}")
plt.xlabel("t"); plt.ylabel("x_t"); plt.tight_layout()
plt.show()
```

### Прогон
```
phi=0.95, c=0.0, sigma_e=0.3122
эмпирически: mean=-0.066, var=1.073   (целевые: 0 и 1)
```

### Замечания
- При $r=0{,}95$ процесс выглядит как «ползущая» траектория, локально плавная, но возвращающаяся к среднему за характерное время $\tau \approx 1/(1-\varphi) = 20$.
- Эмпирическая дисперсия может ощутимо отличаться от $1$ при единственной реализации — это нормально для процессов с близкой к единице авторегрессионной константой.
