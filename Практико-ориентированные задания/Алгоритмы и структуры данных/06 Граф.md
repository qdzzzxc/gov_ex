# Задача 6. Граф.

## Условие
Реализуйте граф на языке программирования по вашему выбору. Необходимо реализовать создание графа из списка либо матрицы смежности, вывод графа в виде списка и матрицы смежности, методы обхода графа.
## Решение
### Идея
Храним граф как словарь смежности `adj[u] = {соседи}`. Из этого строится матрица смежности $A$ размера $n \times n$ ($A_{ij} = 1$, если есть ребро $u_i \to u_j$). Поддержим оба способа задания (список и матрица), вывод в обе формы и обходы BFS / DFS.
### Код

```python
from collections import deque, defaultdict

class Graph:
    def __init__(self, directed=False):
        self.directed = directed
        self.adj = defaultdict(set)
        self.vertices = set()

    @classmethod
    def from_adj_list(cls, adj_list, directed=False):
        g = cls(directed)
        for u, neighbors in adj_list.items():
            g.add_vertex(u)
            for v in neighbors: g.add_edge(u, v)
        return g

    @classmethod
    def from_matrix(cls, vertices, M, directed=False):
        g = cls(directed)
        for v in vertices: g.add_vertex(v)
        for i, u in enumerate(vertices):
            for j, v in enumerate(vertices):
                if M[i][j]: g.add_edge(u, v)
        return g

    def add_vertex(self, u):
        self.vertices.add(u); _ = self.adj[u]

    def add_edge(self, u, v):
        self.add_vertex(u); self.add_vertex(v)
        self.adj[u].add(v)
        if not self.directed: self.adj[v].add(u)

    def to_adj_list(self):
        return {u: sorted(self.adj[u]) for u in sorted(self.vertices)}

    def to_matrix(self):
        verts = sorted(self.vertices)
        idx = {v: i for i, v in enumerate(verts)}
        n = len(verts); M = [[0]*n for _ in range(n)]
        for u in verts:
            for v in self.adj[u]: M[idx[u]][idx[v]] = 1
        return verts, M

    def bfs(self, s):
	    visited, order, q = {s}, [], deque([s])
	    while q:
	        u = q.popleft()
	        order.append(u)
	        for v in sorted(self.adj[u]):
	            if v not in visited:
	                visited.add(v)
	                q.append(v)
	    return order
	
	def dfs(self, s):
	    visited, order, stack = {s}, [], [s]
	    while stack:
	        u = stack.pop()
	        order.append(u)
	        for v in sorted(self.adj[u], reverse=True):
	            if v not in visited:
	                visited.add(v)
	                stack.append(v)
	    return order
```
### Сложности
Пусть $|V| = n$, $|E| = m$.
- Список смежности: память $O(n + m)$, проверка ребра $O(\deg u)$.
- Матрица смежности: память $O(n^2)$, проверка ребра $O(1)$.
- BFS / DFS: $O(n + m)$ на списке смежности.
### Замечания
- Для ориентированного графа `directed=True`, ребро $u \to v$ добавляется только в одну сторону.
