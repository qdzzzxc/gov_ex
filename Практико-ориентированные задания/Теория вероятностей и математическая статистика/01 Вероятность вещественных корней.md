# Задача 1. Вероятность вещественности корней квадратного уравнения.

## Условие
Найти вероятность того, что корни уравнения $x^2 + 7px + 3q = 0$ будут действительными числами, если коэффициенты $p$ и $q$ выбираются наудачу из отрезка $[-2,\ 2]$.

## Решение

### Программа минимум

```python
import numpy as np
rng = np.random.default_rng(1)
p, q = rng.uniform(-2, 2, (2, 1_000_000))  # случайные коэффициенты из квадрата
print((49*p*p - 12*q >= 0).mean())         # доля случаев с D >= 0
```

### Полное решение

### Идея (геометрическая вероятность)
Корни вещественны $\iff$ дискриминант $D \ge 0$:
$$D = (7p)^2 - 4 \cdot 1 \cdot 3q = 49 p^2 - 12 q \ge 0 \iff q \le \frac{49 p^2}{12}.$$

$(p, q)$ равномерно распределена в квадрате $[-2;\,2]^2$ площади $S_\square = 16$. Искомая вероятность — отношение площади «допустимой» области к 16:
$$P = \frac{1}{16} \iint_{q \le 49 p^2 / 12,\ |p|,|q|\le 2} \mathrm{d}p\,\mathrm{d}q.$$

### Геометрический разбор
Парабола $q = \tfrac{49}{12} p^2$ пересекает прямую $q = 2$ при $p = \pm p_0$, где
$$p_0 = \sqrt{\tfrac{24}{49}} = \tfrac{2\sqrt 6}{7} \approx 0{,}700.$$

- При $|p| > p_0$ парабола выше уровня $q = 2$, поэтому $q \le \tfrac{49p^2}{12}$ выполнено для всех $q \in [-2;\,2]$ — допустима полоса $4 \cdot (\,\text{длина по }p)$.
- При $|p| \le p_0$ — допустима лишь часть $q \in [-2;\,49p^2/12]$, длина $49p^2/12 + 2$.

### Подсчёт площади
$$S = \underbrace{4 \cdot 2(2 - p_0)}_{\text{края по }|p|>p_0} + \underbrace{\int_{-p_0}^{p_0}\!\left(\tfrac{49 p^2}{12} + 2\right)\mathrm{d}p}_{\text{середина}}.$$

Середина:
$$\int_{-p_0}^{p_0}\!\!\left(\tfrac{49 p^2}{12} + 2\right)\mathrm{d}p = 2\!\int_{0}^{p_0}\!\left(\tfrac{49 p^2}{12} + 2\right)\!\mathrm{d}p = \tfrac{49 p_0^3}{18} + 4 p_0.$$

Итого:
$$S = 16 - 8 p_0 + \tfrac{49 p_0^3}{18} + 4 p_0 = 16 - 4 p_0 + \tfrac{49 p_0^3}{18}.$$

Подставляем $p_0 = \tfrac{2\sqrt 6}{7}$, $p_0^3 = \tfrac{48\sqrt 6}{343}$:
$$\tfrac{49 p_0^3}{18} = \tfrac{49 \cdot 48 \sqrt 6}{343 \cdot 18} = \tfrac{8 \sqrt 6}{21},\qquad 4 p_0 = \tfrac{8 \sqrt 6}{7}.$$

$$S = 16 - \tfrac{8\sqrt 6}{7} + \tfrac{8\sqrt 6}{21} = 16 + 8\sqrt 6 \!\left(\tfrac{1}{21} - \tfrac{1}{7}\right) = 16 - \tfrac{16 \sqrt 6}{21}.$$

$$\boxed{P = \frac{S}{16} = 1 - \frac{\sqrt 6}{21} \approx 0{,}8834.}$$

### Проверка по Монте-Карло

```python
import numpy as np
rng = np.random.default_rng(7)
N = 10**7
ps = rng.uniform(-2, 2, N); qs = rng.uniform(-2, 2, N)
print((49*ps**2 >= 12*qs).mean())   # → 0.8834
```

Совпадение с аналитической формулой — до 4-го знака.

### Ответ
$$P = 1 - \frac{\sqrt 6}{21} \approx 0{,}883.$$
