# Задача 1. Вероятность вещественности корней квадратного уравнения.

## Условие
Найти вероятность того, что корни уравнения $x^2 + 7px + 3q = 0$ будут действительными числами, если коэффициенты $p$ и $q$ выбираются наудачу из отрезка $[-2,\ 2]$.

## Решение
## Идея (геометрическая вероятность)
Корни вещественны $\iff D \ge 0$: $$D = 49 p^2 - 12 q \ge 0 \iff q \le \frac{49 p^2}{12}$$ $(p, q)$ равномерно в квадрате $[-2, 2]^2$ площади 16. Благоприятная область — под параболой, неблагоприятная — маленькая «чашка» над параболой при $|p| \le p_0$. Считаем через дополнение: $$P = 1 - \frac{S_{\text{над параболой}}}{16}$$
## Расчёт
Парабола $q = \tfrac{49}{12} p^2$ пересекает $q = 2$ при $p_0 = \sqrt{24/49} = \tfrac{2\sqrt 6}{7}$. Площадь чашки (интеграл «верх минус низ»): $$S = \int_{-p_0}^{p_0}!\left(2 - \frac{49 p^2}{12}\right) dp = 4 p_0 - \frac{49 p_0^3}{18}$$ Подставляем $p_0 = \tfrac{2\sqrt 6}{7}$, $p_0^3 = \tfrac{48\sqrt 6}{343}$:
- $4 p_0 = \tfrac{8\sqrt 6}{7} = \tfrac{24\sqrt 6}{21}$
- $\tfrac{49 p_0^3}{18} = \tfrac{8\sqrt 6}{21}$ $$S = \frac{24\sqrt 6}{21} - \frac{8\sqrt 6}{21} = \frac{16\sqrt 6}{21}$$ $$P = 1 - \frac{16\sqrt 6 / 21}{16} = 1 - \frac{\sqrt 6}{21} \approx 0{,}883$$
## Проверка по Монте-Карло
```python
import numpy as np
rng = np.random.default_rng(7)
N = 10**7
ps = rng.uniform(-2, 2, N)
qs = rng.uniform(-2, 2, N)
print((49*ps**2 >= 12*qs).mean())
```
Результат 0.8834 — совпадает с аналитикой до 4-го знака.
## Ответ
$$P = 1 - \frac{\sqrt 6}{21} \approx 0{,}883$$
