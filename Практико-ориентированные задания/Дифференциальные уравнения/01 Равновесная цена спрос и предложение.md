# Задача 1. Равновесная цена: функции спроса и предложения.

## Условие
Функции спроса ($D$) и предложения ($S$) имеют вид:
$$D = 2p'' - 2p' + 20,$$
$$S = 2p'' + 2p' + 3p + 8.$$

Найти зависимость равновесной цены $p = p(t)$ от времени, если в начальный момент $p = 4,\ p' = 2$.

## Решение

## Идея
Равновесие на рынке — это $D = S$. Приравниваем: $$2p'' - 2p' + 20 = 2p'' + 2p' + 3p + 8$$ Слагаемые $2p''$ сокращаются, остаётся ОДУ первого порядка: $$4p' + 3p = 12$$
## Решение ОДУ
Линейное неоднородное первого порядка. Решение = однородное + частное. Однородное $4p' + 3p = 0$, разделяем переменные: $$\frac{dp}{p} = -\frac{3}{4},dt \Rightarrow \ln|p| = -\frac{3}{4}t + C_1 \Rightarrow p_h = C e^{-3t/4}$$ Частное (правая часть — константа): $p_p = A$, подставляем в $3p = 12$, получаем $A = 4$. Общее решение: $p(t) = 4 + C e^{-3t/4}$.
## Подбор константы
Из $p(0) = 4$: $4 + C = 4 \Rightarrow C = 0$. $$p(t) \equiv 4$$
## Замечание о начальных условиях
Условие $p'(0) = 2$ противоречит решению: при $p \equiv 4$ имеем $p'(t) = 0 \ne 2$. Задача в строгом прочтении «$D = S$ как алгебраическое равенство» **переопределена** — двух начальных условий для ОДУ первого порядка слишком много. Возможные интерпретации:
- Опечатка в условии (либо $p'(0) = 0$, либо одно из начальных условий лишнее).
- Подразумевалась динамическая модель Уолраса $p'' = D - S$ — тогда ОДУ второго порядка, оба условия согласованы, решение $p(t) = 4 + e^{-t} - e^{-3t}$.
## Ответ
В строгой статической постановке $D = S$: $p(t) \equiv 4$ (равновесная цена постоянна), условие $p'(0) = 2$ нарушается из-за переопределённости задачи.
