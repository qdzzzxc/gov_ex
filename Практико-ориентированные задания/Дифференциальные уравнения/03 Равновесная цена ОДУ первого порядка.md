# Задача 33. Равновесная цена: ОДУ первого порядка.

## Условие
Функции спроса и предложения зависят от цены $p(t)$ и скорости её изменения $\dot p = dp/dt$:
$$y = 25 - 2p + 3\dot p,\qquad x = 15 - p + 4\dot p.$$

Найти $p(t)$ при условии равновесия $x = y$ и начальном условии $p(0) = 9$.

## Решение

### Идея
Приравниваем спрос и предложение:
$$25 - 2p + 3\dot p = 15 - p + 4\dot p.$$
Переносим и упрощаем:
$$10 - p - \dot p = 0 \quad\Longleftrightarrow\quad \boxed{\dot p + p = 10.}$$

Это линейное ОДУ 1-го порядка с постоянными коэффициентами (заодно с разделяющимися переменными: $dp/(10 - p) = dt$).

**Однородное:** $\dot p + p = 0 \Rightarrow p_h = C e^{-t}$.
**Частное:** ищем $p_p = A = \text{const}$, тогда $A = 10$.
**Общее:** $p(t) = 10 + C e^{-t}$.

Из $p(0) = 9$: $10 + C = 9 \Rightarrow C = -1$.

### Код
```python
from sympy import Function, symbols, dsolve, Eq
t = symbols('t')
p = Function('p')
ode = Eq(p(t).diff(t), 10 - p(t))
print(dsolve(ode, p(t), ics={p(0): 9}))
```

### Прогон
```
Eq(p(t), 10 - exp(-t))
```

### Ответ
$$\boxed{p(t) = 10 - e^{-t}.}$$

Проверка: $\dot p = e^{-t}$, и $10 - p = 10 - (10 - e^{-t}) = e^{-t}$ — совпадает ✓.
$p(0) = 10 - 1 = 9$ ✓.
