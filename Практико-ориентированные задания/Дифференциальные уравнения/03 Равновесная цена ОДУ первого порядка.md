# Задача 33. Равновесная цена: ОДУ первого порядка.

## Условие
Функции спроса и предложения зависят от цены $p(t)$ и скорости её изменения $\dot p = dp/dt$:
$$y = 25 - 2p + 3\dot p,\qquad x = 15 - p + 4\dot p.$$

Найти $p(t)$ при условии равновесия $x = y$ и начальном условии $p(0) = 9$.

## Решение

### Идея
Приравниваем спрос и предложение:
$$25 - 2p + 3\dot p = 15 - p + 4\dot p.$$
Переносим и упрощаем:
$$10 - p - \dot p = 0 \quad\Longleftrightarrow\quad \boxed{\dot p + p = 10.}$$

Это линейное ОДУ 1-го порядка с постоянными коэффициентами (заодно с разделяющимися переменными: $dp/(10 - p) = dt$).

**Однородное:** $\dot p + p = 0 \Rightarrow p_h = C e^{-t}$.
**Частное:** ищем $p_p = A = \text{const}$, тогда $A = 10$.
**Общее:** $p(t) = 10 + C e^{-t}$.

Из $p(0) = 9$: $10 + C = 9 \Rightarrow C = -1$.

### Код
```python
from sympy import Function, symbols, dsolve, Eq
t = symbols('t')
p = Function('p')
ode = Eq(p(t).diff(t), 10 - p(t))
print(dsolve(ode, p(t), ics={p(0): 9}))
```

### Прогон
```
Eq(p(t), 10 - exp(-t))
```

### Ответ
$$\boxed{p(t) = 10 - e^{-t}.}$$

Проверка: $\dot p = e^{-t}$, и $10 - p = 10 - (10 - e^{-t}) = e^{-t}$ — совпадает ✓.
$p(0) = 10 - 1 = 9$ ✓.

### Замечания
- Стационарная (равновесная) цена $p^* = 10$ достигается асимптотически при $t \to \infty$: $p(t) \nearrow 10$, отклонение $|p - 10| = e^{-t}$ экспоненциально убывает. Система устойчива (корень характеристического уравнения $\lambda = -1 < 0$).
- Это **другая задача** относительно [[01 Равновесная цена спрос и предложение]]: там модель содержит вторые производные $p''$ и сводится к линейному ОДУ 2-го порядка с двумя начальными условиями $p(0), p'(0)$; здесь — только $\dot p$, ОДУ 1-го порядка, одно начальное условие.
- Экономический смысл коэффициентов при $\dot p$ ($+3$ в спросе, $+4$ в предложении): и покупатели, и продавцы реагируют на ожидаемый тренд цены, причём продавцы чувствительнее — поэтому при росте цены избыточное предложение быстро гасит отклонение.
