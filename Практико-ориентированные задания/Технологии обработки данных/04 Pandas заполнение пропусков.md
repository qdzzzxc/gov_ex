# Задача 4. Pandas: заполнение пропусков в датафрейме.

## Условие
Используя функционал библиотеки `pandas`, для произвольного датафрейма со столбцами `'col1'`, `'col2'`, `'col3'` определить число пропущенных значений в каждом столбце. Заменить в первом столбце пропуски на среднее значение, во втором — на медианное, а в третьем — на самое часто встречающееся. Решение предложить в виде программного кода.

## Решение

### Идея
- `col1`: числовой → среднее (`.mean()`);
- `col2`: числовой → медиана (`.median()`);
- `col3`: категориальный → мода (`.mode()[0]`).

`isna().sum()` даёт число пропусков; `fillna(...)` заполняет.

### Код
```python
import numpy as np, pandas as pd

df = pd.DataFrame({
    "col1": [10.5, np.nan, 12.0,  9.8, np.nan, 11.2, 10.9,  9.5, np.nan, 12.5],
    "col2": [50,    70,    np.nan, 80,    60,   np.nan, 75,   65,    55,   np.nan],
    "col3": ["A",   "A",   None,  "B",   "A",   "C",   "A",  "B",   None, "A"],
})

print("Пропусков по столбцам:")
print(df.isna().sum(), "\n")

df["col1"] = df["col1"].fillna(df["col1"].mean())
df["col2"] = df["col2"].fillna(df["col2"].median())
df["col3"] = df["col3"].fillna(df["col3"].mode()[0])

print("Осталось пропусков:", df.isna().sum().sum())
```
### Замечания
- Для категориального столбца **нельзя** использовать среднее/медиану — поэтому мода.
- `.mode()` возвращает Series (мод может быть несколько); `[0]` — берём первую.
- Альтернативные стратегии: `.ffill()/.bfill()` для временных рядов, KNN-imputer (`sklearn.impute.KNNImputer`) для коррелированных признаков.
