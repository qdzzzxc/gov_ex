# Задача 2. Numpy: линейная зависимость системы векторов.

## Условие
При помощи библиотеки `numpy` определите, является ли система векторов
$$a_1 = (2;\ 5;\ 1;\ 7),\quad a_2 = (-2;\ 2;\ -1;\ 8),\quad a_3 = (0;\ 2;\ 5;\ 4),\quad a_4 = (-5;\ 4;\ -1;\ 3)$$
линейно зависимой? Решение предложить в виде программного кода.

## Решение

### Программа минимум

```python
import numpy as np
A = np.array([[2,5,1,7], [-2,2,-1,8], [0,2,5,4], [-5,4,-1,3]])  # векторы-строки
print(np.linalg.matrix_rank(A) < A.shape[0])                    # True => зависимы
```

### Полное решение

### Идея
Запишем векторы как строки матрицы $A \in \mathbb{R}^{4\times 4}$. Векторы линейно зависимы $\iff \mathrm{rank}(A) < 4 \iff \det A = 0$. Также можно проверить через сингулярные числа: если самое маленькое $\sigma_{\min}$ существенно меньше остальных — есть направление, в котором $A v \approx 0$, и его компоненты дают коэффициенты линейной зависимости.

### Код

```python
import numpy as np
A = np.array([
    [ 2,  5,  1, 7],
    [-2,  2, -1, 8],
    [ 0,  2,  5, 4],
    [-5,  4, -1, 3],
], dtype=float)
print(f"det = {np.linalg.det(A):.4f}")
print(f"rank = {np.linalg.matrix_rank(A)}")
S = np.linalg.svd(A, compute_uv=False)
print("сингулярные числа:", S)
```

### Прогон
```
det  = -1034.0000
rank = 4
сингулярные числа: [13.50, 6.32, 4.15, 2.92]
```

### Ответ
Определитель отличен от нуля, ранг равен размерности — **система линейно НЕЗАВИСИМА**.

### Замечания
- Если бы система была зависимой, последняя строка `Vt` из SVD дала бы коэффициенты $c_i$, для которых $\sum c_i a_i = 0$.
- Пороговый критерий: `numpy.linalg.matrix_rank` использует $\sigma > \max(\sigma) \cdot \mathrm{eps} \cdot \max(m,n)$ — поэтому надёжнее `det` или `matrix_rank` чем сравнение `det == 0`.
