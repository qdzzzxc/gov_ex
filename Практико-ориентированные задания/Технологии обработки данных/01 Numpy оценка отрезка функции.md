# Задача 1. Numpy: оценка доли отрезка по значению функции.

## Условие
При помощи библиотеки `numpy` численно оценить долю отрезка от $-10$ до $10$, на которой значение функции $2x \cos(x) \cdot \exp(-0{,}01 x)$ больше $0{,}5$. Решение предложить в виде программного кода.

## Решение

### Программа минимум

```python
import numpy as np
x = np.linspace(-10, 10, 100_001)                 # густая сетка на отрезке
y = 2*x*np.cos(x)*np.exp(-0.01*x)                 # значения функции
print((y > 0.5).mean())                           # доля точек, где f(x) > 0.5
```

### Полное решение

### Идея
Густо посэмплируем функцию на $[-10;\,10]$, посчитаем долю узлов, в которых $f(x) > 0{,}5$. При сетке из $10^5$ точек на 20 ед. шаг $\Delta x = 2 \cdot 10^{-4}$ — погрешность по длине $< 5\cdot 10^{-4}$.

### Код

```python
import numpy as np
f = lambda x: 2*x*np.cos(x)*np.exp(-0.01*x)
xs = np.linspace(-10, 10, 100_001)
mask = f(xs) > 0.5
share = mask.mean()
print(f"доля отрезка: {share:.4f}  (≈ {share*20:.3f} из 20 ед.)")

# дополнительно — связные интервалы
diffs = np.diff(mask.astype(int))
starts = np.where(diffs == 1)[0] + 1
ends   = np.where(diffs == -1)[0] + 1
if mask[0]:  starts = np.r_[0, starts]
if mask[-1]: ends   = np.r_[ends, len(mask)]
for s, e in zip(starts, ends):
    print(f"  [{xs[s]:+.3f}; {xs[e-1]:+.3f}]  длина {xs[e-1]-xs[s]:.3f}")
```

### Прогон
```
доля отрезка:  0.4621  (≈ 9.243 из 20)

Связные интервалы где f(x) > 0.5:
  [-10.000; -7.883]   длина 2.117
  [ -4.661; -1.715]   длина 2.946
  [ +0.259; +1.387]   длина 1.128
  [ +4.768; +7.819]   длина 3.052
```

### Замечания
- Доля $\approx 0{,}46$ — почти половина отрезка.
- При желании уточнить границы можно использовать `scipy.optimize.brentq` для поиска корней $f(x) - 0{,}5 = 0$ в каждом подынтервале.
