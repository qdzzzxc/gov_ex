# Задача 1. Numpy: оценка доли отрезка по значению функции.

## Условие
При помощи библиотеки `numpy` численно оценить долю отрезка от $-10$ до $10$, на которой значение функции $2x \cos(x) \cdot \exp(-0{,}01 x)$ больше $0{,}5$. Решение предложить в виде программного кода.
## Решение
## Идея
Густо сэмплируем функцию на $[-10;,10]$, считаем долю узлов сетки, в которых $f(x) > 0{,}5$. Сетка из $10^5$ точек даёт шаг $\Delta x = 2 \cdot 10^{-4}$, погрешность доли $\sim 10^{-5}$ — запас огромный. Ключевая идиома: `.mean()` булевого массива в numpy возвращает долю `True`.
## Код
```python
import numpy as np

f = lambda x: 2 * x * np.cos(x) * np.exp(-0.01 * x)
xs = np.linspace(-10, 10, 100_001)
mask = f(xs) > 0.5
share = mask.mean()
print(f"доля отрезка: {share:.4f}  (≈ {share*20:.3f} из 20 ед.)")

diffs = np.diff(mask.astype(int))
starts = np.where(diffs == 1)[0] + 1
ends = np.where(diffs == -1)[0] + 1

if mask[0]:
    starts = np.r_[0, starts]
if mask[-1]:
    ends = np.r_[ends, len(mask)]

for s, e in zip(starts, ends):
    print(f"  [{xs[s]:+.3f}; {xs[e-1]:+.3f}]  длина {xs[e-1]-xs[s]:.3f}")
```

Поиск связных интервалов: `diffs == 1` ловит переход $F \to T$ (начало интервала), `diffs == -1` ловит переход $T \to F$ (конец). Граничные случаи отдельно: если `mask[0]` уже `True`, добавляем 0 в начала; если `mask[-1]` ещё `True`, добавляем длину массива в концы.
