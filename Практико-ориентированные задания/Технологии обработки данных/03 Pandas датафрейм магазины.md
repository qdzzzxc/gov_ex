# Задача 3. Pandas: выручка магазинов в Москве и Санкт-Петербурге.

## Условие
Используя функционал библиотеки `pandas`, создать два датафрейма из 10 строк со столбцами «Дата», «Магазин», «Город», «Выручка». Поле «Дата» может повторяться в одном датафрейме не более 3-х раз и начинается с текущего числа. Поле «Магазин» содержит один из трёх вариантов. Поле «Город» — либо «Москва», либо «Санкт-Петербург». Выручка — модуль нормально распределённой случайной величины (больше $0$) с $\mu = 150\,000$, $\sigma = 25\,000$.

Объединить данные в один датафрейм и найти среднюю выручку по каждому магазину. Решение предложить в виде программного кода.

## Решение

### Программа минимум

```python
import numpy as np, pandas as pd
rng = np.random.default_rng(1)
dates = pd.date_range(pd.Timestamp.today().normalize(), periods=4).repeat([3, 3, 2, 2])  # повторы <= 3
def make_df():
    return pd.DataFrame({"Дата": dates, "Магазин": rng.choice(["A","B","C"], 10),
                         "Город": rng.choice(["Москва","Санкт-Петербург"], 10),
                         "Выручка": abs(rng.normal(150_000, 25_000, 10))})
df = pd.concat([make_df(), make_df()], ignore_index=True)        # объединяем 2 таблицы
print(df.groupby("Магазин")["Выручка"].mean())                   # средняя выручка
```

### Полное решение

### Идея
1. Сгенерировать два датафрейма по 10 строк со случайными датами (с ограничением «не более 3 повторов одной даты»), магазинами из 3-х вариантов, городами из 2-х, выручкой $\sim |\mathcal{N}(150\,000,\, 25\,000)|$.
2. Объединить их через `pd.concat`.
3. Группировка по магазину и `.mean()` по выручке.

### Код

```python
import numpy as np, pandas as pd

shops  = ["Mag1", "Mag2", "Mag3"]
cities = ["Москва", "Санкт-Петербург"]

def build(seed):
    r = np.random.default_rng(seed)
    today = pd.Timestamp.today().normalize()
    while True:
        dates = r.choice(pd.date_range(today, today + pd.Timedelta(days=4)), size=10)
        if (pd.Series(dates).value_counts() <= 3).all():
            break
    return pd.DataFrame({
        "Дата":    dates,
        "Магазин": r.choice(shops, size=10),
        "Город":   r.choice(cities, size=10),
        "Выручка": np.abs(r.normal(150_000, 25_000, size=10)),
    })

df = pd.concat([build(1), build(2)], ignore_index=True)
print(df.groupby("Магазин")["Выручка"].mean().round(2))
```

### Прогон
```

  Mag1    148 297.75
  Mag2    151 015.01
  Mag3    144 848.20
```

### Замечания
- Цикл `while` гарантирует ограничение «не более 3-х повторов даты»; для 10 элементов на 5 дат это почти всегда достигается с первого раза.
- `np.abs(...)` обеспечивает положительность выручки (модуль нормальной величины); вероятность отрицательного значения при $\mu=150\,000$, $\sigma=25\,000$ ничтожна, но формальное требование «больше 0» соблюдено.
