# Задача 6. Максимум прибыли с обратной функцией спроса.

## Условие
Суммарные издержки предприятия на производство продукции объёма $x$ задаются функцией
$$K(x) = \frac{x^2}{30} + 20x + 300,$$
а цена единицы товара — обратной функцией спроса
$$p(x) = 60 - \frac{x}{20}.$$
При каком объёме производства $x$ предприятие получает максимальную прибыль?

## Решение

### Идея
Выручка $TR(x) = p(x)\cdot x$, прибыль $\pi(x) = TR(x) - K(x)$. Поскольку $p(x)$ линейно убывает по $x$, $TR$ — парабола ветвями вниз, и $\pi$ — тоже квадратичная по $x$ с отрицательным старшим коэффициентом. Максимум ищем из условия $\pi'(x)=0$; вторая производная $\pi''(x)<0$ подтверждает, что это именно максимум.

### Выкладки
Выручка:
$$TR(x) = \left(60 - \tfrac{x}{20}\right) x = 60x - \frac{x^2}{20}.$$

Прибыль:
$$\pi(x) = TR(x) - K(x) = 60x - \frac{x^2}{20} - \frac{x^2}{30} - 20x - 300.$$

Сворачиваем квадратичные слагаемые: $\dfrac{1}{20} + \dfrac{1}{30} = \dfrac{3+2}{60} = \dfrac{1}{12}$. Тогда
$$\pi(x) = -\frac{x^2}{12} + 40x - 300.$$

Первая производная:
$$\pi'(x) = -\frac{x}{6} + 40 = 0 \;\Longrightarrow\; x^* = 240.$$

Вторая производная:
$$\pi''(x) = -\frac{1}{6} < 0,$$
значит, в точке $x^*=240$ — глобальный максимум (парабола ветвями вниз).

Максимальная прибыль:
$$\pi(240) = -\frac{240^2}{12} + 40\cdot 240 - 300 = -4800 + 9600 - 300 = 4500.$$

### Код
```python
import sympy as sp

x = sp.symbols('x', positive=True)
K = x**2/30 + 20*x + 300
p = 60 - x/20
pi = sp.expand(p*x - K)
x_star = sp.solve(sp.diff(pi, x), x)[0]
print("π(x)   =", pi)
print("π'(x)  =", sp.diff(pi, x))
print("π''(x) =", sp.diff(pi, x, 2))
print("x*     =", x_star)
print("π(x*)  =", pi.subs(x, x_star))
```

### Прогон
```
π(x)   = -x**2/12 + 40*x - 300
π'(x)  = 40 - x/6
π''(x) = -1/6
x*     = 240
π(x*)  = 4500
```

### Ответ
Оптимальный объём $x^* = 240$, максимальная прибыль $\pi_{\max} = \pi(240) = 4500$ у.е. Условие второго порядка $\pi''(x) = -\tfrac{1}{6} < 0$ выполнено — максимум.

### Замечания
- Эквивалентная формулировка через предельные величины: $MR(x) = TR'(x) = 60 - x/10$, $MC(x) = K'(x) = x/15 + 20$. Условие $MR = MC$ даёт $60 - x/10 = x/15 + 20$, т.е. $40 = x/10 + x/15 = x/6$, откуда $x^* = 240$ — тот же ответ.
- В точке оптимума цена $p(240) = 60 - 12 = 48$, выручка $TR(240) = 48\cdot 240 = 11{,}520$, издержки $K(240) = 5760 + 4800 + 300 = 10{,}860$, прибыль $11{,}520 - 10{,}860 = 660$... — стоп: пересчёт показывает, что это противоречит $\pi=4500$. Проверим: $K(240) = 240^2/30 + 20\cdot 240 + 300 = 1920 + 4800 + 300 = 7020$. Тогда $\pi = 11{,}520 - 7020 = 4500$. ✓ (Типичная ошибка — спутать $x^2/30$ с $x^2/3$ или $24^2$ с $240^2$.)
- Постоянные издержки $300$ не влияют на $x^*$ (производная от константы — ноль), но сдвигают сам максимум прибыли вниз на $300$.
