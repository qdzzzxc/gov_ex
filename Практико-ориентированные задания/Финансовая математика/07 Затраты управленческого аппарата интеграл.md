# Задача 36. Полные и средние издержки на управленческий аппарат.

## Условие
Затраты фирмы на содержание управленческого аппарата за период $0 \le t \le 6$ (полгода) определяются «скоростью» затрат (cost rate)
$$f(t) = 50 + 10\sin^2\!\left(\frac{\pi t}{6}\right).$$
Найти полные ($TC$) и средние ($AC$) издержки за этот период.

## Решение

### Идея
$f(t)$ — скорость затрат, то есть издержки в единицу времени. Полные издержки накапливаются интегрированием по времени:
$$TC = \int_{0}^{6} f(t)\,dt, \qquad AC = \frac{TC}{6}.$$

Понижаем степень синуса по формуле $\sin^2 u = \dfrac{1-\cos 2u}{2}$, где $u = \pi t/6$, тогда $2u = \pi t/3$:
$$10\sin^2\!\frac{\pi t}{6} = 10\cdot\frac{1-\cos(\pi t/3)}{2} = 5 - 5\cos\!\frac{\pi t}{3}.$$

Следовательно,
$$f(t) = 50 + 5 - 5\cos\!\frac{\pi t}{3} = 55 - 5\cos\!\frac{\pi t}{3}.$$

Считаем интеграл:
$$\int_{0}^{6} 55\,dt = 330,$$
$$\int_{0}^{6} \cos\!\frac{\pi t}{3}\,dt = \frac{3}{\pi}\sin\!\frac{\pi t}{3}\bigg|_{0}^{6} = \frac{3}{\pi}\bigl(\sin 2\pi - \sin 0\bigr) = 0.$$

Итого $TC = 330 - 5\cdot 0 = 330$, $AC = 330/6 = 55$.

### Код
```python
import sympy as sp
from scipy.integrate import quad
import math

t = sp.symbols('t')
f = 50 + 10*sp.sin(sp.pi*t/6)**2
TC_sym = sp.integrate(f, (t, 0, 6))
print('TC (sympy)  =', sp.simplify(TC_sym))
print('AC          =', sp.simplify(TC_sym/6))

g = lambda x: 50 + 10*math.sin(math.pi*x/6)**2
TC_num, _ = quad(g, 0, 6)
print('TC (scipy)  =', TC_num)
```

### Прогон
```
TC (sympy)  = 330
AC          = 55
TC (scipy)  = 330.0
```

### Ответ
- Полные издержки: $TC = 330$ (ден. ед. за полгода).
- Средние издержки: $AC = TC/6 = 55$ (ден. ед. в единицу времени).

### Замечания
- В методичке указан ответ $TC = 360$, $AC = 60$. Это **арифметическая ошибка авторов**: в их выкладке записано
  $300 + 10\int_{0}^{6}\!\frac{1-\cos(\pi t/3)}{2}\,dt = 360$,
  но реально $10\cdot\frac{1}{2}\cdot 6 = 30$, а не $60$. Колеблющаяся часть $\int_{0}^{6}\cos(\pi t/3)\,dt$ обнуляется (за $[0,6]$ укладывается ровно один полный период функции $\cos(\pi t/3)$), поэтому правильный результат $300 + 30 = 330$.
- Численное подтверждение через `sympy` и `scipy.quad` даёт $330{,}0$ — расхождение с методичкой не из-за округления.
- Геометрический смысл: $f(t)$ колеблется около среднего уровня $55$ с амплитудой $5$ и периодом $6$; на отрезке длиной ровно в один период средняя $\dfrac{1}{6}\int_0^6 f\,dt = 55$.
