# Задача 3. Иммунизация долга бескупонными и купонными облигациями.

## Условие
Финансовый менеджер ищет способ выплатить через четыре года долг в размере $250{,}000$ у.е. Сколько 3-летних бескупонных облигаций типа A (цена одной такой облигации составляет $772{,}1835$ у.е., номинал $1000$ у.е.) и сколько 6-летних облигаций типа B (купонная ставка $10\%$, номинал $1000$ у.е., доходность к погашению $9\%$) ему следует купить, чтобы защитить средства от изменений процентной ставки?

## Решение

### Программа минимум

```python
import numpy as np
A = np.array([[772.1835, 1044.86], [772.1835*3, 1044.86*4.8192]]) # PV и PV*D
b = np.array([177106.30, 177106.30*4])                            # долг и его дюрация
print(np.linalg.solve(A, b))                                      # числа облигаций A и B
```

### Полное решение

### Идея
Иммунизация Реддингтона: подобрать портфель так, чтобы
1. $PV_{\text{портф}} = PV_{\text{долг}}$;
2. $D_{\text{портф}} = D_{\text{долг}}$ (Маколеевские дюрации совпадают);
3. (желательно) выпуклость портфеля $\ge$ выпуклости долга.

При выполнении (1)+(2) приведённая стоимость портфеля при малых параллельных сдвигах кривой ставок меняется так же, как у долга — обязательство «иммунизировано».

### Параметры

**Облигация A** (3-летняя, бескупонная, $F=1000$, цена $P_A = 772{,}1835$):
$$y_A = \left(\frac{1000}{772{,}1835}\right)^{1/3} - 1 = 9\%,\quad D_A = 3.$$

**Облигация B** (6-летняя, купон 10% годовых, $F=1000$, $y_B = 9\%$):
- Цена:
$$P_B = \sum_{t=1}^{6} \frac{100}{1{,}09^t} + \frac{1000}{1{,}09^6} = 100 \cdot 4{,}4859 + 1000 \cdot 0{,}5963 \approx 1044{,}86.$$
- Маколеевская дюрация:
$$D_B = \frac{1}{P_B}\sum_{t=1}^{6} t \cdot \frac{C_t}{1{,}09^t} \approx 4{,}819.$$

**Долг**: $\$250\,000$ через 4 года, дисконтируем по той же $y = 9\%$:
$$PV_d = \frac{250\,000}{1{,}09^4} \approx 177\,106{,}30,\quad D_d = 4.$$

### Система иммунизации
Пусть $N_A, N_B$ — количества облигаций. Имеем линейную систему:
$$\begin{cases} N_A P_A + N_B P_B = PV_d, \\ N_A P_A D_A + N_B P_B D_B = PV_d \cdot D_d. \end{cases}$$

Подставляя:
$$\begin{cases} 772{,}1835\, N_A + 1044{,}86\, N_B = 177\,106{,}30, \\ 2316{,}55\, N_A + 5035{,}40\, N_B = 708\,425{,}21. \end{cases}$$

### Решение

```python
import numpy as np
A = np.array([[772.1835,         1044.86         ],
              [772.1835*3,       1044.86*4.8192  ]])
b = np.array([177106.30, 177106.30*4])
N = np.linalg.solve(A, b)
# → N_A ≈ 103.28,  N_B ≈ 93.17
```

Численно: $N_A \approx 103{,}28$, $N_B \approx 93{,}17$.

### Ответ
В округлении до целого: купить **≈ 103 шт. облигаций A** и **≈ 93 шт. облигаций B**.

Проверка: $103{,}28 \cdot 772{,}18 + 93{,}17 \cdot 1044{,}86 \approx 177\,106 = PV_d$ ✓; $D_{\text{портф}} = 4{,}000 = D_d$ ✓.

### Замечания
- В нецелочисленной формулировке стратегия точно реализуема; на практике покупаются целые облигации, и небольшое расхождение покрывается денежным резервом.
- Иммунизация защищает только от **малых параллельных** сдвигов; при крупных или непараллельных изменениях кривой требуется ребалансировка.
- У портфеля бескупонной + долгосрочной купонной выпуклость выше, чем у одиночного бескупонного долга — третье условие Реддингтона выполняется автоматически.
