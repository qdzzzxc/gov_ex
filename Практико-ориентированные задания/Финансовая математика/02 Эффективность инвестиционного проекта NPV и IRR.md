# Задача 2. Эффективность инвестиционного проекта (NPV и IRR).

## Условие
Рассматривается инвестиционный проект. Начальные инвестиции оцениваются в $2$ млн руб., чистый доход по окончании первого года — $1{,}40$ млн руб., второго — $1{,}3$ млн руб. Оценить эффективность проекта, если ставка дисконтирования составляет $10\%$, используя критерий чистого приведённого дохода и внутреннюю норму доходности. Проанализировать эффективность проекта в зависимости от ставки дисконтирования.

## Решение

### Программа минимум

```python
from scipy.optimize import brentq
cf = [-2, 1.4, 1.3]                                      # поток денег по годам
npv = lambda r: sum(c/(1+r)**t for t, c in enumerate(cf)) # чистый приведённый доход
print(npv(0.10), brentq(npv, -0.5, 1.5))                  # NPV при 10% и IRR
```

### Полное решение

### Идея
Поток платежей (млн руб.): $C_0 = -2;\ C_1 = 1{,}4;\ C_2 = 1{,}3$.

**NPV** (Net Present Value):
$$\mathrm{NPV}(r) = \sum_{t=0}^{2} \frac{C_t}{(1+r)^t}.$$

**IRR** — ставка, при которой $\mathrm{NPV}(r) = 0$.

### Расчёт NPV при $r = 10\%$
$$\mathrm{NPV} = -2 + \frac{1{,}4}{1{,}1} + \frac{1{,}3}{1{,}21} = -2 + 1{,}2727 + 1{,}0744 = 0{,}3471 \text{ млн руб.}$$

### Расчёт IRR
Уравнение $-2 + 1{,}4 x + 1{,}3 x^2 = 0$, где $x = 1/(1+r)$. Корни $x \approx -1{,}8906$ или $x \approx 0{,}8137$. Берём положительный $\Rightarrow r = 1/x - 1 \approx 0{,}2289$ — точнее численно:

```python
from scipy.optimize import brentq
brentq(lambda r: -2 + 1.4/(1+r) + 1.3/(1+r)**2, -0.5, 1.5)
# → 0.22892
```

$\mathrm{IRR} \approx 22{,}89\%$.

### Анализ по ставке

| $r$  | NPV        |
|------|------------|
| 5%   | $+0{,}5125$ |
| 10%  | $+0{,}3471$ |
| 15%  | $+0{,}2004$ |
| 20%  | $+0{,}0694$ |
| 22,9%| $\approx 0$ ⟵ IRR |
| 25%  | $-0{,}0480$ |

### Ответ и вывод
- $\mathrm{NPV}(10\%) \approx 0{,}347$ млн руб. > 0 — проект эффективен при заданной ставке;
- $\mathrm{IRR} \approx 22{,}9\%$ — запас прочности существенный (более чем в 2 раза превышает 10%);
- проект остаётся прибыльным при $r < 22{,}9\%$, при более высоких ставках его принимать не стоит.

### Замечания
- Знак менялся один раз ($C_0 < 0$, остальные положительные) — IRR единственная и хорошо определена (не возникает «множественных» IRR из правила Декарта).
- На практике в дополнение к IRR смотрят MIRR и срок окупаемости — IRR не учитывает разные ставки реинвестирования.
