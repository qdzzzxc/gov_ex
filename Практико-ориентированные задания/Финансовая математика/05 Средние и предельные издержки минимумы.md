# Задача 32. Средние и предельные издержки: значения и минимумы.

## Условие
Суммарные издержки фирмы заданы функцией
$$K(x) = \frac{x^3}{3} - 8x^2 + 70x,$$
где $x$ — объём выпуска. Требуется:
1. найти средние и предельные издержки при $x = 5$;
2. определить, при каком $x$ средние издержки минимальны;
3. определить, при каком $x$ предельные издержки минимальны.

## Решение

### Идея
- **Средние издержки** $AC(x) = K(x)/x$ — издержки на единицу продукции.
- **Предельные издержки** $MC(x) = K'(x)$ — прирост суммарных издержек при увеличении выпуска на единицу.

Из явного вида $K(x)$:
$$AC(x) = \frac{x^2}{3} - 8x + 70,\qquad MC(x) = x^2 - 16x + 70.$$

Минимумы — из условия первого порядка $AC'(x) = 0$ и $MC'(x) = 0$ (вторая производная у обеих функций положительна — это параболы ветвями вверх, так что найденная критическая точка автоматически минимум).

### Значения при $x = 5$
$$AC(5) = \frac{25}{3} - 40 + 70 = \frac{25}{3} + 30 = \frac{115}{3} \approx 38{,}33,$$
$$MC(5) = 25 - 80 + 70 = 15.$$

При $x=5$ предельные издержки заметно ниже средних — каждая следующая единица обходится дешевле текущего среднего, значит $AC$ ещё убывает.

### Минимум $AC$
$$AC'(x) = \frac{2x}{3} - 8 = 0 \;\Rightarrow\; x^* = 12.$$
Значение в точке минимума: $AC(12) = 48 - 96 + 70 = 22$.

В этой точке $MC(12) = 144 - 192 + 70 = 22 = AC(12)$ — классическое свойство: кривая $MC$ пересекает $AC$ ровно в точке минимума $AC$.

### Минимум $MC$
$$MC'(x) = 2x - 16 = 0 \;\Rightarrow\; x^{**} = 8.$$
Значение в точке минимума: $MC(8) = 64 - 128 + 70 = 6$.

### Код (проверка через sympy)

```python
import sympy as sp
x = sp.symbols('x', positive=True)
K  = x**3/3 - 8*x**2 + 70*x
AC = sp.simplify(K/x)
MC = sp.diff(K, x)
print('AC(5) =', AC.subs(x,5), '≈', float(AC.subs(x,5)))
print('MC(5) =', MC.subs(x,5))
print('argmin AC:', sp.solve(sp.diff(AC,x), x), '→ AC =', AC.subs(x,12))
print('argmin MC:', sp.solve(sp.diff(MC,x), x), '→ MC =', MC.subs(x,8))
```

### Прогон
```
AC(5) = 115/3 ≈ 38.333333333333336
MC(5) = 15
argmin AC: [12] → AC = 22
argmin MC: [8]  → MC = 6
```

### Ответ
- $AC(5) = \dfrac{115}{3} \approx 38{,}33$, $\; MC(5) = 15$.
- Средние издержки минимальны при $x = 12$ (там $AC = MC = 22$).
- Предельные издержки минимальны при $x = 8$ (там $MC = 6$).

### Замечания
- Порядок точек закономерен: $\arg\min MC = 8 < \arg\min AC = 12$. Сначала перестаёт убывать предельная стоимость следующей единицы, и только потом — средняя; на участке $x \in (8, 12)$ $MC$ уже растёт, но всё ещё ниже $AC$, поэтому $AC$ продолжает падать.
- В точке $x^* = 12$ выполняется тождество $MC(x^*) = AC(x^*)$: дифференцируя $AC(x) = K(x)/x$, получаем $AC'(x) = (MC(x) - AC(x))/x$, откуда $AC'=0 \Leftrightarrow MC = AC$.
- Экономический смысл: при $x < 8$ производство «разгоняется» (предельная стоимость падает за счёт эффектов специализации/масштаба), при $x > 12$ — наступает фаза убывающей отдачи, и каждая дополнительная единица увеличивает средние издержки.
