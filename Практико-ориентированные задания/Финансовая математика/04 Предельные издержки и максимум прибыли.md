# Задача 4. Предельные издержки и максимум прибыли.

## Условие
Функция предельных издержек фирмы имеет вид $f(x) = 80 + 0{,}04 x$ (руб./ед.). Известно, что издержки производства $50$ единиц продукции составляют $4300$ руб. Найти:
1. функцию полных издержек $TC(x)$;
2. величину фиксированных (постоянных) издержек $FC$;
3. полные издержки производства $200$ единиц продукции;
4. максимальное значение прибыли, если продукция реализуется по цене $p = 100$ руб./ед.
## Решение

### Идея
Предельные издержки — производная функции полных издержек: $MC(x) = TC'(x)$. Значит, полные издержки восстанавливаются интегрированием:
$$TC(x) = \int MC(x)\, dx + C = \int (80 + 0{,}04 x)\, dx + C = 80 x + 0{,}02 x^2 + C.$$

Константа $C$ — фиксированные издержки $FC = TC(0)$ (расходы, не зависящие от объёма выпуска: аренда, оклады АУП, амортизация и т.п.). Находим её из условия $TC(50) = 4300$:
$$80 \cdot 50 + 0{,}02 \cdot 50^2 + C = 4300 \;\Rightarrow\; 4000 + 50 + C = 4300 \;\Rightarrow\; C = 250.$$

Итого:
$$\boxed{TC(x) = 0{,}02 x^2 + 80 x + 250,\quad FC = 250.}$$

Подставляем $x = 200$:
$$TC(200) = 0{,}02 \cdot 40\,000 + 80 \cdot 200 + 250 = 800 + 16\,000 + 250 = 17\,050.$$

Выручка $TR(x) = p x = 100 x$, прибыль:
$$\pi(x) = TR(x) - TC(x) = 100 x - 0{,}02 x^2 - 80 x - 250 = -0{,}02 x^2 + 20 x - 250.$$

Условие максимума: $\pi'(x) = -0{,}04 x + 20 = 0 \;\Rightarrow\; x^* = 500$. Вторая производная $\pi''(x) = -0{,}04 < 0$ — действительно максимум. Экономический смысл условия: $MR = MC$, т.е. $100 = 80 + 0{,}04 x^* \Rightarrow x^* = 500$.

$$\pi_{\max} = \pi(500) = -0{,}02 \cdot 250\,000 + 20 \cdot 500 - 250 = -5000 + 10\,000 - 250 = 4750.$$

### Код
```python
from sympy import symbols, integrate, solve, diff, Rational
x, C = symbols('x C')
MC = 80 + Rational(4, 100) * x
TC = integrate(MC, x) + C
C_sol = solve(TC.subs(x, 50) - 4300, C)[0]
TC = TC.subs(C, C_sol)
pi = 100 * x - TC
x_star = solve(diff(pi, x), x)[0]
print('TC(x) =', TC)
print('FC =', TC.subs(x, 0))
print('TC(200) =', TC.subs(x, 200))
print('x* =', x_star, ' pi_max =', pi.subs(x, x_star))
```

### Прогон
```
TC(x) = x**2/50 + 80*x + 250
FC = 250
TC(200) = 17050
x* = 500  pi_max = 4750
```

### Ответ
1. $TC(x) = 0{,}02 x^2 + 80 x + 250$ руб.;
2. $FC = 250$ руб.;
3. $TC(200) = 17\,050$ руб.;
4. $\pi_{\max} = 4750$ руб. при оптимальном выпуске $x^* = 500$ ед.

### Замечания
- Условие первого порядка $\pi'(x) = 0$ равносильно классическому правилу $MR = MC$: при линейной цене $MR = p = \text{const}$, а $MC$ возрастает, поэтому оптимум единственный.
- $FC$ присутствует в $TC$, но не влияет на $x^*$ — он зависит только от соотношения предельных величин. Изменение $FC$ сдвигает $\pi_{\max}$ на ту же величину.
- При $x^* = 500$: $TC(500) = 0{,}02 \cdot 250\,000 + 80 \cdot 500 + 250 = 5000 + 40\,000 + 250 = 45\,250$, $TR = 50\,000$, проверка: $50\,000 - 45\,250 = 4750$. Сходится.
- Условие $\pi > 0$ выполнено в окрестности максимума, но при больших $x$ прибыль уходит в минус (квадрат с минусом): корни $\pi(x) = 0$ дают точки безубыточности $x_{1,2} = 500 \pm \sqrt{250\,000 - 12\,500} \approx 500 \pm 487{,}3$, т.е. примерно $x_1 \approx 12{,}7$ и $x_2 \approx 987{,}3$.
