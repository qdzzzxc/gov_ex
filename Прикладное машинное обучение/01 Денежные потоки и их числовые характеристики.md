# Денежные потоки и их числовые характеристики. Чистая приведённая стоимость. Ренты. Объединение и замена рент. Инвестиционные потоки. Внутренняя норма доходности.
## Денежный поток
Последовательность платежей ${C_0, C_1, \dots, C_n}$ в моменты $t_0 = 0, t_1, \dots, t_n$. Положительные — поступления, отрицательные — выплаты. Ставка $r$ — коэффициент дисконтирования (депозит, стоимость капитала, альтернативные вложения).
## Приведённая стоимость (PV)
Один платёж: $PV = \dfrac{C}{(1 + r)^t}$. Поток: $PV = \sum_{k=0}^n \dfrac{C_k}{(1+r)^{t_k}}$. При непрерывном начислении: $PV = \int_0^T f(t) e^{-rt},dt$.
## NPV (Net Present Value)
PV в контексте инвестпроекта, где $C_0 < 0$ (вложение): $$NPV = C_0 + \sum_{k=1}^n \frac{C_k}{(1+r)^k}$$ Критерий принятия: $NPV > 0$ — проект выгоден.
## Ренты (аннуитеты)
Регулярный поток — одинаковые платежи $C$ через равные периоды. Постнумерандо (платёж в конце периода, стандарт): $$PV = C \cdot a_{n, r}, \quad a_{n, r} = \frac{1 - (1+r)^{-n}}{r}$$Пренумерандо (платёж в начале периода): $PV = C \cdot a_{n, r} \cdot (1+r)$.
Бессрочная рента ($n \to \infty$, $r > 0$): $PV = C/r$.
Растущая (модель Гордона, рост платежа на g) при $r > g$: $PV = C/(r - g)$. Используется в модели дисконтирования дивидендов.
## Принцип эквивалентности
Два потока эквивалентны, если их приведённые стоимости равны при заданной ставке. Основа для:
- замены одной ренты на другую с изменёнными параметрами;
- объединения нескольких рент в одну;
- реструктуризации кредитов и пенсионных схем.
## IRR — внутренняя норма доходности
Ставка $r^*$, при которой $NPV = 0$: $$\sum_{k=0}^n \frac{C_k}{(1 + r^*)^k} = 0$$ Содержательно — встроенная доходность проекта. Критерий принятия: $IRR >$ стоимости капитала. Единственность: если знак $C_k$ меняется один раз ($C_0 < 0$, далее $\ge 0$) — единственный положительный IRR (правило Декарта). При многократной смене знаков IRR может быть несколько или ни одного.
## NPV vs IRR
Для одного проекта «брать/не брать» — IRR работает. При сравнении взаимоисключающих проектов надёжнее NPV: IRR не учитывает масштаб вложений.
