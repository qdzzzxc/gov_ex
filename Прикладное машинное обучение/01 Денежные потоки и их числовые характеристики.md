# Денежные потоки и их числовые характеристики. Чистая приведённая стоимость. Ренты. Объединение и замена рент. Инвестиционные потоки. Внутренняя норма доходности.

## TL;DR
**Денежный поток (CF)** — последовательность платежей $\{(t_k, C_k)\}$. **Чистая приведённая стоимость (NPV)** — $\sum C_k/(1+r)^{t_k}$ при ставке дисконтирования $r$. **Рента (аннуитет)** — регулярный поток $C, C, \dots, C$; PV постнумерандо: $C\cdot a_{n,r}$, $a_{n,r} = (1-(1+r)^{-n})/r$; пренумерандо: $C\cdot \ddot a_{n,r} = a_{n,r}\cdot(1+r)$. Бессрочная ($n\to\infty$): $C/r$. Объединение и замена — на основе равенства приведённых стоимостей. **IRR** — ставка $r$, при которой NPV = 0; критерий принятия проекта IRR > стоимости капитала.

## Развёрнуто

### Денежный поток
Последовательность $\{C_0, C_1,\dots,C_n\}$ платежей в моменты $t_0=0, t_1,\dots,t_n$. Положительные $C_k$ — поступления, отрицательные — выплаты.

### Числовые характеристики
- **Приведённая стоимость (PV)**: $PV = \sum_{k=0}^n C_k\,(1+r)^{-t_k}$.
- **Будущая стоимость (FV)**: $FV = \sum C_k\,(1+r)^{T-t_k}$ — на горизонте $T$.
- **Чистая приведённая стоимость (NPV)**: то же, что PV, при условии что $C_0$ — инвестиция (отрицательная). Критерий принятия: $NPV > 0$.
- При непрерывном начислении: $PV = \sum C_k\,e^{-r t_k}$ или для непрерывного потока $f(t)$: $PV = \int_0^T f(t)e^{-rt}\,dt$.

### Ренты (аннуитеты)
**Постнумерандо** (платёж в конце периода): $C$ выплачивается в моменты $1, 2, \dots, n$.
$$PV_{\text{post}} = C\cdot a_{n,r} = C\cdot \frac{1-(1+r)^{-n}}{r}.$$
$$FV_{\text{post}} = C\cdot s_{n,r} = C\cdot \frac{(1+r)^n - 1}{r}.$$

**Пренумерандо** (платёж в начале периода):
$$PV_{\text{pre}} = C\cdot \ddot a_{n,r} = a_{n,r}(1+r),\quad FV_{\text{pre}} = s_{n,r}(1+r).$$

**Бессрочная рента (perpetuity)** при $n\to\infty$, $r>0$: $PV = C/r$ (постнумерандо).

**Растущая рента** (Gordon): $PV = C/(r-g)$ при $r>g$.

### Объединение и замена рент
**Принцип эквивалентности**: два потока равноценны, если их PV (при заданной ставке) равны.

**Объединение нескольких рент**: складываются текущие стоимости. Например, ренту длительности $n$ и ренту длительности $m$ можно заменить одной рентой длительности $\max(n,m)$ с эквивалентным платежом, найденным из равенства PV.

**Замена** одной ренты на другую: подбираем параметры новой ренты ($C'$, $n'$, ставка) так, чтобы PV нового потока равнялась PV исходного.

### Инвестиционные потоки
$C_0 < 0$ — начальная инвестиция, $C_1,\dots,C_n$ — будущие поступления (могут быть как положительными, так и отрицательными).

Критерии оценки:
- **NPV**: $\sum C_k(1+r)^{-k}$. Принимаем при $NPV>0$.
- **PI** (Profitability Index): $PI = PV(\text{доходов})/|C_0|$. Принимаем при $PI>1$.
- **Срок окупаемости** (Payback Period) — момент, когда накопленный поток выходит в плюс. Простой и дисконтированный.
- **IRR** — внутренняя норма доходности.

### Внутренняя норма доходности (IRR)
$IRR$ — ставка $r^*$, при которой $NPV(r^*) = 0$:
$$\sum_{k=0}^n \frac{C_k}{(1+r^*)^k} = 0.$$
Критерий принятия проекта: $IRR > $ стоимости капитала (барьерной ставки).

**Свойства / нюансы**:
- Если знак $C_k$ меняется один раз ($C_0<0$, далее $\ge 0$), то по правилу Декарта у уравнения в переменной $z=(1+r)^{-1}$ не больше одного положительного корня; положительная IRR ($r>0$) существует, например, если $NPV(0)>0$.
- Множественные перемены знаков → может быть несколько IRR или ни одной.
- Не учитывает масштаб проекта (для сравнения проектов лучше NPV).
- Для нестандартных потоков используют **MIRR** (modified IRR): реинвестирование по ставке $r$, заём — по другой.

### Пример
Проект: $C_0 = -1000$, $C_1 = 400$, $C_2 = 500$, $C_3 = 300$. Ставка $r=10\%$.
$$NPV = -1000 + 400/1{,}1 + 500/1{,}21 + 300/1{,}331 \approx -1000 + 363{,}6 + 413{,}2 + 225{,}4 = 2{,}2.$$
$NPV \approx 2{,}2 > 0$ — слабо прибыльный. IRR удовлетворяет $NPV(r^*)=0$ — численно $r^*\approx 10{,}1\%$.

### Подводные камни
- IRR проекта неэквивалентен реальной доходности портфеля (у IRR неявное предположение о реинвестировании по той же ставке).
- При сравнении взаимоисключающих проектов NPV — корректнее IRR (масштаб!).
- Ставка должна соответствовать риску потока (CAPM, WACC).
- Бессрочная формула $C/r$ не работает при $r=0$.
