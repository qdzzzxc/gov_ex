# Типы задач машинного обучения: классификация, регрессия, кластеризация. Обучение с учителем и без учителя. Машинное обучение как оптимизационная задача.

## Парадигмы
**Обучение с учителем (supervised):** есть размеченные пары $(x, y)$. Цель — найти $f: \mathcal{X} \to \mathcal{Y}$, обобщающуюся на новые данные.
- Классификация — дискретный $y$: бинарная, мультикласс, мультилейбл. Лоссы: cross-entropy, hinge.
- Регрессия — непрерывный $y \in \mathbb{R}$. Лоссы: MSE, MAE, Huber.

**Обучение без учителя (unsupervised):** только ${x_i}$, меток нет.
- Кластеризация — разбить на группы (k-means, DBSCAN, иерархическая, GMM, спектральная). Главная задача без учителя.
- Понижение размерности — выжать суть (PCA, t-SNE, UMAP, автокодировщики). Применяется для борьбы с проклятием размерности, визуализации, ускорения.
- Оценка плотности, поиск аномалий, поиск ассоциаций — другие задачи без учителя.

## ML как оптимизационная задача
Любая supervised задача формулируется как минимизация **эмпирического риска** — среднего лосса на выборке: $$\hat{R}(\theta) = \frac{1}{n} \sum_{i=1}^{n} L(f_\theta(x_i), y_i)$$ Обучение модели = минимизация: $$\hat{\theta} = \arg\min_\theta \hat{R}(\theta) + \lambda \Omega(\theta)$$ Это формальная постановка любой задачи обучения с учителем. Многие unsupervised тоже сводятся к оптимизации (например, k-means минимизирует сумму квадратов расстояний внутри кластеров).
## Регуляризация $\Omega(\theta)$
Штраф на сложность параметров, защита от переобучения:
- L1 (Lasso): $|\theta|_1$ — занулят неважные веса (feature selection).
- L2 (Ridge): $|\theta|_2^2$ — сжимает все веса.
- Dropout — стохастическая регуляризация для нейросетей.
## Эмпирический vs истинный риск
**Истинный риск** $R(\theta) = \mathbb{E}_{(x, y) \sim P} L(f_\theta(x), y)$ — то, что мы хотим минимизировать (ожидание лосса на всём распределении $P$). Проблема: $P$ неизвестно, считаем $\hat R$ на выборке как приближение $R$. Если выборка маленькая или модель сложная — $\hat R$ мал, но $R$ велик → переобучение. Разница $|R - \hat R|$ ограничивается теорией обобщения (VC dimension, Радемахер).
## Способы решения
1. Аналитический (closed-form) — линейная регрессия: $\hat \theta = (X^\top X)^{-1} X^\top y$.
2. Градиентные методы — SGD, momentum, Adam. Для гладких функций. Стандарт для нейросетей.
3. Специализированные — EM для GMM и скрытых переменных, координатный спуск для Lasso, жадные алгоритмы для деревьев и бустинга.
## Связь компонентов

|компонент|что|
|---|---|
|лосс $L$|что минимизируем|
|гипотеза $\mathcal{F}$|какие функции рассматриваем|
|регуляризация $\Omega$|контроль сложности|
|алгоритм оптимизации|как находим $\hat \theta$|
|метрика на тесте|что измеряем (может ≠ лоссу)|
|Лосс и метрика отличаются: лосс должен быть дифференцируемым (для градиента), метрика — отражать бизнес-задачу. Обучаем через cross-entropy, оцениваем через AUC-ROC, F1, recall.||
