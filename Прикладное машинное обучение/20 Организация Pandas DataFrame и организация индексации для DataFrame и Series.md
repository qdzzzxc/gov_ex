# Организация Pandas DataFrame и организация индексации для DataFrame и Series. Операция GroupBy в Pandas DataFrame и реализация в ней подхода «разбиение, применение и объединение».

## Series
Одномерный массив + именованный индекс (любого типа: int, str, datetime, MultiIndex). Главное отличие от numpy: операции выравниваются по меткам индекса, а не по позиции.
```python
s = pd.Series([10, 20, 30], index=['a', 'b', 'c'])
s['a']      # по метке
s.iloc[0]   # позиционно
s1 + s2     # выравнивание по индексу, NaN для несовпадающих
```
## DataFrame
Двумерный: набор Series-столбцов с общим индексом строк + индекс столбцов. Хранение под капотом: через `BlockManager` — данные хранятся по столбцам, столбцы одного `dtype` объединяются в общий NumPy-блок. Поэтому операции по столбцам быстрые (векторизация), по строкам — медленные. Атрибуты: `df.index`, `df.columns`, `df.dtypes`, `df.values`.
## Индексация
`.iloc[i, j]` — позиционная, только целые. Срезы НЕ включают правый конец (как обычный Python). `.loc['a', 'col']` — по меткам. Срезы ВКЛЮЧАЮТ правый конец (особенность Pandas).
```python
df.iloc[1:3]   # строки 1, 2
df.loc[1:3]    # строки 1, 2, 3
```
Булева и fancy:
```python
df.loc[df['age'] > 27, ['name', 'city']]
df.iloc[[0, 2]]
```
`df['col']` возвращает Series, `df[['col']]` — DataFrame с одним столбцом. MultiIndex — иерархический индекс (несколько уровней) для панельных данных и многоуровневых агрегаций; доступ через `df.loc[('Moscow', 'Anna')]` или `df.xs('Anna', level='name')`.
## GroupBy: Split-Apply-Combine
Парадигма Wickham (2011):
1. Split — разбить DataFrame по значениям ключа.
2. Apply — применить функцию к каждой группе.
3. Combine — собрать результаты обратно.
```python
g = df.groupby('city')   # ленивый, ничего не считает
```
## Три типа apply

|тип|возвращает|размер результата|
|---|---|---|
|`agg`|агрегат|одна строка на группу|
|`transform`|преобразование|как исходный (все строки)|
|`filter`|подмножество групп|меньше, но группы целые|
|Aggregation — сжимает группу в одно значение:|||
```python
g['age'].mean()
g.agg({'age': 'mean', 'salary': 'sum'})
```
Transformation — для нормализации внутри группы, рангов, кумулятивных операций:
```python
df['z'] = g['salary'].transform(lambda s: (s - s.mean()) / s.std())
```
Filtration — отбрасывает группы целиком по условию:
```python
g.filter(lambda d: len(d) >= 10)
```
## Реализация GroupBy
- Pandas строит индекс групп через хеширование ключей.
- Встроенные агрегаты (`mean, sum, count, std, quantile`) — C/Cython реализация, очень быстро.
- `apply` с Python-функцией медленный — Python-цикл по группам.
- Группировка по нескольким ключам даёт Series или DataFrame с MultiIndex.
