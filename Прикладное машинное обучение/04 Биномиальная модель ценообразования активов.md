# Биномиальная модель ценообразования активов. Оценка опционов с использованием биномиальной модели. Модель Кокса-Росса-Рубинштейна.

## TL;DR
**Биномиальная модель** дискретизирует динамику цены: за один шаг $S\to uS$ или $S\to dS$ ($u>1>d>0$). В отсутствие арбитража и при ставке $r$ существует **риск-нейтральная вероятность** $p^* = (e^{r\Delta t} - d)/(u-d)$, в которой ожидаемая доходность $S$ равна $r$. Стоимость производного инструмента — дисконтированное риск-нейтральное ожидание выплат: $V_0 = e^{-rT}\mathbb E^*[V_T]$. **Модель Кокса–Росса–Рубинштейна (CRR)**: $u = e^{\sigma\sqrt{\Delta t}}$, $d = 1/u$. При $N\to\infty$ цена европейского опциона сходится к формуле Блэка–Шоулза. Для американских опционов на каждом узле выбирается max(intrinsic, continuation).

## Развёрнуто

### Однопериодная биномиальная модель
Цена $S_0$ за один шаг $\Delta t$:
$$S_1 = \begin{cases} u S_0 & \text{с вероятностью } p \\ d S_0 & \text{с вероятностью } 1-p\end{cases}$$
Безрисковая ставка $r$ (непрерывно начисляемая); требуем $d < e^{r\Delta t} < u$ — условие отсутствия арбитража.

**Выплата производного** на шаге 1: $V_1^u, V_1^d$. Хеджирующий портфель: купим $\Delta$ акций и положим $B$ на счёт. Подберём так, чтобы воспроизвести $V_1$:
$$\Delta uS_0 + Be^{r\Delta t} = V_1^u,\quad \Delta dS_0 + Be^{r\Delta t} = V_1^d.$$
Решение:
$$\Delta = \frac{V_1^u - V_1^d}{(u-d)S_0},\quad B = \frac{uV_1^d - dV_1^u}{(u-d)e^{r\Delta t}}.$$
Стоимость портфеля сейчас:
$$V_0 = \Delta S_0 + B = e^{-r\Delta t}\bigl[p^*\,V_1^u + (1-p^*)\,V_1^d\bigr],$$
$$\boxed{p^* = \frac{e^{r\Delta t} - d}{u - d}.}$$
$p^*$ — **риск-нейтральная** вероятность; не путать с реальной вероятностью $p$.

### Многопериодная модель
$N$ шагов. Цена в узле $(n, j)$ ($n$ — шаг, $j$ — число «вверх»): $S_{n,j} = S_0\,u^j\,d^{n-j}$.

**Цена опциона** считается обратной индукцией:
1. На последнем шаге $T = N\Delta t$: $V_{N,j} = $ выплата опциона при $S_{N,j}$.
2. Для $n = N-1, \dots, 0$:
$$V_{n,j} = e^{-r\Delta t}\bigl[p^*\,V_{n+1,j+1} + (1-p^*)\,V_{n+1,j}\bigr].$$
3. $V_0 = V_{0,0}$.

Для **американских опционов** на каждом шаге берётся максимум между внутренней стоимостью (немедленным исполнением) и стоимостью продолжения:
$$V_{n,j} = \max\bigl(\text{intrinsic}(S_{n,j}),\ e^{-r\Delta t}\bigl[p^*V_{n+1,j+1} + (1-p^*)V_{n+1,j}\bigr]\bigr).$$

### Модель Кокса–Росса–Рубинштейна (CRR)
Калибровка $u, d$ по волатильности $\sigma$:
$$u = e^{\sigma\sqrt{\Delta t}},\quad d = 1/u,\quad p^* = \frac{e^{r\Delta t} - d}{u - d}.$$
$\Delta t = T/N$. Для больших $N$:
- $\ln(S_T/S_0)$ имеет в риск-нейтральной мере дисперсию $\approx \sigma^2 T$ и среднее $\approx (r - \sigma^2/2)T$;
- цена европейского опциона сходится к формуле Блэка–Шоулза при $N\to\infty$.

Альтернативная калибровка (Jarrow–Rudd): $u, d$ выбираются так, чтобы $p^* = 1/2$.

### Пример (одношаговый колл-опцион)
$S_0 = 100$, $u=1{,}1$, $d=0{,}9$, $r=5\%$, $\Delta t=1$, цена страйка $K=100$.
$p^* = (e^{0{,}05}-0{,}9)/(1{,}1-0{,}9) = (1{,}0513-0{,}9)/0{,}2 = 0{,}7563$.
Выплаты: $V^u = \max(110-100,0)=10$, $V^d = 0$.
$$C_0 = e^{-0{,}05}(0{,}7563\cdot 10 + 0{,}2437\cdot 0) = 0{,}9512\cdot 7{,}563 \approx 7{,}19.$$

### Связь с теоремами финансовой математики
- **Первая фундаментальная теорема**: рынок без арбитража $\iff$ существует риск-нейтральная мера.
- **Вторая фундаментальная теорема**: рынок полон $\iff$ риск-нейтральная мера единственна. Однопериодная биномиальная модель полна (одна риск-нейтральная вероятность).

### Применение
- Экзотические опционы (барьерные, азиатские) при адаптации структуры дерева.
- Американские опционы: закрытая европейская формула Блэка–Шоулза напрямую не применяется, нужны дерево/PDE/численные схемы; исключение — американский call без дивидендов, равный европейскому.
- Опционы на дивидендные акции (модификация $u, d$ или сетка с дивидендами).
- Реальные опционы в корпоративных финансах.

### Подводные камни
- Параметры $u, d$ выбираются произвольно — разные калибровки дают разные численные результаты при одинаковом пределе.
- Для барьерных опционов нужно следить, чтобы граница точно попадала в узлы дерева, иначе появляется погрешность.
- При $N$ малом риск-нейтральная вероятность может быть вне $[0,1]$ — модель некорректна.
- Биномиальная модель — однофакторная, не учитывает стохастической волатильности и скачков.
