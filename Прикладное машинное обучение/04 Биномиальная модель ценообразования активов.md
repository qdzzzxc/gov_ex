# Биномиальная модель ценообразования активов. Оценка опционов с использованием биномиальной модели. Модель Кокса-Росса-Рубинштейна.

### Опцион

**Право** (не обязанность) купить/продать актив по цене $K$ (strike) в момент $T$.

**Колл-опцион выплата:** $V_T = \max(S_T - K, 0)$.

Задача: оценить $V_0$ — стоимость опциона сегодня.

### Однопериодная биномиальная модель

За шаг $\Delta t$ цена меняется: $$S \to \begin{cases} uS & \text{с реальной вероятностью } p \ dS & \text{с } 1 - p \end{cases}$$

**Условие отсутствия арбитража:** $d < e^{r \Delta t} < u$.

### Идея репликации

Создаём портфель $\Delta$ акций + $B$ на счёте, точно воспроизводящий выплаты опциона в обоих состояниях. Цена опциона = цена портфеля (иначе арбитраж).

Из системы $\Delta uS_0 + Be^{r\Delta t} = V^u$, $\Delta dS_0 + Be^{r\Delta t} = V^d$ получаем: $$\Delta = \frac{V^u - V^d}{(u-d)S_0}.$$

### Риск-нейтральная вероятность

$$p^* = \frac{e^{r\Delta t} - d}{u - d}.$$

**Свойство:** в этой мере ожидаемая доходность акции равна безрисковой ставке $r$. Реальная $p$ из формулы цены **полностью выпадает** — цена опциона не зависит от того, насколько часто акция реально растёт.

### Формула цены (центральная)

$$V_0 = e^{-r\Delta t} \left[ p^* V^u + (1 - p^*) V^d \right].$$

"Дисконтированное риск-нейтральное ожидание выплаты."

### Многопериодная модель

$N$ шагов, дерево с рекомбинацией. Цена в узле $(n, j)$: $$S_{n, j} = S_0 \cdot u^j \cdot d^{n-j},$$ где $j$ — число шагов "вверх".

**Обратная индукция:**

1. На последнем шаге $V_{N, j} = $ терминальная выплата.
2. Для $n = N-1, \dots, 0$: $$V_{n, j} = e^{-r\Delta t} \left[ p^* V_{n+1, j+1} + (1 - p^*) V_{n+1, j} \right].$$
3. $V_0 = V_{0, 0}$.

### Американские опционы

На каждом узле берём максимум между исполнить сейчас и продолжить: $$V_{n, j} = \max\left( \text{intrinsic}(S_{n, j}), \ e^{-r\Delta t}[p^* V_{n+1, j+1} + (1-p^*) V_{n+1, j}] \right).$$

Блэк-Шоулз для американских не работает — биномиальная модель основной инструмент.

### Модель Кокса-Росса-Рубинштейна (CRR)

Калибровка $u, d$ через волатильность $\sigma$: $$u = e^{\sigma \sqrt{\Delta t}}, \quad d = 1/u, \quad p^* = \frac{e^{r\Delta t} - d}{u - d}.$$

$\Delta t = T/N$.

**Свойство:** $\ln(S_T / S_0)$ в риск-нейтральной мере имеет дисперсию $\approx \sigma^2 T$ и среднее $\approx (r - \sigma^2/2) T$ — согласовано с геометрическим броуновским движением.

**Альтернатива (Jarrow-Rudd):** $u, d$ выбираются так, чтобы $p^* = 1/2$.

### Связь с Блэком-Шоулзом

При $N \to \infty$ цена европейского опциона в CRR сходится к формуле Блэка-Шоулза: $$C_0 = S_0 \Phi(d_1) - K e^{-rT} \Phi(d_2).$$

Биномиальная модель — **дискретный аналог** Блэка-Шоулза. При $N = 50$-$1000$ численная точность практически неотличима.

### Зачем биномиальная при наличии Блэка-Шоулза
1. **Американские опционы** — Блэк-Шоулз не работает.
2. **Опционы на дивидендные акции** — модификация $u, d$ или дерево с дивидендами.
3. **Экзотические опционы** (барьерные, азиатские) — адаптация структуры дерева.
4. **Реальные опционы** в корпоративных финансах.

### Фундаментальные теоремы
- **Первая:** рынок без арбитража $\iff$ существует риск-нейтральная мера.
- **Вторая:** рынок полон $\iff$ риск-нейтральная мера единственна. Однопериодная биномиальная модель полна.

### Пример

$S_0 = 100$, $u = 1.1$, $d = 0.9$, $r = 5\%$, $\Delta t = 1$, $K = 100$.

$p^* = (e^{0.05} - 0.9) / 0.2 = 0.7563$.

$V^u = 10$, $V^d = 0$.

$C_0 = e^{-0.05}(0.7563 \cdot 10 + 0.2437 \cdot 0) \approx 7.19$.

### Главное на экзамене

1. **Опцион** — право, выплата $\max(S_T - K, 0)$ для колла.
2. **Идея репликации** — портфель акция + вклад, воспроизводящий опцион.
3. **Риск-нейтральная вероятность** $p^* = (e^{r\Delta t} - d)/(u - d)$ — реальная $p$ не нужна.
4. **Формула цены** $V_0 = e^{-r\Delta t}[p^* V^u + (1-p^*) V^d]$.
5. **Многопериодная модель** через обратную индукцию.
6. **CRR**: $u = e^{\sigma\sqrt{\Delta t}}$, $d = 1/u$. Сходится к Блэку-Шоулзу.
7. **Американские опционы** — max(intrinsic, continuation) в каждом узле. Главное преимущество биномиальной.
