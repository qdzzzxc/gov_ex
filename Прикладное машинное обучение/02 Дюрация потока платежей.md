# Дюрация потока платежей. Дюрация облигации и ее свойства. Выпуклость облигаций. Хеджирование риска изменений процентной ставки. Теорема об иммунизации.

## TL;DR
**Дюрация Маколея** $D = \sum t_k\,PV(C_k)/PV$ — средневзвешенный срок поступления платежей; для обычных положительных потоков это «эффективный срок до получения денег». **Модифицированная дюрация** $D^* = D/(1+r)$ — мера чувствительности цены к изменению ставки: $\Delta P/P\approx -D^*\Delta r$. **Выпуклость** $C = \sum t_k(t_k+1) PV(C_k)/(PV(1+r)^2)$ — поправка второго порядка: $\Delta P/P\approx -D^*\Delta r + \tfrac12 C(\Delta r)^2$. Хеджирование процентного риска — иммунизация: PV активов = PV обязательств и дюрация активов = дюрация обязательств. **Теорема Самуэльсона/Реддингтона**: такое согласование защищает от малых параллельных сдвигов кривой ставок.

## Развёрнуто

### Дюрация Маколея
Поток платежей $\{(t_k, C_k)\}$, ставка $r$:
$$P = \sum_{k} \frac{C_k}{(1+r)^{t_k}}.$$
**Дюрация Маколея**:
$$D = \frac{\sum_k t_k\cdot \frac{C_k}{(1+r)^{t_k}}}{\sum_k \frac{C_k}{(1+r)^{t_k}}} = \frac{\sum t_k\,PV(C_k)}{P}.$$
Интерпретация: средневзвешенный срок платежей, веса — доли PV.

Свойства:
- при положительных приведённых платежах $D$ лежит между минимальным и максимальным $t_k$;
- для бескупонной облигации $D = T$ (срок до погашения);
- для обычной купонной облигации $D < T$;
- $D$ убывает с ростом ставки $r$ и купона.

### Модифицированная дюрация
$$D^* = \frac{D}{1+r}.$$
Связь с чувствительностью цены:
$$\frac{dP}{dr} = -D^*\cdot P\Rightarrow \frac{\Delta P}{P}\approx -D^*\,\Delta r.$$
Для непрерывного дисконтирования $D^* = D$.

### Выпуклость (Convexity)
Поправка второго порядка:
$$C = \frac{1}{P}\sum_k t_k(t_k+1)\frac{C_k}{(1+r)^{t_k+2}} = \frac{1}{P(1+r)^2}\sum_k t_k(t_k+1)PV(C_k).$$
Тейлоровская формула:
$$\frac{\Delta P}{P}\approx -D^*\Delta r + \tfrac12 C(\Delta r)^2.$$
Выпуклость **положительна** для обычных облигаций — это «бонус»: при росте ставок цена падает медленнее, при падении растёт быстрее, чем предсказывает линейная аппроксимация по дюрации.

### Дюрация облигации и её свойства
Для облигации с купоном $c$ (раз в период), номиналом $F$, $T$ периодов:
$$P = \sum_{k=1}^T \frac{c F}{(1+r)^k} + \frac{F}{(1+r)^T}.$$
Дюрация считается по общей формуле.

Свойства:
- $D$ растёт с ростом срока до погашения;
- $D$ убывает с ростом купонной ставки и доходности к погашению (YTM);
- бескупонная облигация — экстремальная по дюрации (равна сроку).

### Хеджирование процентного риска и иммунизация
**Задача**: имеем обязательства (либо известные будущие платежи, либо рыночную позицию) и активы. Мы хотим, чтобы изменение ставок не повлияло на чистый собственный капитал ($PV(\text{актива}) - PV(\text{обязательств})$).

**Стратегия иммунизации (Реддингтон)**: подобрать активы так, чтобы:
1. $PV(A) = PV(L)$;
2. $D(A) = D(L)$;
3. долларовая выпуклость активов больше выпуклости обязательств (или, при равных PV, $C(A) \ge C(L)$).

При малых параллельных сдвигах кривой ставок выпуклость активов «работает в плюс», и чистая позиция остаётся неотрицательной.

### Теорема об иммунизации (Реддингтон, Фишер–Вэйл)
Если $PV(A)=PV(L)$, $D(A) = D(L)$ и долларовая выпуклость активов не меньше выпуклости обязательств, то для **любого** малого параллельного сдвига $\Delta r$:
$$PV(A) - PV(L) \ge 0 + O((\Delta r)^2),$$
причём в первом порядке изменение нулевое, во втором — неотрицательное. То есть позиция «иммунизирована» от малых сдвигов.

Ограничения:
- защита от **параллельных** сдвигов; неравномерные изменения формы кривой требуют key-rate duration;
- работает локально (малые $\Delta r$);
- требует периодической ребалансировки (с течением времени дюрации меняются).

### Пример
3-летняя облигация, купон 8% (ежегодно), номинал 1000, YTM = 10%. Платежи: $80, 80, 1080$.
PV платежей: $80/1{,}1 + 80/1{,}21 + 1080/1{,}331 = 72{,}73 + 66{,}12 + 811{,}42 = 950{,}27$.
$D = (1\cdot 72{,}73 + 2\cdot 66{,}12 + 3\cdot 811{,}42)/950{,}27 = (72{,}73 + 132{,}24 + 2434{,}26)/950{,}27 = 2{,}777$ года.
$D^* = 2{,}777/1{,}1 \approx 2{,}52$.

При $\Delta r = +0{,}5\%$: $\Delta P/P\approx -2{,}52\cdot 0{,}005 \approx -1{,}26\%$. Цена упадёт примерно на 12 рублей.

### Подводные камни
- Дюрация — линейная аппроксимация: при больших $\Delta r$ ошибка растёт, поэтому добавляют выпуклость.
- Кривая доходности редко сдвигается параллельно — для серьёзных портфелей используют **key-rate duration** (чувствительность к каждой точке кривой).
- При наличии встроенных опционов (callable, putable) — нужны эффективная дюрация/выпуклость, считаемые через перезагрузку модели.
