# Классификация и характеристика различных видов сетей: безмасштабные сети, сетии тесного мира, фрактальные сети.

## Базовые характеристики
Граф $G = (V, E)$ — узлы и рёбра. Три главных параметра:
- $P(k)$ — распределение степеней (главная характеристика).
- $L$ — средняя длина пути.
- $C$ — коэффициент кластеризации (доля треугольников).
## Случайные сети Эрдёша-Реньи (ER) — базовый контекст
Механизм: каждое ребро существует с вероятностью $p$ независимо от других. Никакой структуры, чистая случайность.
- $P(k)$ — пуассоновское.
- $L \sim \ln n$ — короткие пути.
- $C \sim p$ — низкая кластеризация.
- Нет хабов. Не моделирует реальные сети — в них кластеризация выше и есть хабы.
## Сети тесного мира (small-world)
Главное: одновременно малая средняя длина пути $L$ и высокая кластеризация $C$. Феномен «шести рукопожатий» (эксперимент Милгрэма 1967). Механизм построения (модель Уоттса-Строгаца):
1. Стартуем с кольцевой решётки: $n$ узлов по окружности, каждый соединён с $k$ ближайшими соседями.
2. Каждое ребро с вероятностью $p$ перевешиваем — отрываем один конец и переподключаем в случайный узел. Локальная структура решётки даёт высокую кластеризацию. Несколько случайных шорткатов резко уменьшают $L$, не разрушая локальную плотность связей. Идея: «локальные связи + редкие дальние знакомые». Встречается в соцсетях, нейронных сетях мозга, энергосистемах.
## Безмасштабные сети (scale-free)
Главное: распределение степеней — степенное: $$P(k) \propto k^{-\gamma}, \quad \gamma \in (2, 3)$$ Хабы — несколько узлов с гигантской степенью, большинство с маленькой. Имя «безмасштабное» — степенная функция самоподобна при масштабировании, нет характерного размера степени. Механизм построения (модель Барабаши-Альберт, preferential attachment):
1. Стартуем с малой связной сети.
2. Добавляем узлы по одному; каждый новый соединяется с существующими пропорционально их степени: $P(\text{связь с } i) = k_i / \sum_j k_j$. «Богатые богатеют» — узел с большой степенью получает новые рёбра чаще. Положительная обратная связь даёт степенное распределение $P(k) \propto k^{-3}$. Свойства:
- $L \sim \ln \ln n$ — ультракороткие пути через хабы.
- Устойчивы к случайным удалениям, уязвимы к атакам на хабы.
- Нет порога эпидемий при $\gamma \le 3$. Встречается в Интернете, WWW, цитировании, соцсетях, метаболических сетях. Степенное распределение проверяется не визуально, а статистическими тестами — на лог-лог графике любая ограниченная выборка может выглядеть линейной.
## Фрактальные сети
Главное: самоподобие на разных масштабах. Модульная структура из подмодулей внутри подмодулей. Формально: покрытие «коробками» радиуса $\ell_B$: $$N_B(\ell_B) \propto \ell_B^{-d_B}$$ $d_B$ — фрактальная размерность, находится методом box-covering. Механизм построения (иерархический рост, Сонг-Хавлин-Максе): на каждом шаге каждый узел заменяется маленькой копией структуры. Рекурсивно получается самоподобная сеть «модули в модулях». Идея: иерархическая модульная организация (биология: клетка → ткань → орган; корпорация: команда → отдел → компания). Многие реальные сети одновременно scale-free и фрактальные (биология, WWW). В чистой модели Барабаши-Альберт фрактальной структуры нет — хабы соединены и разрушают модульность; в фрактальных хабы изолированы внутри своих модулей.
