# Классификация и характеристика различных видов сетей: безмасштабные сети, сетии тесного мира, фрактальные сети.

## TL;DR — Классификация сетей

### Базовые характеристики
Граф $G = (V, E)$, узлы и рёбра. Три главных параметра:
- **$P(k)$** — распределение степеней (главная характеристика).
- **$L$** — средняя длина пути.
- **$C$** — коэффициент кластеризации (доля треугольников).

### Случайные сети Эрдёша-Реньи (ER) — базовый контекст
Каждое ребро существует с вероятностью $p$ независимо.
- $P(k)$ — **пуассоновское**.
- $L \sim \ln n$ — короткие пути.
- $C \sim p$ — низкая кластеризация.
- **Нет хабов**.

Не моделирует реальные сети — в них кластеризация выше и есть хабы.

### Сети тесного мира (small-world)
**Главное:** одновременно
- Малая средняя длина пути $L$.
- Высокая кластеризация $C$.

Феномен "шести рукопожатий".

**Модель Уоттса-Строгаца:** регулярная решётка + случайное перевешивание рёбер с вероятностью $p$. При $p \in (0, 1)$ — small-world.

**Где:** соцсети, нейронные сети мозга, энергосистемы.

### Безмасштабные сети (scale-free)
**Главное:** распределение степеней — **степенное**: $$P(k) \propto k^{-\gamma}, \quad \gamma \in (2, 3).$$

**Хабы** — несколько узлов с гигантской степенью. Большинство — с маленькой.

Имя "безмасштабное" — степенная функция самоподобна при масштабировании, нет характерного размера степени.

**Модель Барабаши-Альберт (preferential attachment):**

1. Стартуем с малой связной сети.
2. Добавляем узлы; каждый новый соединяется с существующими **пропорционально их степени** ("богатые богатеют").

Получается $P(k) \propto k^{-3}$.

**Свойства:**
- $L \sim \ln \ln n$ — ультракороткие пути через хабы.
- Устойчивы к случайным удалениям.
- Уязвимы к атакам на хабы.
- Нет порога эпидемий при $\gamma \le 3$.

**Где:** Интернет, WWW, цитирование, соцсети, метаболические сети.

### Фрактальные сети

**Главное:** **самоподобие** на разных масштабах. Модульная структура из подмодулей.

**Формально:** покрытие "коробками" радиуса $\ell_B$: $$N_B(\ell_B) \propto \ell_B^{-d_B}.$$

$d_B$ — **фрактальная размерность**, находится методом **box-covering**.

**Связь:** многие реальные сети одновременно scale-free и фрактальные (биология, WWW). В чистой модели Барабаши-Альберт фрактальной структуры нет — хабы соединены, разрушают модульность.

### Сравнительная таблица

|Тип|$P(k)$|$L$|$C$|Хабы|
|---|---|---|---|---|
|ER|Пуассон|$\ln n$|низкий|нет|
|Small-world (WS)|узкое|$\ln n$|**высокий**|нет|
|Scale-free (BA)|$k^{-\gamma}$|$\ln \ln n$|низкий|**есть**|
|Фрактальная|$k^{-\gamma}$|$n^{1/d_B}$|модульный|в модулях|

### Применения
- Анализ устойчивости (Интернет, энергосети).
- Эпидемиология — распространение болезней.
- Маркетинг — вирусное продвижение через хабы.
- Биологические сети.
- Алгоритмы поиска и навигации.

### Подводные камни
- Степенное распределение надо проверять не визуально, а строгими тестами (Кляйм-Ньюман-Шалицев).
- Реальные сети — гибрид (scale-free + small-world + community structure).
- При $\gamma \le 2$ нет конечного среднего степени — теория ломается.
- Малые сети могут давать псевдо-степенное поведение из-за конечности.

### Главное на экзамене
1. **Три параметра** сети: $P(k)$, $L$, $C$.
2. **Small-world**: малое $L$ + высокое $C$. Модель Уоттса-Строгаца.
3. **Scale-free**: $P(k) \propto k^{-\gamma}$, есть хабы. Модель Барабаши-Альберт через preferential attachment.
4. **Фрактальные**: самоподобие на масштабах, фрактальная размерность $d_B$.
5. **Сравнение с ER** — почему классические случайные сети не подходят для реальности.
