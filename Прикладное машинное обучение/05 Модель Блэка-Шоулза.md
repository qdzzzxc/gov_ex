# Модель Блэка-Шоулза. Уравнение Блэка-Шоулза. Оценка производных финансовых инструментов с использованием модели Блэка-Шоулза.

## TL;DR
**Модель Блэка–Шоулза**: цена базового актива следует геометрическому броуновскому движению $dS_t = \mu S_t\,dt + \sigma S_t\,dW_t$. Стоимость производного $V(S,t)$ удовлетворяет дифференциальному уравнению **Блэка–Шоулза**: $\partial_t V + rS\,\partial_S V + \tfrac12\sigma^2 S^2\,\partial_S^2 V - rV = 0$. Для европейского call: $C = S_0\Phi(d_1) - K e^{-rT}\Phi(d_2)$, $d_{1,2} = (\ln(S_0/K) + (r\pm\sigma^2/2)T)/(\sigma\sqrt T)$; для put — через паритет $C - P = S_0 - Ke^{-rT}$. Допущения сильные: непрерывная торговля, постоянная $\sigma$, нет дивидендов и трансакционных издержек, нормальность лог-доходностей.

## Развёрнуто

### Допущения модели
1. Цена базового актива $S_t$: геометрическое броуновское движение (GBM)
$$dS_t = \mu S_t\,dt + \sigma S_t\,dW_t,$$
$\mu$ — снос (drift), $\sigma$ — волатильность, $W_t$ — стандартный винеровский процесс.
2. Постоянная безрисковая ставка $r$.
3. Постоянная волатильность $\sigma$.
4. Возможна непрерывная торговля без трансакционных издержек, налогов, ограничений на короткие продажи.
5. Базовый актив не платит дивидендов (есть расширения).
6. Рынок без арбитража.
7. Для закрытой формулы ниже рассматривается европейский производный инструмент (исполнение только в $T$); сама рыночная модель Блэка–Шоулза шире и может использоваться в численных схемах для других выплат.

При этих допущениях $\ln(S_T/S_0)\sim N((\mu-\sigma^2/2)T, \sigma^2 T)$ — **логнормальное** распределение цен.

### Уравнение Блэка–Шоулза
Рассматриваем самофинансируемый портфель $\Pi = V - \Delta\,S$ с $\Delta = \partial V/\partial S$. По формуле Ито:
$$dV = \left(\partial_t V + \mu S\,\partial_S V + \tfrac12\sigma^2 S^2\,\partial_S^2 V\right)dt + \sigma S\,\partial_S V\,dW.$$
$d\Pi = dV - \Delta\,dS$, при выборе $\Delta = \partial_S V$ стохастический член $dW$ исчезает. По безарбитражности $d\Pi = r\Pi\,dt$:
$$\partial_t V + \tfrac12\sigma^2 S^2\,\partial_S^2 V + rS\,\partial_S V - rV = 0.$$
Это **уравнение Блэка–Шоулза** (PDE параболического типа). Граничные условия — выплата опциона при $t=T$.

### Формула Блэка–Шоулза (европейский колл)
Решение для $V(S,T) = \max(S-K, 0)$:
$$\boxed{C(S_0, t=0) = S_0\Phi(d_1) - K e^{-rT}\Phi(d_2),}$$
$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt T},\quad d_2 = d_1 - \sigma\sqrt T.$$
$\Phi$ — функция стандартного нормального распределения.

**Европейский put** (через паритет $C - P = S_0 - Ke^{-rT}$):
$$P = K e^{-rT}\Phi(-d_2) - S_0\Phi(-d_1).$$

**Интерпретация**: $\Phi(d_1) = \partial C/\partial S$ — дельта (хеджевый коэффициент); $\Phi(d_2)$ — риск-нейтральная вероятность исполнения опциона.

### Греки (чувствительности)
$$\Delta = \partial_S C = \Phi(d_1),\quad \Gamma = \partial^2_S C = \frac{\phi(d_1)}{S_0\sigma\sqrt T},$$
$$\nu = \partial_\sigma C = S_0\phi(d_1)\sqrt T \quad (\text{vega}),\quad \Theta = \partial_t C,\quad \rho = \partial_r C.$$
Используются для хеджирования и оценки чувствительности.

### Расширения
- **Дивидендный yield** $q$: $C = S_0 e^{-qT}\Phi(d_1) - K e^{-rT}\Phi(d_2)$, $d_1 = (\ln(S_0/K) + (r-q+\sigma^2/2)T)/(\sigma\sqrt T)$.
- **Опционы на валюту (Garman–Kohlhagen)** — заменяем $q$ на иностранную ставку.
- **Опционы на фьючерсы (Black model)**.
- Стохастическая волатильность (Heston), скачки (Merton, Kou) — выходят за рамки BS.

### Применение
- Оценка ванильных опционов (call/put) — стандарт индустрии.
- Расчёт **подразумеваемой волатильности** (implied volatility): из рыночной цены опциона решаем уравнение $C^{\text{рын}} = C^{\text{BS}}(\sigma)$. Полученная $\sigma$ — параметр волатильности модели Блэка–Шоулза, подразумеваемый рыночной ценой; это не обязательно несмещённый прогноз будущей реализованной волатильности.
- Дельта-хеджирование позиций по опционам.
- Real options — оценка управленческих опций в корпоративных финансах.
- Капитал у банков и страховщиков — оценка встроенных опций в страховых продуктах.

### Пример
$S_0 = 100$, $K=100$, $r=5\%$, $\sigma = 20\%$, $T=1$ год.
$d_1 = (0 + (0{,}05+0{,}02))/0{,}2 = 0{,}35$, $d_2 = 0{,}35 - 0{,}2 = 0{,}15$.
$\Phi(0{,}35)\approx 0{,}6368$, $\Phi(0{,}15)\approx 0{,}5596$.
$C = 100\cdot 0{,}6368 - 100\,e^{-0{,}05}\cdot 0{,}5596 = 63{,}68 - 95{,}12\cdot 0{,}5596 \approx 63{,}68 - 53{,}23 \approx 10{,}45$.

### Подводные камни
- Постоянная $\sigma$ нереалистична — рынок показывает «улыбку волатильности» (smile/skew).
- Лог-нормальность не учитывает тяжёлые хвосты и скачки — катастрофы недооцениваются.
- В рамках своих допущений формула точна; около экспирации и при больших $\sigma\sqrt T$ особенно заметны численная чувствительность, резкие изменения греков и нереалистичность постоянной волатильности.
- Нет аналитики для американских опционов на дивидендные акции (нужна численная схема).
