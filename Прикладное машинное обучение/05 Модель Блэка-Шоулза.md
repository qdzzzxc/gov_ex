# Модель Блэка-Шоулза. Уравнение Блэка-Шоулза. Оценка производных финансовых инструментов с использованием модели Блэка-Шоулза.

### Модель

**Цена базового актива** следует геометрическому броуновскому движению (GBM): $$dS_t = \mu S_t , dt + \sigma S_t , dW_t.$$

- $\mu$ — drift (тренд).
- $\sigma$ — волатильность.
- $W_t$ — стандартный винеровский процесс.

Следствие: $\ln(S_T/S_0) \sim N((\mu - \sigma^2/2) T, \sigma^2 T)$ — цена логнормальна.

### Допущения

1. Цена — GBM.
2. **Постоянная** безрисковая ставка $r$.
3. **Постоянная** волатильность $\sigma$.
4. Непрерывная торговля без трансакционных издержек, налогов, ограничений.
5. Актив не платит дивидендов.
6. Нет арбитража.
7. Опцион европейский.

### Уравнение Блэка-Шоулза

**Идея вывода:** строим хеджирующий портфель $\Pi = V - \Delta S$ с $\Delta = \partial_S V$. По формуле Ито стохастический член $dW$ исчезает при таком выборе $\Delta$. Безарбитражность даёт $d\Pi = r\Pi , dt$.

**Уравнение:** $$\partial_t V + r S \partial_S V + \tfrac{1}{2} \sigma^2 S^2 \partial_S^2 V - r V = 0.$$

Параболическое PDE для $V(S, t)$. Граничное условие — выплата опциона при $t = T$:

- Колл: $V(S, T) = \max(S - K, 0)$.
- Пут: $V(S, T) = \max(K - S, 0)$.

### Формула Блэка-Шоулза для колла

$$C_0 = S_0 \Phi(d_1) - K e^{-rT} \Phi(d_2),$$

$$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2) T}{\sigma \sqrt T}, \quad d_2 = d_1 - \sigma \sqrt T.$$

**Содержательно:**

- $S_0 \Phi(d_1)$ — ожидаемая стоимость акции при исполнении.
- $K e^{-rT} \Phi(d_2)$ — приведённая стоимость уплаты $K$, взвешенная на риск-нейтральную вероятность исполнения.
- $\Phi(d_1)$ — дельта колла.
- $\Phi(d_2)$ — риск-нейтральная вероятность исполнения опциона.

### Колл-пут паритет

$$C - P = S_0 - K e^{-rT}.$$

**Вывод:** два портфеля

- $A$: 1 колл + $K e^{-rT}$ в банке.
- $B$: 1 пут + 1 акция.

В момент $T$ оба дают $\max(S_T, K)$. По безарбитражности их стоимости сегодня равны.

**Применение:** зная цену колла из Блэка-Шоулза, цена пута: $$P = K e^{-rT} \Phi(-d_2) - S_0 \Phi(-d_1).$$

**Важно:** паритет работает только для **европейских** опционов. Для американских — только неравенства. Паритет универсален — следует из безарбитражности, не зависит от модели цены.

### Греки (упомянуть кратко)

Частные производные цены опциона по параметрам:
- $\Delta = \Phi(d_1)$ — чувствительность к $S$ (для хеджирования).
- $\Gamma$ — выпуклость по $S$.
- $\nu$ (vega) — чувствительность к $\sigma$.
- $\Theta$ — чувствительность к времени.
- $\rho$ — чувствительность к $r$.

### Применения
- **Ванильные опционы** на акции, индексы, валюту — стандарт индустрии.
- **Implied volatility** — из рыночной цены опциона извлекают рыночную оценку $\sigma$.
- **Дельта-хеджирование** позиций по опционам.

### Связь с биномиальной моделью
Блэк-Шоулз — непрерывный предел биномиальной CRR-модели при $N \to \infty$, $\Delta t \to 0$. Биномиальная — дискретная версия; они дают одинаковые цены европейских опционов.

### Расширения
- **Дивидендный yield $q$:** $C = S_0 e^{-qT} \Phi(d_1) - K e^{-rT} \Phi(d_2)$.
- **Garman-Kohlhagen** для валютных опционов.
- **Black model** для опционов на фьючерсы.
- **Heston, Merton** — стохастическая волатильность, скачки.

### Подводные камни
- **Постоянная $\sigma$** нереалистична — рынок показывает "улыбку волатильности".
- **Логнормальность** недооценивает тяжёлые хвосты и катастрофы.
- Для **американских опционов на дивидендные акции** аналитики нет — нужна биномиальная модель или численная схема.
- **Непрерывная торговля без издержек** — идеализация.

### Главное на экзамене
1. **Модель**: цена — GBM $dS = \mu S , dt + \sigma S , dW$. Допущения.
2. **Уравнение Блэка-Шоулза** — структура PDE и идея вывода через хеджирующий портфель + формулу Ито.
3. **Формула для колла**: $C = S_0 \Phi(d_1) - K e^{-rT} \Phi(d_2)$.
4. **Пут через паритет**: $C - P = S_0 - K e^{-rT}$. Вывод через два эквивалентных портфеля.
5. **Биномиальная модель → Блэк-Шоулз** при $N \to \infty$.
