# Модель Блэка-Шоулза. Уравнение Блэка-Шоулза. Оценка производных финансовых инструментов с использованием модели Блэка-Шоулза.

**Модель Блэка–Шоулза**
**1. Модель.** Цена актива — геометрическое броуновское движение: $$dS = \mu S,dt + \sigma S,dW$$ Следствие: $\ln(S_T/S_0) \sim \mathcal{N}!\left((\mu - \sigma^2/2)T,\ \sigma^2 T\right)$ — цена логнормальна.
**2. Допущения.** Постоянные $r, \sigma$; без дивидендов, издержек, арбитража; европейский опцион; непрерывная торговля.
**3. Уравнение Блэка–Шоулза.** $$\partial_t V + rS,\partial_S V + \tfrac{1}{2}\sigma^2 S^2 \partial_S^2 V - rV = 0$$

В терминах греков: $$\underbrace{\Theta}_{\text{утекает со временем}} + \underbrace{rS\Delta}_{\text{ловит дрейф}} + \underbrace{\tfrac{1}{2}\sigma^2 S^2 \Gamma}_{\text{ловит тряску}} = \underbrace{rV}_{\text{безриск. рост}}$$
Вывод: портфель $\Pi = V - \Delta S$ с $\Delta = \partial_S V$ → по формуле Ито $dW$ исчезает → безарбитражность даёт $d\Pi = r\Pi,dt$.
Зачем: уравнение — общий закон для цены любого опциона в непрерывной модели; формула для колла ниже — его частное решение для ванильного европейского случая. Для экзотических опционов то же уравнение решается численно.
**4. Формула для колла.** $$C = S_0\Phi(d_1) - Ke^{-rT}\Phi(d_2)$$ $$d_1 = \frac{\ln(S_0/K) + (r + \sigma^2/2)T}{\sigma\sqrt{T}}, \quad d_2 = d_1 - \sigma\sqrt{T}$$
Здесь $\Phi$ — функция распределения $\mathcal{N}(0,1)$, $K$ — страйк. Смысл: $\Phi(d_1) = \Delta$, $\Phi(d_2)$ — риск-нейтральная вероятность исполнения.
**5. Колл-пут паритет.** $$C - P = S_0 - Ke^{-rT}$$
Вывод: портфели «колл + $Ke^{-rT}$ в банке» и «пут + акция» оба дают $\max(S_T, K)$ в момент $T$ → по безарбитражности их цены сегодня равны.

Важно: паритет следует только из отсутствия арбитража, не зависит от модели цены (работает и в CRR, и в БШ, и в любой другой). Только для европейских опционов. Из БШ для колла + паритет → $P = Ke^{-rT}\Phi(-d_2) - S_0\Phi(-d_1)$.

**6. Связь с CRR.** БШ — непрерывный предел биномиальной модели CRR при $N \to \infty$, $\Delta t \to 0$. На европейских опционах дают одинаковые цены. CRR удобнее для американских и экзотических — там БШ-формулы нет, считают численно.
