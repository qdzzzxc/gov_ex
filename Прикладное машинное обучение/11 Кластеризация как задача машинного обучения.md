# Кластеризация как задача машинного обучения. Метод ближайших соседей (kNN) и его обобщения. Подбор числа k. Методы метрической классификации в библиотеке Scikit-Learn.

## Кластеризация — постановка
Обучение без учителя: разбить ${x_i}$ на группы похожих. Похожесть определяется метрикой (евклидова, косинусная) или ядром. Основные алгоритмы: **k-means** (минимизация WCSS, чувствителен к выбросам, любит сферические кластеры), **иерархическая** (дендрограмма, разбиение разрезанием), **DBSCAN** (по плотности, не нужно $K$, выделяет шум), **GMM** (вероятностная, soft-кластеризация через EM), **spectral** (через лапласиан графа сходства).
## kNN — метрический классификатор
Для классификации $x$: найти $k$ ближайших обучающих примеров → мажоритарное голосование. Для регрессии — среднее $y$ соседей. **Lazy:** нет обучения, вся работа на этапе предсказания. Обучающая выборка целиком используется при каждом запросе. **Метрики:** евклидова, манхэттенская, Минковского, косинусная. **Масштабирование признаков обязательно** (StandardScaler) — иначе признак с большой шкалой будет доминировать в расстоянии.
## Подбор $k$
- Малое $k$ → высокая variance (чувствительность к шуму).
- Большое $k$ → высокий bias (сглаживание границы).
- Кросс-валидация по сетке $k = 1, 3, 5, 7, 11, \dots$
- Эмпирика: $k \approx \sqrt{n}$ как старт.
- Нечётное $k$ для бинарной классификации — избежать ничьих.
## Обобщения kNN
- **Weighted kNN** — голос соседа взвешен обратно расстоянию (`weights='distance'`).
- **Radius Neighbors** — все соседи в радиусе $r$ вместо фиксированного числа.
- **Nearest Centroid** — класс ассоциирован с центроидом обучающих точек этого класса.
- **LWR (Locally Weighted Regression)** — kNN с локальной моделью вместо усреднения.
## Ускорение поиска соседей
- **KD-Tree** — бинарное дерево по осям, $O(\log n)$ в среднем для малых $d$.
- **Ball-Tree** — гиперсферические разбиения, лучше при средних $d$.
- **HNSW, FAISS, Annoy** — приближённые методы для больших $d$ и $n$, продакшен. В sklearn: `algorithm='auto' | 'kd_tree' | 'ball_tree' | 'brute'`.
## Scikit-Learn — метрическая классификация
- `KNeighborsClassifier`, `KNeighborsRegressor`.
- `RadiusNeighborsClassifier`, `RadiusNeighborsRegressor`.
- `NearestCentroid`. Кластеризация (для контекста): `KMeans`, `AgglomerativeClustering`, `DBSCAN`, `GaussianMixture`, `SpectralClustering`.
## Проклятие размерности для kNN
При большом $d$ расстояния между точками выравниваются — все точки оказываются примерно равноудалёнными. Близкие соседи становятся неотличимы от далёких → kNN теряет смысл. Решение: перед kNN — PCA, UMAP, feature selection. Для больших $n$ — индексные структуры или приближённые методы поиска.
