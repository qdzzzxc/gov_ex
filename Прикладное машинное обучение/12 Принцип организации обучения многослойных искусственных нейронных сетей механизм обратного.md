# Принцип организации обучения многослойных искусственных нейронных сетей: механизм обратного распространения ошибки дифференцируемое программирование и граф потока вычислений.

## Многослойная сеть (MLP)

$$h^{(0)} = x, \quad h^{(l)} = \sigma_l(W^{(l)} h^{(l-1)} + b^{(l)}), \quad \hat y = h^{(L)}$$ Композиция аффинных слоёв и нелинейностей $\sigma$ (ReLU, sigmoid, tanh, GELU, softmax на выходе). Без нелинейности композиция линейных слоёв = линейная функция — сеть вырождается в линейную модель. Нелинейность нужна, чтобы получить выразительность.
## Принцип обучения
Минимизация эмпирического риска: $$\hat R(\theta) = \frac{1}{n} \sum_{i=1}^n L(f_\theta(x_i), y_i)$$ $L$ — функция потерь (MSE, cross-entropy). **SGD на мини-партиях** $B$: $$\theta_{t+1} = \theta_t - \eta_t \nabla_\theta \hat R_B(\theta_t)$$ Используется вместо полного градиентного спуска: в память не влезет, сходится быстрее, шум помогает выходить из локальных минимумов. Расширения: momentum, Adam, RMSProp.
## Backprop — алгоритм
**Прямой проход:** считаем $z^{(l)} = W^{(l)} h^{(l-1)} + b^{(l)}$, $h^{(l)} = \sigma_l(z^{(l)})$, в конце — лосс $L$. Запоминаем активации. **Обратный проход:** цепное правило производных. Вводим $\delta^{(l)} = \partial L / \partial z^{(l)}$ — градиент по предактивации, центральная величина.
- На выходе: $\delta^{(L)} = \nabla_{\hat y} L \odot \sigma_L'(z^{(L)})$.
- Для softmax + cross-entropy упрощается: $\delta^{(L)} = \hat y - y$. **Рекурсия:** $$\delta^{(l)} = (W^{(l+1)\top} \delta^{(l+1)}) \odot \sigma_l'(z^{(l)})$$ **Градиенты по параметрам:** $$\frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} (h^{(l-1)})^\top, \quad \frac{\partial L}{\partial b^{(l)}} = \delta^{(l)}$$ Сложность: обратный проход $\sim$ прямого. По памяти линейно от глубины — нужно хранить все активации.
## Дифференцируемое программирование
Модель = программа из дифференцируемых операций. Каждая операция знает свой forward (как вычислить значение) и backward (как пропустить градиент назад). Можно дифференцировать любую программу: условия, циклы, даже дискретные операции (через релаксации типа Gumbel-Softmax).
## Граф потока вычислений
Computation graph — DAG: узлы = операции/тензоры, рёбра = потоки данных. Forward строит граф, backward обходит обратно с применением правил backprop в каждом узле. Реализуется в современных фреймворках (PyTorch, TensorFlow, JAX) как autograd — автоматическое построение и обход графа.
## Автоматическое дифференцирование
**Reverse-mode AD (= backprop):** обратный проход по графу, считает $\partial L / \partial \theta$ для скалярного $L$ за один проход. Эффективен при **много входов (параметров), один выход (лосс)** — типичная ситуация в DL: миллионы параметров и один скаляр потерь. Поэтому именно reverse-mode стал стандартом для обучения нейросетей.
