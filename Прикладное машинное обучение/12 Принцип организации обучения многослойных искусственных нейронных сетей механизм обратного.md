# Принцип организации обучения многослойных искусственных нейронных сетей: механизм обратного распространения ошибки дифференцируемое программирование и граф потока вычислений.

## TL;DR

### Многослойная сеть (MLP)

$$h^{(0)} = x, \quad h^{(l)} = \sigma_l(W^{(l)} h^{(l-1)} + b^{(l)}), \quad \hat y = h^{(L)}.$$

Композиция аффинных слоёв и нелинейностей $\sigma$ (ReLU, sigmoid, tanh, GELU, softmax на выходе).

**Без нелинейности** композиция линейных слоёв = линейная функция — сеть вырождается в линейную модель.

### Принцип обучения

Минимизация эмпирического риска: $$\hat R(\theta) = \frac{1}{n} \sum_{i=1}^n L(f_\theta(x_i), y_i).$$

$L$ — функция потерь (MSE, cross-entropy и т.д.).

**SGD на мини-партиях** $B$: $$\theta_{t+1} = \theta_t - \eta_t \nabla_\theta \hat R_B(\theta_t).$$

Используется вместо полного градиентного спуска: в память не влезет, сходится быстрее, шум помогает выходить из локальных минимумов.

**Расширения:** momentum, Nesterov, **Adam**, RMSProp, AdaGrad. **Schedulers:** cosine annealing, step decay, warmup.

### Backprop — алгоритм

**Прямой проход:** считаем $z^{(l)} = W^{(l)} h^{(l-1)} + b^{(l)}$, $h^{(l)} = \sigma_l(z^{(l)})$, в конце — лосс $L$. Запоминаем активации.

**Обратный проход:** цепное правило производных. Вводим $\delta^{(l)} = \partial L / \partial z^{(l)}$ — градиент по предактивации.

- На выходе: $\delta^{(L)} = \nabla_{\hat y} L \odot \sigma_L'(z^{(L)})$.
- Для softmax + cross-entropy упрощается: $\delta^{(L)} = \hat y - y$.

**Рекурсия:** $\delta^{(l)} = (W^{(l+1)\top} \delta^{(l+1)}) \odot \sigma_l'(z^{(l)})$.

**Градиенты по параметрам:** $$\frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} (h^{(l-1)})^\top, \quad \frac{\partial L}{\partial b^{(l)}} = \delta^{(l)}.$$

**Сложность:** обратный проход $\sim$ прямого. По памяти линейно от глубины (хранить все активации).

### Дифференцируемое программирование

Модель = программа из дифференцируемых операций. Каждая операция знает свой forward и backward.

**Граф потока вычислений (computation graph)** — DAG: узлы = операции/тензоры, рёбра = потоки данных. Forward строит граф, backward обходит обратно с применением правил backprop в каждом узле.

### Автоматическое дифференцирование

**Forward-mode AD:** проход в направлении forward, считает $\partial y / \partial x$ для одного входа за один проход. Эффективен при **мало входов, много выходов**.

**Reverse-mode AD (= backprop):** обратный проход, считает $\partial L / \partial \theta$ для скалярного $L$ за один проход. Эффективен при **много входов (параметров), один выход (лосс)**. Стандарт DL.

### Фреймворки

- **PyTorch** — динамический граф (define-by-run), удобен для отладки, питоновские условия и циклы работают естественно.
- **TensorFlow 2 / Keras** — статический граф через `tf.function`, быстрее, лучше оптимизируется.
- **JAX** — функциональный autograd через `jax.grad`, JIT-компиляция, `vmap`/`pmap`.

Дифференцируется любая программа: условия (через REINFORCE/Gumbel-Softmax), циклы, даже дискретные операции (через релаксации).

### Особенности оптимизации нейросетей

**Невыпуклый ландшафт** — много локальных минимумов, седловых точек, плато. SGD на практике находит хорошие.

**Vanishing / exploding gradients** в глубоких сетях:

- Vanishing: градиенты затухают (sigmoid/tanh насыщаются).
- Exploding: градиенты растут экспоненциально.

**Решения:**
- ReLU вместо sigmoid/tanh.
- Residual connections (skip connections).
- Batch normalization, layer normalization.
- Правильная инициализация.
- Gradient clipping для exploding.

### Инициализация весов

- **Xavier (Glorot)** $\mathcal{N}(0, 2/(n_{\text{in}} + n_{\text{out}}))$ — для **sigmoid, tanh** (симметричные).
- **He (Kaiming)** $\mathcal{N}(0, 2/n_{\text{in}})$ — для **ReLU** (учитывает обнуление половины активаций).

Без правильной инициализации глубокая сеть не обучается.

### Регуляризация
- **L2 (weight decay)** — штраф $|\theta|^2$.
- **Dropout** — случайное обнуление нейронов в обучении.
- **Label smoothing** — мягкие метки.
- **Data augmentation** — расширение выборки.
- **Early stopping** — остановка по валидации.

### Подводные камни
- ReLU не дифференцируема в 0 — на практике берут субградиент 0 или 1.
- Softmax + log → численная нестабильность, используется **log-sum-exp trick**.
- Не у всех операций есть градиент (argmax, sampling) — нужно репараметризовать (Gumbel, reparameterization trick).
- Несбалансированные данные / плохая инициализация → обучение зависает.

### Главное на экзамене
1. **MLP** — формула слоя $h^{(l)} = \sigma(W h^{(l-1)} + b)$, зачем нелинейность.
2. **Backprop** — прямой проход + обратный через цепное правило.
3. **$\delta^{(l)}$** — градиент по предактивации, центральная величина, считается рекуррентно.
4. **Computation graph** — DAG операций, реализуется autograd в фреймворках.
5. **Reverse-mode AD** — стандарт DL, эффективен при много параметров и одном лоссе.
6. **Vanishing/exploding gradient** — главные проблемы глубоких сетей, решения: ReLU, residual, BN, инициализация.
7. **He для ReLU, Xavier для tanh/sigmoid**.
