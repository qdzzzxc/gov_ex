# Задача выявления сообществ в сетях, понятие модулярности, алгоритмы Гирвана-Ньюмана, Лувенский алгоритм, дискретный метод распространения меток.

## Задача выявления сообществ
Сообщество — группа узлов, между которыми много рёбер, а с остальной сетью мало. Задача: разбить узлы $V$ на сообщества $C_1, \dots, C_K$ автоматически по структуре графа. Число $K$ обычно неизвестно. Это обучение без учителя на графах. Применения: соцсети, биология (модули белковых сетей), рекомендации, антифрод.
## Модулярность Ньюмана $Q$
Мера качества разбиения — сравнение с случайной сетью с теми же степенями (configuration model): $$Q = \frac{1}{2m} \sum_{i, j} \left( A_{ij} - \frac{k_i k_j}{2m} \right) \delta(c_i, c_j)$$
- $A_{ij} - k_i k_j / (2m)$ — избыток реальных рёбер над случайностью.
- $\delta(c_i, c_j)$ — учитываем только пары в одном сообществе. Диапазон: $Q \in [-1, 1]$. Для реальных сетей $Q \approx 0{,}3$–$0{,}7$. Resolution limit — главный недостаток: $Q$ не различает мелкие сообщества, стремится их объединять.
## Гирван-Ньюман (2002)
Идея: рёбра между сообществами — «мостики» с высокой edge-betweenness centrality (число кратчайших путей, проходящих через ребро). Алгоритм:
1. Посчитать edge-betweenness всех рёбер.
2. Удалить ребро с максимальной.
3. Если граф разбился — зафиксировать разбиение.
4. Пересчитать edge-betweenness, повторить.
5. Получаем дендрограмму разбиений; выбираем разрез с максимальным $Q$. Сложность: $O(m^2 n)$ — очень медленно для больших сетей. Используется для малых сетей и как эталон.
## Лувенский алгоритм (2008)
Жадная оптимизация модулярности с многоуровневым агрегированием. Стандарт индустрии. Фаза 1 — локальное движение:
- Каждый узел — своё сообщество.
- Для каждого узла рассматриваем перенос к соседу, выбираем максимальный $\Delta Q$.
- Повторяем по всем узлам, пока есть улучшения. Фаза 2 — агрегирование:
- Сжимаем каждое сообщество в один узел.
- Рёбра становятся взвешенными, внутренние — петлями.
- Возвращаемся к Фазе 1 на новом графе. Повторяем до стабилизации $Q$. Сложность: $O(n \log n)$. $\Delta Q$ при переносе считается локально за $O(\deg(i))$ через сохранённые статистики. Наследует resolution limit модулярности — мелкие сообщества склеиваются.
## Дискретный метод распространения меток (LPA, 2007)
Самый простой и быстрый алгоритм. Алгоритм:
1. Каждому узлу — уникальная метка.
2. Каждый узел принимает самую частую метку среди соседей. Ничьи разрешаются случайно.
3. Повторяем до стабилизации. Свойства:
- $O(m)$ за итерацию, сходится за $O(\log n)$.
- Без параметров.
- Недетерминирован — разные запуски дают разные результаты.
- Может вырождаться в одну большую метку. Используется для гигантских сетей и разведочного анализа.
