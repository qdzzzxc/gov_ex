# Вероятностные пространства. Независимость событий. Формула полной вероятности. Формула Байеса.

### Вероятностное пространство

Тройка $(\Omega, \mathcal{F}, P)$:

- **$\Omega$** — пространство **элементарных исходов** (всё, что может случиться).
- **$\mathcal{F}$** — **$\sigma$-алгебра событий** — семейство подмножеств $\Omega$, для которых корректно определена вероятность. Замкнута относительно дополнений и счётных объединений. Для дискретных $\Omega$ — все подмножества. Для непрерывных — выбор "хороших" множеств.
- **$P: \mathcal{F} \to [0, 1]$** — **вероятностная мера**.

### Аксиомы Колмогорова

1. **Неотрицательность:** $P(A) \ge 0$.
2. **Нормировка:** $P(\Omega) = 1$.
3. **Счётная аддитивность:** для попарно **несовместных** $A_1, A_2, \dots$: $P(\bigsqcup A_i) = \sum P(A_i)$.

### Базовые свойства

- $P(\emptyset) = 0$, $P(\bar A) = 1 - P(A)$.
- Монотонность: $A \subseteq B \Rightarrow P(A) \le P(B)$.
- Включения-исключения: $P(A \cup B) = P(A) + P(B) - P(A \cap B)$.

### Типичные модели

- **Классическая:** конечное равновероятное $\Omega$, $P(A) = |A|/|\Omega|$.
- **Геометрическая:** $P(A) = \mu(A)/\mu(\Omega)$ через меру.
- **Дискретная:** $\Omega = {\omega_1, \omega_2, \dots}$, $P(\omega_i) = p_i \ge 0$, $\sum p_i = 1$.

### Условная вероятность

$$P(A | B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0.$$

Идея: сужаем пространство до $B$, считаем долю $A$ внутри $B$.

**Формула умножения:** $$P(A \cap B) = P(B) \cdot P(A | B) = P(A) \cdot P(B | A).$$

Для нескольких: $P(A_1 \cap \dots \cap A_n) = P(A_1) P(A_2|A_1) \cdots P(A_n | A_1 \cap \dots \cap A_{n-1})$.

### Независимость

$A$ и $B$ **независимы**: $$P(A \cap B) = P(A) \cdot P(B).$$

Эквивалентно: $P(A|B) = P(A)$ — знание $B$ не влияет на оценку $A$.

**Несовместность $\ne$ независимость.** Несовместные ($A \cap B = \emptyset$) обычно зависимы (одно исключает другое).

**Попарная vs совместная независимость:**

- Попарная: $P(A_i \cap A_j) = P(A_i) P(A_j)$ для каждой пары.
- В совокупности: $P(A_{i_1} \cap \dots \cap A_{i_k}) = \prod P(A_{i_j})$ для **любого** подмножества.

Попарная не влечёт совместную. Контрпример Бернштейна (тетраэдр с цветными гранями).

### Полная группа гипотез

${H_1, \dots, H_n}$ — **разбиение** $\Omega$:

1. Попарно несовместны: $H_i \cap H_j = \emptyset$.
2. Покрывают всё: $\bigsqcup H_i = \Omega$.
3. $P(H_i) > 0$.

### Формула полной вероятности

$$P(A) = \sum_i P(H_i) \cdot P(A | H_i).$$

Раскладываем $P(A)$ по сценариям $H_i$.

### Формула Байеса

$$P(H_k | A) = \frac{P(H_k) \cdot P(A | H_k)}{P(A)} = \frac{P(H_k) P(A|H_k)}{\sum_i P(H_i) P(A|H_i)}.$$

**Априорная** $P(H_k)$ — до наблюдения. **Апостериорная** $P(H_k | A)$ — после наблюдения $A$.

**Содержательный смысл:** как пересчитать мнение, узнав что-то новое.

### Каноничный пример: медицинский тест

- $P(\text{болен}) = 0.01$, $P(+|\text{болен}) = 0.99$, $P(+|\text{здоров}) = 0.05$.
- $P(+) = 0.01 \cdot 0.99 + 0.99 \cdot 0.05 = 0.0594$.
- $P(\text{болен}|+) = 0.0099 / 0.0594 \approx 0.167$.

**Только 17%** при положительном тесте — из-за низкой базовой ставки. Большинство положительных — ложные срабатывания на огромной популяции здоровых.

### Подводные камни

- **Ошибка прокурора:** $P(A|B) \ne P(B|A)$. Путают $P(\text{улика}|\text{невиновен})$ с $P(\text{невиновен}|\text{улика})$. Без априорной вероятности виновности утверждение бессмысленно. Реальный кейс — Sally Clark.
- **Парадокс ложных срабатываний** — даже точный тест на редкой болезни даёт много ложных +.
- Попарная независимость **не** влечёт независимость в совокупности.
- $\sigma$-аддитивность принципиальна — конечной аддитивности недостаточно для теории.

### Применения формулы Байеса

- **Наивный Байес** в ML (классификация спама, текстов).
- **Медицинская диагностика** — пересмотр вероятности болезни после теста.
- **Байесовская статистика** — общий фреймворк инференса.
- **Антифрод**, спам-фильтры, антивирусы.

### Главное на экзамене

1. **$(\Omega, \mathcal{F}, P)$** — три компонента вероятностного пространства.
2. **Три аксиомы Колмогорова** — неотрицательность, нормировка, счётная аддитивность.
3. **Условная вероятность** — формула и идея сужения пространства.
4. **Независимость** — $P(A \cap B) = P(A) P(B)$.
5. **Формула полной вероятности** — разложение по гипотезам.
6. **Формула Байеса** — пересчёт априорной в апостериорную.
7. **Пример** с медицинским тестом, показывающий важность базовой ставки.
