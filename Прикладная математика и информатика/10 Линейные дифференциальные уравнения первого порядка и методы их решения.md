# Линейные дифференциальные уравнения первого порядка и методы их решения. Задача Коши. Существование и единственность решения задачи Коши.
## Постановка
Линейное ОДУ 1-го порядка: $$y' + p(x) y = q(x)$$
$p, q$ — заданные непрерывные функции. $y$ и $y'$ входят линейно (в первой степени).
**Однородное:** $q(x) \equiv 0$. **Неоднородное:** $q(x) \ne 0$.
## Решение однородного
$y' = -p(x) y$ — переменные разделяются: $$\frac{dy}{y} = -p(x) dx \Rightarrow y = C \cdot e^{-\int p(x) dx}$$
## Структура решения неоднородного
$$y = y_o + y_{\text{ч}}$$
где $y_o$ — общее решение однородного (с константой $C$), $y_{\text{ч}}$ — любое частное решение неоднородного.
Почему: разность двух решений неоднородного — решение однородного. Значит любое решение = фиксированное частное + любое из однородного.
## Метод интегрирующего множителя
Умножаем уравнение на $\mu(x) = e^{\int p(x) dx}$. Левая часть становится производной произведения $(\mu y)'$: $$(\mu y)' = \mu q$$
Интегрируем: $$y = \frac{1}{\mu(x)} \left( \int \mu(x) q(x) dx + C \right)$$
Альтернативные подходы: **метод Бернулли** (подстановка $y = uv$, выбираем $v$ так чтобы $v' + pv = 0$) и **метод Лагранжа** (вариация постоянной — в решении однородного заменяем $C \to C(x)$). Все три эквивалентны.
## Общая формула
$$y(x) = e^{-P(x)} \left( \int q(x) e^{P(x)} dx + C \right), \quad P(x) = \int p(x) dx$$
## Задача Коши
$$y' + p(x) y = q(x), \quad y(x_0) = y_0$$
ОДУ + начальное условие. Алгоритм:
1. Найти общее решение.
2. Подставить $x = x_0$, $y = y_0$.
3. Найти $C$.
4. Подставить $C$ обратно — получаем частное решение.
## Теорема существования и единственности (Пикара-Линделёфа)
Для общего ОДУ $y' = f(x, y), , y(x_0) = y_0$:
**Условия:**
1. $f$ непрерывна по $(x, y)$ в окрестности $(x_0, y_0)$.
2. $f$ удовлетворяет условию Липшица по $y$ ($\partial f / \partial y$ ограничена).
**Утверждение:** существует и единственное решение в окрестности $x_0$.
Для **линейного** ОДУ $f = -p(x) y + q(x)$ — Липшицевость по $y$ автоматическая при непрерывных $p, q$. Решение продолжается на весь интервал непрерывности $p, q$.
Доказательство — методом последовательных приближений Пикара через теорему Банаха о неподвижной точке.
## Пример
$y' - 2y/x = x^3$, $y(1) = 3$, $x > 0$.
$p = -2/x$, $\mu = e^{\int(-2/x) dx} = 1/x^2$.
$(y/x^2)' = x \Rightarrow y/x^2 = x^2/2 + C \Rightarrow y = x^4/2 + Cx^2$.
Условие: $1/2 + C = 3 \Rightarrow C = 5/2$.
Ответ: $y = x^4/2 + 5x^2/2$.
