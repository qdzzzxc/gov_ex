# Булевы функции. Важнейшие классы булевых функций. Теорема о полноте (теорема Поста).

### Булева функция
$f: {0, 1}^n \to {0, 1}$. Принимает $n$ битов, выдаёт один.

Всего функций от $n$ переменных: $2^{2^n}$ (для $n=2$ — 16, для $n=3$ — 256).

**Способы задания:** таблица истинности, формула, СДНФ/СКНФ, **полином Жегалкина**.

### Полином Жегалкина

Единственное представление через $\oplus$ (XOR), $\land$ и константы 0, 1: $$f = c_0 \oplus c_1 x_1 \oplus c_2 x_2 \oplus c_{12} x_1 x_2 \oplus \dots$$

XOR-сумма мономов с коэффициентами 0 или 1.

**Как построить** (метод неопределённых коэффициентов):

1. Записать общий вид с неизвестными $c_S$.
2. Подставить все $2^n$ комбинаций из таблицы.
3. Решать в $\mathbb{F}_2$ ($1 \oplus 1 = 0$) — каждое уравнение определяет один коэффициент.

**Примеры:**

- $\neg x = 1 \oplus x$
- $x \lor y = x \oplus y \oplus xy$
- $x \to y = 1 \oplus x \oplus xy$

### 5 замкнутых классов Поста

|Класс|Определение|
|---|---|
|$T_0$|$f(0,\dots,0) = 0$ — сохраняющие 0|
|$T_1$|$f(1,\dots,1) = 1$ — сохраняющие 1|
|$S$|$f(\neg x_1, \dots, \neg x_n) = \neg f(x_1, \dots, x_n)$ — самодвойственные|
|$M$|монотонные: $\vec{x} \le \vec{y} \Rightarrow f(\vec{x}) \le f(\vec{y})$|
|$L$|линейные: полином Жегалкина степени $\le 1$ (без произведений)|

**Замкнутость:** суперпозиция функций из класса даёт функцию того же класса. Из монотонных → только монотонные; из линейных → только линейные, и т.д.

### Принадлежность элементарных функций

|Функция|$T_0$|$T_1$|$S$|$M$|$L$|
|---|---|---|---|---|---|
|$0$|✓|✗|✗|✓|✓|
|$1$|✗|✓|✗|✓|✓|
|$x$|✓|✓|✓|✓|✓|
|$\neg x$|✗|✗|✓|✗|✓|
|$\land$|✓|✓|✗|✓|✗|
|$\lor$|✓|✓|✗|✓|✗|
|$\oplus$|✓|✗|✗|✗|✓|
|$\to$|✗|✓|✗|✗|✗|
|$\mid$|✗|✗|✗|✗|✗|
|$\downarrow$|✗|✗|✗|✗|✗|

Штрих Шеффера ($x \mid y = \neg(x \land y)$) и стрелка Пирса ($x \downarrow y = \neg(x \lor y)$) **не лежат ни в одном из 5 классов**.

### Функциональная полнота

Система $\Sigma$ **функционально полна**, если любая булева функция выражается суперпозицией функций из $\Sigma$.

### Теорема Поста о полноте

> $\Sigma$ функционально полна $\iff$ для **каждого** из 5 классов $T_0, T_1, S, M, L$ в $\Sigma$ найдётся функция, **не входящая** в этот класс.

То есть $\Sigma$ должна "**пробивать**" все 5 классов.

**Если хотя бы один класс не пробит** — система неполна, потому что из этого класса не выйдешь суперпозицией.

### Как проверить полноту

Составь таблицу принадлежности классам для всех функций системы. В каждом столбце должно быть **хотя бы одно ✗**.

### Полные системы

- ${\neg, \land}$ — полная. $\neg$ пробивает $T_0, T_1, M$; $\land$ пробивает $S, L$.
- ${\neg, \lor}$ — полная (аналогично).
- ${\oplus, \land, 1}$ — полная (Жегалкин).
- $\{\mid\}$ — полная (Шеффер один).
- ${\downarrow}$ — полная (Пирс один).

### Неполные системы

- ${\land, \lor}$ — обе функции в $T_0, T_1, M$. Три класса не пробиты. Никогда не получишь $\neg x$ или другую немонотонную функцию.
- ${\to}$ — лежит в $T_1$ полностью, не пробивает.

### Применения

- **Дизайн микросхем:** все процессоры строят из NAND-вентилей (штрих Шеффера) — одного типа компонента достаточно.
- **Теория сложности** в дискретной математике.
- Связь с **теоремой Колмогорова-Арнольда** в непрерывном случае.

### Подводные камни

- Замкнутость классов: из монотонных только монотонные. Чтобы получить функцию вне класса — нужно **добавить** функцию извне.
- Самодвойственность: $\neg x$ самодвойственна, но $\neg(\land)$ нет.

### Главное на экзамене

1. **Булева функция** — $f: {0,1}^n \to {0,1}$, всего $2^{2^n}$.
2. **Полином Жегалкина** — единственное представление через $\oplus$ и $\land$.
3. **5 классов Поста** — определения.
4. **Теорема Поста** — критерий полноты через пробивание классов.
5. **Шеффер и Пирс** — каждый один даёт полный базис, потому что не лежит ни в одном классе.
