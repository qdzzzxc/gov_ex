# Булевы функции. Важнейшие классы булевых функций. Теорема о полноте (теорема Поста).

## TL;DR
Булева функция — $f: \{0,1\}^n\to\{0,1\}$, всего их $2^{2^n}$. Способы задания: таблица, формула, СДНФ/СКНФ, полином Жегалкина. **5 замкнутых классов Поста**: $T_0$ (сохраняющие 0), $T_1$ (сохраняющие 1), $S$ (самодвойственные), $M$ (монотонные), $L$ (линейные). **Теорема Поста о полноте**: система функций функционально полна $\iff$ для каждого из этих 5 классов в системе найдётся функция, в этот класс не входящая. Минимальные базисы: штрих Шеффера $\{|\}$, стрелка Пирса $\{\downarrow\}$.

## Развёрнуто

### Определение
$f: \{0,1\}^n\to\{0,1\}$ — булева функция от $n$ переменных. Способы задания:
- **таблица истинности** (вектор значений длины $2^n$);
- формула в каком-либо базисе;
- СДНФ/СКНФ;
- **полином Жегалкина** $f = c_0 \oplus c_1 x_1 \oplus c_2 x_2 \oplus c_{12} x_1 x_2 \oplus\dots$ — однозначное представление через $\oplus$ (XOR), $\land$ и константы.

Число булевых функций от $n$ переменных: $2^{2^n}$ (для $n=2$ — 16, для $n=3$ — 256).

### Существенные и фиктивные переменные
$x_i$ **фиктивна**, если значение $f$ не зависит от $x_i$ при фиксации остальных. Иначе — **существенная**.

### Элементарные булевы функции (от 1–2 переменных)
$0, 1, x, \neg x, x\land y, x\lor y, x\oplus y, x\to y, x\leftrightarrow y$, штрих Шеффера $x|y = \neg(x\land y)$, стрелка Пирса $x\downarrow y = \neg(x\lor y)$.

### СДНФ и СКНФ
Любая ненулевая $f$ имеет единственную СДНФ; любая нееединичная — единственную СКНФ. Алгоритм построения по таблице — см. вопрос 06.

### Полином Жегалкина
Для $f$ существует и единственно представление $f(x_1,\dots,x_n) = \bigoplus_{S\subseteq\{1,\dots,n\}} c_S \prod_{i\in S} x_i$. Его можно получить из СДНФ, заменяя дизъюнкцию непересекающихся минтермов на $\oplus$, отрицания по правилу $\neg x = 1\oplus x$ и раскрывая скобки по модулю 2; другой способ — метод неопределённых коэффициентов.

### Замкнутые классы Поста
Класс $K$ замкнут, если суперпозиция функций из $K$ снова в $K$. Пять важнейших:

| Класс | Определение |
|---|---|
| $T_0$ | $f(0,\dots,0) = 0$ — сохраняющие 0 |
| $T_1$ | $f(1,\dots,1) = 1$ — сохраняющие 1 |
| $S$  | $f(\neg x_1,\dots,\neg x_n) = \neg f(x_1,\dots,x_n)$ — самодвойственные |
| $M$  | $f$ монотонна: $\tilde x\le \tilde y \Rightarrow f(\tilde x)\le f(\tilde y)$ |
| $L$  | $f$ линейна: $f = c_0\oplus c_1 x_1\oplus\dots\oplus c_n x_n$ (полином Жегалкина степени $\le 1$) |

Принадлежность элементарных функций:
| Функция | $T_0$ | $T_1$ | $S$ | $M$ | $L$ |
|---|---|---|---|---|---|
| 0 | + | – | – | + | + |
| 1 | – | + | – | + | + |
| $x$ | + | + | + | + | + |
| $\neg x$ | – | – | + | – | + |
| $\land$ | + | + | – | + | – |
| $\lor$ | + | + | – | + | – |
| $\oplus$ | + | – | – | – | + |
| $|$ | – | – | – | – | – |
| $\downarrow$ | – | – | – | – | – |

### Теорема Поста о функциональной полноте
Система $\Sigma\subseteq P_2$ (множество всех булевых функций) **функционально полна**, если любая булева функция выражается суперпозицией из $\Sigma$ (с использованием переменных).

**Теорема Поста.** $\Sigma$ полна $\iff$ $\Sigma\not\subseteq T_0$, $\Sigma\not\subseteq T_1$, $\Sigma\not\subseteq S$, $\Sigma\not\subseteq M$, $\Sigma\not\subseteq L$. Иначе говоря, для каждого из 5 классов в $\Sigma$ найдётся функция, не лежащая в нём.

**Следствия.**
- $\{\neg, \land\}$, $\{\neg, \lor\}$, $\{\neg, \to\}$ — полные (минимальные) базисы. Например, проверка для $\{\neg,\land\}$: $\neg\notin T_0,T_1,M$, $\land\notin S,L$ — все 5 классов «пробиты».
- Один штрих Шеффера $|$ полон (так как не лежит ни в одном из 5 классов). Аналогично стрелка Пирса $\downarrow$.
- $\{\land,\lor\}$ не полна (всё в $T_0\cap T_1\cap M$).
- $\{\oplus,\land,1\}$ — полна (Жегалкин).

### Пример
$f = (1,0,0,1)$ для $(x,y)\in\{(0,0),(0,1),(1,0),(1,1)\}$. Это $\neg(x\oplus y)\equiv x\leftrightarrow y$. Полином Жегалкина: $f = 1\oplus x\oplus y$. Класс: $f(0,0)=1\Rightarrow \notin T_0$, $f(1,1)=1\Rightarrow \in T_1$, $f$ линейна $\Rightarrow \in L$.

### Подводные камни
- $\{\to\}$ не полна (содержит, например, всё $\subseteq T_1$).
- Самодвойственность: $\neg x$ самодвойственна, но $\neg(\land)$ — не обязательно.
- Иногда базисом называют функционально полную систему, минимальную по включению.
