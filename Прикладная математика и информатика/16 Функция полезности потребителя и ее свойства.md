# Функция полезности потребителя и ее свойства. CES-функция полезности и ее частные случаи.

## TL;DR
**Функция полезности** $U: X\to\mathbb R$ численно представляет предпочтения потребителя на наборах благ $x = (x_1,\dots,x_n)$. Существует для непрерывного, рефлексивного, транзитивного и полного предпочтения (теорема Дебрё). **Свойства**: монотонность ($\partial U/\partial x_i \ge 0$), квазивогнутость (выпуклые предпочтения), непрерывность. **Предельная норма замещения** $MRS_{ij} = MU_i/MU_j$. Во внутреннем дифференцируемом оптимуме $MRS = p_i/p_j$. **CES**: $U = (\sum \alpha_i x_i^\rho)^{1/\rho}$ с эластичностью замещения $\sigma = 1/(1-\rho)$. Частные случаи: $\rho=1$ — линейная (совершенные субституты, $\sigma=\infty$); $\rho\to 0$ — Кобба–Дугласа ($\sigma=1$); $\rho\to-\infty$ — Леонтьева ($\sigma=0$, предельная форма совершенных комплементов).

## Развёрнуто

### Предпочтения и существование функции полезности
$X$ — потребительское множество (часто $\mathbb R^n_{\ge 0}$). Отношение предпочтения $\succsim$ на $X$ удовлетворяет аксиомам:
1. **Рефлексивность**: $x\succsim x$.
2. **Транзитивность**: $x\succsim y, y\succsim z\Rightarrow x\succsim z$.
3. **Полнота (связность)**: $\forall x,y$ либо $x\succsim y$, либо $y\succsim x$.
4. **Непрерывность**: множества $\{y: y\succsim x\}$ и $\{y: x\succsim y\}$ замкнуты.

**Теорема Дебрё.** Для непрерывного предпочтения $\succsim$ на связном $X\subseteq\mathbb R^n$ существует непрерывная функция полезности $U: X\to\mathbb R$ такая, что $x\succsim y\iff U(x)\ge U(y)$.

### Кардинальный vs ординальный подход
- **Ординальный**: важен только порядок, $U$ определена с точностью до строго возрастающего преобразования $f(U)$.
- **Кардинальный**: численные значения имеют смысл (например, при оценке риска через ожидаемую полезность фон Неймана–Моргенштерна).

### Свойства
- **Монотонность** (ненасыщаемость): если $x \ge y$ и $x\ne y$, то $U(x) > U(y)$. Эквивалентно $\partial U/\partial x_i > 0$.
- **Выпуклость предпочтений**: множества $\{y: y\succsim x\}$ выпуклы $\iff U$ **квазивогнута**: $U(\lambda x + (1-\lambda)y)\ge \min(U(x),U(y))$. Экономически — потребитель предпочитает «смешанные» наборы.
- **Дифференцируемость** $U$ позволяет работать с предельными величинами.

### Предельная полезность и MRS
$MU_i = \partial U/\partial x_i$ — **предельная полезность** $i$-го блага. **Закон убывающей предельной полезности (Госсена)**: $\partial^2 U/\partial x_i^2 < 0$ — каждая следующая единица блага приносит меньше дополнительной полезности.

**Предельная норма замещения** двух благ:
$$MRS_{ij} = -\frac{dx_j}{dx_i}\bigg|_{U=\text{const}} = \frac{MU_i}{MU_j}.$$
Сколько единиц $j$-го блага потребитель готов отдать за дополнительную единицу $i$-го, оставаясь на той же кривой безразличия. Убывает по $x_i$ при выпуклых предпочтениях.

### Кривые безразличия
$\{x: U(x) = c\}$ — линии уровня полезности. Свойства: непрерывны, не пересекаются, расположены выше при больших $c$, выпуклы (при выпуклых предпочтениях).

### Задача потребителя
$$\max U(x)\ \text{при}\ p^\top x\le I,\ x\ge 0.$$
При локальной ненасыщаемости бюджет расходуется полностью: $p_1 x_1+\dots+p_n x_n = I$. Для внутреннего дифференцируемого оптимума лагранжиан $\mathcal L = U - \lambda(p^\top x - I)$ даёт условия первого порядка:
$$\frac{MU_i}{p_i} = \lambda\quad\forall i,$$
эквивалентно
$$MRS_{ij} = \frac{p_i}{p_j}.$$
Геометрически — касание кривой безразличия и бюджетной линии. Если оптимум угловой, равенство MRS и отношения цен заменяется неравенствами/условиями Куна-Таккера.

### CES-функция полезности
**Constant Elasticity of Substitution**:
$$U(x) = \left(\sum_{i=1}^n \alpha_i x_i^\rho\right)^{1/\rho}, \quad \alpha_i > 0,\ \rho \le 1,\ \rho\ne 0.$$
Эластичность замещения между любой парой благ постоянна:
$$\sigma = \frac{1}{1-\rho}.$$

### Частные случаи CES
- **Линейная (совершенные субституты)**: $\rho = 1$. $U = \sum \alpha_i x_i$. $\sigma = \infty$. Кривые безразличия — параллельные прямые. Потребитель полностью замещает блага.
- **Кобба–Дугласа**: $\rho\to 0$ (с нормировкой $\sum\alpha_i = 1$). $U = \prod x_i^{\alpha_i}$ (или $\ln U = \sum\alpha_i\ln x_i$). $\sigma = 1$. Доли расходов на блага постоянны: $p_i x_i = \alpha_i I$.
- **Леонтьева (совершенные комплементы)**: $\rho\to -\infty$. В симметричной записи предел даёт $U=\min_i x_i$; для заданных технологических/потребительских пропорций используют нормированную форму $U=\min_i(x_i/a_i)$. $\sigma = 0$. Блага потребляются в строгой пропорции.
- При $0 < \rho < 1$ — несовершенные субституты, $\sigma > 1$; при $\rho < 0$ — комплементарные тенденции, $\sigma < 1$.

### Пример
Кобба–Дугласа $U = x_1^{0,4} x_2^{0,6}$, цены $p_1=2$, $p_2=3$, доход $I=60$. Доли: $p_1 x_1 = 0{,}4\cdot 60 = 24$, $p_2 x_2 = 0{,}6\cdot 60 = 36$. $x_1=12, x_2=12$. $MRS_{12} = (0{,}4 x_2)/(0{,}6 x_1) = (0{,}4\cdot 12)/(0{,}6\cdot 12) = 2/3 = p_1/p_2$. ✓

### Подводные камни
- $U$ определена с точностью до возрастающего преобразования — нельзя интерпретировать абсолютные значения и их разности (для ординальной полезности).
- При невыпуклых предпочтениях оптимум может быть в угловой точке.
- CES при $\rho = 0$ требует предельного перехода (Кобба–Дугласа); подставлять $\rho=0$ в формулу нельзя (деление на 0).
