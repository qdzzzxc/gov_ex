# Функция полезности потребителя и ее свойства. CES-функция полезности и ее частные случаи.

## Функция полезности
$U: X \to \mathbb{R}$ численно представляет предпочтения потребителя на наборах благ $x = (x_1, \dots, x_n)$.
**Правило выбора:** $x \succsim y \iff U(x) \ge U(y)$.
## Аксиомы предпочтений
1. **Рефлексивность:** $x \succsim x$.
2. **Транзитивность:** $x \succsim y$ и $y \succsim z$ → $x \succsim z$.
3. **Полнота:** для любых $x, y$ либо $x \succsim y$, либо $y \succsim x$.
4. **Непрерывность:** малые изменения не дают скачков предпочтений.
**Теорема Дебрё:** при этих аксиомах существует непрерывная $U$, представляющая предпочтения.
Полезность бывает **ординальной** (важен только порядок, $U$ определена с точностью до строго возрастающего преобразования) и **кардинальной** (численные значения имеют смысл, используется в теории ожидаемой полезности).
## Свойства
- **Монотонность:** $\partial U / \partial x_i > 0$ — больше блага, больше полезности.
- **Квазивогнутость:** $U(\lambda x + (1-\lambda) y) \ge \min(U(x), U(y))$. Соответствует выпуклости предпочтений — потребитель предпочитает смешанные наборы крайним.
- **Дифференцируемость** — для работы с предельными величинами.
## Предельная полезность и MRS
**Предельная полезность:** $MU_i = \partial U / \partial x_i$.
**Закон убывающей $MU$ (Госсен):** $\partial^2 U / \partial x_i^2 < 0$ — каждая следующая единица даёт меньше прироста.
**Предельная норма замещения:** $$MRS_{ij} = \frac{MU_i}{MU_j}$$
Сколько единиц $j$-го блага потребитель готов отдать за дополнительную единицу $i$-го, оставаясь на той же кривой безразличия. При выпуклых предпочтениях $MRS$ убывает по $x_i$.
## Кривые безразличия
${x : U(x) = c}$ — наборы с одинаковой полезностью. Не пересекаются, выше = больше полезности, выпуклы к началу координат.
## Задача потребителя
$$\max U(x) \quad \text{при} \quad p^\top x \le I, , x \ge 0$$

Через Лагранжа $\mathcal{L} = U - \lambda(p^\top x - I)$ получаем условие оптимума: $$\frac{MU_i}{p_i} = \lambda \quad \forall i \quad \Leftrightarrow \quad MRS_{ij} = \frac{p_i}{p_j}$$
$\lambda$ — предельная полезность денег (теневая цена дохода). Геометрически: кривая безразличия касается бюджетной линии.
## CES-функция полезности
$$U(x) = \left( \sum_{i=1}^n \alpha_i x_i^\rho \right)^{1/\rho}, \quad \alpha_i > 0, , \rho \le 1, , \rho \ne 0$$
**Эластичность замещения** между любой парой благ постоянна: $$\sigma = \frac{1}{1 - \rho}$$
$\rho$ — параметр формулы, $\sigma$ — экономическая интерпретация. Они жёстко связаны.
## Частные случаи CES

|$\rho$|$\sigma$|Вид|Содержание|
|---|---|---|---|
|$\rho = 1$|$\infty$|$U = \sum \alpha_i x_i$|Совершенные субституты|
|$\rho \to 0$|$1$|$U = \prod x_i^{\alpha_i}$|Кобба-Дуглас|
|$\rho \to -\infty$|$0$|$U = \min_i (x_i / \alpha_i)$|Леонтьева (комплементы)|
**Линейная** (субституты): кривые безразличия — параллельные прямые. Полное замещение.
**Кобба-Дуглас:** доли расходов на блага постоянны: $p_i x_i = \alpha_i I$.
**Леонтьева** (комплементы): блага потребляются в строгой пропорции. Кривые безразличия — L-образные.
## Пример (Кобба-Дуглас)

$U = x_1^{0.4} x_2^{0.6}$, $p_1 = 2$, $p_2 = 3$, $I = 60$.

Доли расходов: $p_1 x_1 = 0{,}4 \cdot 60 = 24 \Rightarrow x_1 = 12$; $p_2 x_2 = 36 \Rightarrow x_2 = 12$.

Проверка: $MRS_{12} = (0{,}4/0{,}6)(x_2/x_1) = 2/3 = p_1/p_2$. ✓
