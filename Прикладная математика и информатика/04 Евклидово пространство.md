# Евклидово пространство. Выпуклые множества. Отделяющая гиперплоскость. Разделяющая гиперплоскость.

### Евклидово пространство

$\mathbb{R}^n$ со **скалярным произведением** $\langle x, y \rangle = \sum x_i y_i$.
Три свойства скалярного произведения:
1. Симметричность: $\langle x, y \rangle = \langle y, x \rangle$.
2. Линейность по первому аргументу. $\langle \alpha x_1 + \beta x_2, y \rangle = \alpha \langle x_1, y \rangle + \beta \langle x_2, y \rangle.$
3. Положительная определённость: $\langle x, x \rangle > 0$ при $x \ne 0$.

**Норма:** $|x| = \sqrt{\langle x, x \rangle}$ — длина вектора.
**Метрика:** $d(x, y) = |x - y|$ — расстояние между точками.
**Неравенство Коши-Буняковского:** $|\langle x, y \rangle| \le |x| \cdot |y|$.
**Ортогональность:** $x \perp y \iff \langle x, y \rangle = 0$.
Зачем: из скалярного произведения получаются норма (длина), метрика (расстояние), углы и ортогональность. Без этого нет ни гиперплоскостей, ни замкнутости, ни компактности.
### Выпуклые множества
$M \subseteq \mathbb{R}^n$ **выпукло**, если: $$\forall x, y \in M, \lambda \in [0, 1]: \quad \lambda x + (1 - \lambda) y \in M.$$Отрезок между любыми двумя точками лежит целиком в $M$.
**Примеры выпуклых:** шар, полупространство, отрезок, симплекс, многогранник ${x : Ax \le b}$, $\mathbb{R}^n$, $\emptyset$.
**Невыпуклые:** кольцо, звезда, множества с выемкой.
**Свойство:** пересечение любого семейства выпуклых — выпукло. Объединение — не обязательно.
### Гиперплоскость

$$H = {x : \langle a, x \rangle = b}, \quad a \ne 0.$$
$a$ — нормаль, $b$ — сдвиг. В $\mathbb{R}^2$ это прямая, в $\mathbb{R}^3$ — плоскость.
**Полупространства:** ${x : \langle a, x \rangle \le b}$ и ${x : \langle a, x \rangle \ge b}$. Оба выпуклые.
### Отделимость
**Слабое отделение:** $\langle a, x \rangle \le b$ на $A$ и $\langle a, y \rangle \ge b$ на $B$. Точки могут лежать **на** гиперплоскости.
**Строгое разделение:** $\langle a, x \rangle < b$ на $A$ и $\langle a, y \rangle > b$ на $B$. Есть зазор, точки не на гиперплоскости.
### Теоремы
**Теорема об отделяющей гиперплоскости:** если $A, B$ — **выпуклые, непустые, непересекающиеся**, то существует гиперплоскость, их отделяющая (слабо).

**Теорема о строго разделяющей гиперплоскости:** если дополнительно $A$ **замкнуто**, $B$ **компактно** (замкнуто + ограничено) — отделение строгое.

**Контрпример:** два касающихся круга — есть отделяющая (касательная), но строго разделяющей нет (точка касания на обеих).

### Применения
- **SVM** в ML — разделяющая гиперплоскость с максимальным зазором между классами.
- **Линейное программирование** — двойственность, лемма Фаркаша.
- **Выпуклая оптимизация** — условия KKT.
### Главное на экзамене
1. **Евклидово пространство** = $\mathbb{R}^n$ + скалярное произведение (3 свойства). Норма и метрика из него.
2. **Выпуклое множество** — отрезок между точками целиком внутри.
3. **Гиперплоскость** $\langle a, x \rangle = b$, делит на два полупространства.
4. **Отделимость:** выпуклые непересекающиеся → отделяющая существует.
5. **Строгое разделение:** + замкнутость + компактность.
6. **SVM** — главное приложение.
