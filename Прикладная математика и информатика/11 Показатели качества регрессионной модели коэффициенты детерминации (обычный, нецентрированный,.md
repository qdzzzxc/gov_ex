# Показатели качества регрессионной модели: коэффициенты детерминации (обычный, нецентрированный, скорректированный), информационные критерии(Акайке, Шварца). Тестирование значимости оценок параметров, некоторой части оценок параметров, значимости регрессии в целом.

## Разложение вариации
Модель $y = X\beta + \varepsilon$, оценка $\hat y$, остатки $e = y - \hat y$.
- **TSS** $= \sum (y_i - \bar y)^2$ — общая вариация.
- **ESS** $= \sum (\hat y_i - \bar y)^2$ — объяснённая моделью.
- **RSS** $= \sum e_i^2$ — остаточная.
При наличии константы: $TSS = ESS + RSS$.
## Обычный коэффициент детерминации $R^2$

$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS} \in [0, 1]$$

Доля вариации $y$, объяснённая регрессией.
Ограничения:
- Корректен только для моделей с константой.
- Не убывает при добавлении любого регрессора, даже бесполезного → стимул к перегрузке.
- Сравнивать модели по $R^2$ можно только при одинаковом $y$ и $n$.
## Нецентрированный $R^2_u$
Для моделей **без константы**: $$R^2_u = 1 - \frac{RSS}{\sum y_i^2}$$
Не сравним с обычным $R^2$.
## Скорректированный $\bar R^2$
$$\bar R^2 = 1 - \frac{RSS / (n - k)}{TSS / (n - 1)} = 1 - (1 - R^2) \cdot \frac{n - 1}{n - k}$$
$n$ — число наблюдений, $k$ — число параметров (включая константу).
Делим на степени свободы — штрафует за число регрессоров. Растёт только если новый регрессор повышает объяснение больше, чем шум. Может быть отрицательным.
## Информационные критерии
$$AIC = -2 \ln L + 2k, \quad BIC = -2 \ln L + k \ln n$$
$L$ — правдоподобие модели. Чем меньше AIC/BIC — тем лучше.
При $n > e^2 \approx 7$ BIC сильнее штрафует за сложность → выбирает более простые модели. BIC состоятелен (находит истинную модель), AIC ориентирован на прогноз.
## t-тест значимости одного коэффициента
**Гипотеза:** $H_0: \beta_j = 0$.
**Статистика:** $$t = \frac{\hat\beta_j}{\text{SE}(\hat\beta_j)} \sim t_{n - k}$$
$\text{SE}$ — стандартная ошибка оценки. $|t| > t_{1 - \alpha/2, n-k}$ → отвергаем $H_0$, коэффициент значим.
## F-тест значимости группы коэффициентов
**Гипотеза:** $H_0$ — $q$ выбранных коэффициентов равны нулю.
Сравниваем полную ($RSS_U$) и редуцированную ($RSS_R$) модели: $$F = \frac{(RSS_R - RSS_U) / q}{RSS_U / (n - k)} \sim F_{q, n - k}$$
Большое $F$ → отвергаем $H_0$, группа значима.
## Общий F-тест значимости регрессии
**Гипотеза:** $H_0: \beta_1 = \dots = \beta_{k-1} = 0$ (все коэффициенты кроме константы).
$$F = \frac{ESS / (k - 1)}{RSS / (n - k)} = \frac{R^2 / (k - 1)}{(1 - R^2) / (n - k)} \sim F_{k - 1, n - k}$$
Большое $F$ → модель в целом значима.
