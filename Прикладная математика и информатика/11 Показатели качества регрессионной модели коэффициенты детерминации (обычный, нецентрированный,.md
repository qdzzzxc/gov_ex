# Показатели качества регрессионной модели: коэффициенты детерминации (обычный, нецентрированный, скорректированный), информационные критерии(Акайке, Шварца). Тестирование значимости оценок параметров, некоторой части оценок параметров, значимости регрессии в целом.

## TL;DR
Показатели качества регрессии — мера того, насколько модель объясняет вариацию $y$. Базовые: TSS = ESS + RSS (вариация $=$ объяснённая $+$ остаточная). $R^2 = 1 - RSS/TSS$ — обычный коэф. детерминации (для модели с константой); **нецентрированный** $R^2_u = 1 - RSS/\sum y_i^2$ — для моделей без константы; **скорректированный** $\bar R^2 = 1 - (RSS/(n-k))/(TSS/(n-1))$ штрафует за число регрессоров. **AIC, BIC** — информационные критерии (BIC сильнее штрафует сложность). Тесты значимости: $t$-тест для одного коэффициента, $F$-тест Вальда для нескольких, общий $F$-тест значимости регрессии; точные $t/F$-распределения в малой выборке требуют нормальности ошибок, иначе используются асимптотические приближения.

## Развёрнуто

### Разложение вариации
Модель $y = X\beta + \varepsilon$, оценка $\hat y = X\hat\beta$, остатки $e = y - \hat y$. Если в модели есть константа, верно:
$$\underbrace{\sum (y_i - \bar y)^2}_{TSS} = \underbrace{\sum(\hat y_i - \bar y)^2}_{ESS} + \underbrace{\sum e_i^2}_{RSS}.$$
TSS — общая, ESS — объяснённая (по модели), RSS — остаточная (необъяснённая).

### Обычный коэффициент детерминации
$$R^2 = \frac{ESS}{TSS} = 1 - \frac{RSS}{TSS}, \quad R^2\in[0,1].$$
Доля вариации $y$, объяснённая регрессией. **Корректен только для моделей с константой** (иначе разложение TSS = ESS + RSS не выполняется и $R^2$ может быть отрицательным).

**Свойства/ограничения**:
- $R^2$ не убывает при добавлении любого регрессора, даже бесполезного → стимул к перегрузке;
- не показывает причинности;
- при сравнении моделей с разным $y$ или разным числом наблюдений сравнивать $R^2$ нельзя.

### Нецентрированный $R^2$
Для моделей **без константы**:
$$R^2_u = 1 - \frac{RSS}{\sum y_i^2} = \frac{\sum \hat y_i^2}{\sum y_i^2}.$$
Принимает значения в $[0,1]$ при оценке МНК. Не сравним напрямую с обычным $R^2$.

### Скорректированный $\bar R^2$
$$\bar R^2 = 1 - \frac{RSS/(n-k)}{TSS/(n-1)} = 1 - (1-R^2)\cdot\frac{n-1}{n-k},$$
$n$ — наблюдения, $k$ — число параметров (включая константу). Штрафует за добавление регрессоров: растёт только если новый регрессор повышает объяснение «больше, чем шум». Может быть отрицательным.

### Информационные критерии
$$AIC = -2\ln L + 2k, \qquad BIC = -2\ln L + k\ln n,$$
$\ln L$ — лог-правдоподобие, $k$ — число оцениваемых параметров правдоподобия (в нормальной регрессии обычно включая дисперсию; если она общая для сравниваемых моделей, её вклад часто уходит в константу). Для нормальной регрессии: $-2\ln L = n\ln(RSS/n) + \mathrm{const}$. Меньше = лучше. BIC сильнее штрафует за число параметров при $n>e^2\approx 7$, поэтому склонен выбирать более простые модели и **состоятелен** при выборе истинной модели. AIC ориентирован на минимизацию KL-расхождения и эффективен для прогноза.

### Тесты значимости

**1. $t$-тест значимости одного коэффициента $\beta_j$.** $H_0: \beta_j = 0$. Статистика:
$$t = \frac{\hat\beta_j}{\mathrm{SE}(\hat\beta_j)} \sim t_{n-k}$$
при нормальности ошибок; без неё это асимптотический тест.
$|t| > t_{1-\alpha/2, n-k}$ — отвергаем $H_0$. $p$-value $< \alpha$ — коэффициент значим.

**2. $F$-тест значимости группы коэффициентов** ($q$ ограничений). Сравнение **полной** (RSS_U) и **редуцированной** (RSS_R) моделей:
$$F = \frac{(RSS_R - RSS_U)/q}{RSS_U/(n-k)} \sim F_{q,\,n-k}$$
при нормальности ошибок. Для $H_0$: заданная группа из $q$ коэффициентов (или линейных ограничений) равна нулю.

**3. Общий $F$-тест значимости регрессии**. $H_0: \beta_2 = \dots = \beta_k = 0$ (все коэффициенты, кроме константы):
$$F = \frac{ESS/(k-1)}{RSS/(n-k)} = \frac{R^2/(k-1)}{(1-R^2)/(n-k)}\sim F_{k-1, n-k}$$
при нормальности ошибок.
Если $F$ велик — модель содержательна. Этот тест эквивалентен $F$-тесту с $q=k-1$.

### Альтернативные метрики качества прогноза
$$MSE = \frac{1}{n}\sum e_i^2,\quad RMSE = \sqrt{MSE},\quad MAE = \frac{1}{n}\sum |e_i|.$$
Применяются для **out-of-sample** оценки (на тестовой выборке) — там и $R^2$ может быть отрицательным.

### Пример
$n=100$, $k=4$ (включая константу), $TSS=1000$, $RSS=250$.
$R^2 = 1 - 250/1000 = 0{,}75$.
$\bar R^2 = 1 - 0{,}25\cdot 99/96 \approx 0{,}742$.
$F = (750/3)/(250/96) = 250/2{,}604 \approx 96{,}0$ — крайне велико, регрессия высокозначима ($p\to 0$).

### Подводные камни
- Высокий $R^2$ не гарантирует адекватности модели (можно подогнать спецификацию).
- При мультиколлинеарности отдельные $t$-тесты могут не отвергать $H_0$, хотя $F$-тест значим.
- Сравнение моделей по $R^2$ корректно только при одинаковом $y$ и одинаковом $n$.
