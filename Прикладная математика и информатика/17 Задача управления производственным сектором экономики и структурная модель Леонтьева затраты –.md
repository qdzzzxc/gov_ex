# Задача управления производственным сектором экономики и структурная модель Леонтьева «затраты – выпуск».

## Билет 17: Модель Леонтьева "затраты-выпуск"

### Постановка задачи
Экономика из $n$ отраслей. Каждая производит свой продукт. Часть валового выпуска уходит на **промежуточное потребление** другими отраслями (как сырьё), часть — на **конечное потребление** $Y$ (домохозяйства, экспорт, инвестиции).

**Задача планирования:** при заданном конечном спросе $Y$ найти валовые выпуски $X = (X_1, \dots, X_n)$, обеспечивающие баланс во всех отраслях.

### Уравнение баланса
Для каждой отрасли $i$: $$X_i = \sum_j x_{ij} + Y_i,$$

где $x_{ij}$ — продукция $i$, ушедшая в отрасль $j$.

### Матрица прямых затрат
**Гипотеза линейной технологии:** $$a_{ij} = \frac{x_{ij}}{X_j}.$$

$a_{ij}$ — сколько продукции $i$ нужно для производства одной единицы продукции $j$.

Матрица $A = (a_{ij}) \ge 0$.

### Уравнение Леонтьева
$$X = AX + Y \quad \Leftrightarrow \quad (E - A) X = Y.$$

**Решение:** $$X = (E - A)^{-1} Y.$$

### Матрица полных затрат
$B = (E - A)^{-1}$ — **матрица полных (прямых и косвенных) затрат**.

$b_{ij}$ — суммарный выпуск отрасли $i$, необходимый для производства одной единицы конечного продукта отрасли $j$, с учётом всех технологических цепочек.

**Разложение в ряд:** $$(E - A)^{-1} = E + A + A^2 + A^3 + \dots$$

- $E$ — сама единица продукта.
- $A$ — прямые затраты.
- $A^2$ — затраты на затраты.
- $A^k$ — k-я цепочка.

### Продуктивность матрицы

$A$ **продуктивна**, если для любого $Y \ge 0$ существует $X \ge 0$ с $(E-A) X = Y$. Эквивалентно: $(E-A)^{-1} \ge 0$.

**Содержательно:** экономика может удовлетворить любой осмысленный конечный спрос.

### Критерии продуктивности

1. **Спектральный радиус** $\rho(A) < 1$.
2. **Критерий Хокинса-Саймона:** все главные миноры матрицы $E - A$ положительны.
3. Существует $X^0 > 0$, при котором $X^0 > A X^0$.
4. Сходимость ряда $E + A + A^2 + \dots$.

### Пример (2 отрасли)

$A = \begin{pmatrix} 0.2 & 0.3 \ 0.4 & 0.1 \end{pmatrix}$, $Y = \begin{pmatrix} 100 \ 200 \end{pmatrix}$.

$E - A = \begin{pmatrix} 0.8 & -0.3 \ -0.4 & 0.9 \end{pmatrix}$, $\det = 0.60$.

Главные миноры: $0.8 > 0$, $0.60 > 0$ → Хокинс-Саймон выполнен, продуктивна.

$(E-A)^{-1} = \frac{1}{0.60} \begin{pmatrix} 0.9 & 0.3 \ 0.4 & 0.8 \end{pmatrix} = \begin{pmatrix} 1.5 & 0.5 \ 0.67 & 1.33 \end{pmatrix}$.

$X = \begin{pmatrix} 1.5 & 0.5 \\ 0.67 & 1.33 \end{pmatrix} \begin{pmatrix} 100 \ 200 \end{pmatrix} = \begin{pmatrix} 250 \ 333.33 \end{pmatrix}$.

Для конечного спроса (100, 200) нужно произвести (250, 333) валового продукта. Разница — промежуточное потребление.

### Подводные камни
- Линейность технологии — допущение, на практике может нарушаться (масштаб, эффекты замены).
- Если $\rho(A) \ge 1$, экономика непродуктивна — задача неразрешима.
- Матрица $A$ должна быть стабильна во времени для статической модели.

### Главное на экзамене
1. **Баланс:** $X = AX + Y$ — валовый выпуск = промежуточное + конечное.
2. **Матрица прямых затрат** $A$, $a_{ij} = x_{ij}/X_j$.
3. **Решение:** $X = (E-A)^{-1} Y$.
4. **Матрица полных затрат** $(E-A)^{-1}$ — учитывает все цепочки.
5. **Разложение в ряд:** $E + A + A^2 + \dots$ — прямые + косвенные затраты.
6. **Продуктивность:** $\rho(A) < 1$ или Хокинс-Саймон (главные миноры $E-A$ положительны).
