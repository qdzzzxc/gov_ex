# Метрические пространства. Теорема Банаха о неподвижной точке сжимающего отображения.
## Метрика
Функция $d: X \times X \to \mathbb{R}_{\ge 0}$ с тремя аксиомами:
1. **Тождество:** $d(x, y) = 0 \iff x = y$.
2. **Симметрия:** $d(x, y) = d(y, x)$.
3. **Неравенство треугольника:** $d(x, z) \le d(x, y) + d(y, z)$.
Примеры в $\mathbb{R}^n$: евклидова $\sqrt{\sum (x_i - y_i)^2}$, манхэттенская $\sum |x_i - y_i|$, Чебышёва $\max_i |x_i - y_i|$.
## Сходимость
$x_n \to x \iff d(x_n, x) \to 0$. Это базовое понятие, без него не определить полноту.
## Фундаментальная (Коши) последовательность
$\forall \varepsilon > 0, \exists N: \forall m, n \ge N \quad d(x_m, x_n) < \varepsilon$.
Элементы попарно сближаются, без упоминания предела.
Все сходящиеся — фундаментальные. Не все фундаментальные — сходящиеся.
Пример: $1, 1.4, 1.41, \dots$ в $\mathbb{Q}$ — фундаментальная, но в $\mathbb{Q}$ предела ($\sqrt{2}$) нет.
## Полное пространство
Пространство где все фундаментальные последовательности сходятся в этом же пространстве. «Нет дыр».
- $\mathbb{R}^n$ — полное.
- $\mathbb{Q}$ — неполное.
- $C[a, b]$ с метрикой $\max|f - g|$ — полное.
- Замкнутое подмножество полного — полное.
## Сжимающее отображение
$T: X \to X$ — сжатие, если $\exists q \in [0, 1)$: $$d(Tx, Ty) \le q \cdot d(x, y) \quad \forall x, y \in X$$
Условие $q < 1$ **строго** — критично.
## Теорема Банаха о неподвижной точке
**Условия:** $(X, d)$ — полное метрическое пространство, $T: X \to X$ — сжатие с коэффициентом $q < 1$.
**Утверждение:**
1. У $T$ существует единственная неподвижная точка $x^*$: $Tx^* = x^*$.
2. Итерации $x_{n+1} = T x_n$ сходятся к $x^*$ из любого $x_0 \in X$.
3. Геометрическая скорость сходимости: $$d(x_n, x^*) \le \frac{q^n}{1 - q} \cdot d(x_1, x_0)$$
## Что критично в условиях
- **Полнота $X$**: иначе неподвижная точка может «выпасть» из пространства. Пример: $T(x) = x/2$ на $(0, 1]$ — итерации → 0 ∉ (0, 1].
- **$q < 1$ строго**: при $q = 1$ теорема не работает. Пример: $T(x) = x + 1$ на $\mathbb{R}$ — изометрия, неподвижной точки нет.
- **$T: X \to X$**: должно отображать в то же пространство.
## Пример
$T(x) = x/2 + 1/4$ на $[0, 1]$. $q = 1/2$. Неподвижная: $x^* = 1/2$. Итерации с $x_0 = 0$: $1/4 \to 3/8 \to 7/16 \to \dots \to 1/2$.
Типовое применение теоремы — **теорема Пикара** о существовании и единственности решения задачи Коши для ОДУ; итерации Пикара $y_{n+1} = T y_n$ это и есть итерации Банаха для конкретного оператора из ОДУ.
