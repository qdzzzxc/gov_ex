# Метрические пространства. Теорема Банаха о неподвижной точке сжимающего отображения.

### Метрика
Функция $d: X \times X \to \mathbb{R}_{\ge 0}$ с тремя аксиомами:
1. **Тождество:** $d(x, y) = 0 \iff x = y$.
2. **Симметрия:** $d(x, y) = d(y, x)$.
3. **Неравенство треугольника:** $d(x, z) \le d(x, y) + d(y, z)$.

**Примеры в $\mathbb{R}^n$:**
- Евклидова: $\sqrt{\sum (x_i - y_i)^2}$.
- Манхэттенская: $\sum |x_i - y_i|$.
- Чебышёва: $\max_i |x_i - y_i|$.
- Дискретная: $0$ если равны, $1$ иначе.

### Топологические понятия
**Открытый шар:** $B(x_0, r) = {x : d(x, x_0) < r}$.
**Открытое множество:** для каждой точки есть шар вокруг неё, целиком в множестве. Граница не входит.

**Замкнутое:** дополнение открытого; содержит все предельные точки.

**Сходимость:** $x_n \to x \iff d(x_n, x) \to 0$.

**Непрерывность:** $f(x_n) \to f(x_0)$ при $x_n \to x_0$ (эквивалентно $\varepsilon$-$\delta$).

### Фундаментальная (Коши) последовательность

$\forall \varepsilon > 0\ \exists N: \forall m, n \ge N\ d(x_m, x_n) < \varepsilon$.

**Содержательно:** элементы попарно сближаются, без упоминания предела.

- Все сходящиеся — фундаментальные.
- Не все фундаментальные — сходящиеся.

Пример: $1, 1.4, 1.41, \dots$ в $\mathbb{Q}$ — фундаментальная, но в $\mathbb{Q}$ предела ($\sqrt 2$) нет.

### Полное пространство

Где **все фундаментальные сходятся** в этом же пространстве. "Нет дыр".

- $\mathbb{R}^n$ — полное.
- $\mathbb{Q}$ — неполное.
- $C[a, b]$ с метрикой $\max|f - g|$ — полное.
- Замкнутое подмножество полного — полное.

### Компактность

$K$ секвенциально компактно — из любой последовательности выделяется сходящаяся в $K$ подпоследовательность.

В метрических пространствах: **компактность ⟺ полнота + полная ограниченность**.

### Сжимающее отображение

$T: X \to X$ — **сжатие**, если $\exists q \in [0, 1)$: $$d(Tx, Ty) \le q \cdot d(x, y) \quad \forall x, y \in X.$$

$q < 1$ **строго** — критично. Это сильнее липшицевости.

### Теорема Банаха о неподвижной точке

**Условия:** $(X, d)$ — полное метрическое пространство, $T: X \to X$ — сжатие с коэффициентом $q < 1$.

**Утверждение:**

1. У $T$ существует **единственная** неподвижная точка $x^_$: $T x^_ = x^*$.
2. Итерации $x_{n+1} = T x_n$ сходятся к $x^*$ из любого $x_0 \in X$.
3. **Геометрическая** скорость сходимости: $$d(x_n, x^*) \le \frac{q^n}{1 - q} \cdot d(x_1, x_0).$$

### Идея доказательства

- ${x_n}$ — фундаментальная (из $d(x_{n+1}, x_n) \le q^n d(x_1, x_0)$ + геометрическая прогрессия).
- В полном $X$ есть предел $x^*$.
- $T$ непрерывно (липшицево), значит $Tx^* = \lim Tx_n = \lim x_{n+1} = x^*$.
- Единственность: $d(x^_, y^_) \le q \cdot d(x^_, y^_)$ при $q < 1$ → $d = 0$.

### Что критично в условиях

- **Полнота $X$:** иначе неподвижная точка может "выпасть" из пространства. Пример: $T(x) = x/2$ на $(0, 1]$ — итерации → 0 ∉ (0, 1].
- **$q < 1$ строго:** при $q = 1$ теорема не работает. Пример: $T(x) = x + 1/x$ на $[1, \infty)$ — производная < 1, но не сжатие в смысле Банаха; нет неподвижной точки.
- **$T: X \to X$:** должно отображать в то же пространство.

### Применения

- **Теорема Пикара** — существование и единственность решения задачи Коши $y' = f(x, y)$. Сводится к $y = T(y)$ через интегрирование, $T$ — сжатие.
- **Интегральные уравнения** Фредгольма, Вольтерра.
- **Метод простых итераций** для $x = \varphi(x)$ при $|\varphi'| < 1$.
- **Численные методы** для линейных систем (Якоби, Гаусс-Зейдель).
- **Итерационные алгоритмы** в оптимизации и ML.

### Пример

$T(x) = x/2 + 1/4$ на $[0, 1]$. $q = 1/2$. Неподвижная: $x^* = 1/2$. Итерации с $x_0 = 0$: $1/4 \to 3/8 \to 7/16 \to \dots \to 1/2$.

### Главное на экзамене

1. **3 аксиомы метрики.**
2. **Полное пространство** — все фундаментальные сходятся.
3. **Сжатие** $d(Tx, Ty) \le q \cdot d(x, y)$, $q < 1$.
4. **Теорема Банаха** — 3 утверждения: существование, единственность, геометрическая сходимость.
5. **Условия критичны:** полнота, $q < 1$ строго.
6. **Главное применение** — теорема Пикара для ОДУ.
