# Градиентные методы поиска локального экстремума функции многих переменных: метод наискорейшего подъема (или спуска), метод проекции градиента, метод условного градиента.

## TL;DR
Задача $\min_{x\in\mathbb R^n} f(x)$. Антиградиент $-\nabla f(x)$ — направление наискорейшего убывания. Итерация $x_{k+1} = x_k - \alpha_k\nabla f(x_k)$. Шаг $\alpha_k$ — постоянный, дроблением (Армихо) или из условия наискорейшего спуска $\alpha_k = \arg\min_\alpha f(x_k - \alpha\nabla f(x_k))$ (тогда соседние градиенты ортогональны). Для $L$-липшицева градиента шаг $0<\alpha<2/L$ даёт убывание на гладких задачах; сходимость к оптимуму требует дополнительных условий (выпуклость/сильная выпуклость), а в невыпуклом случае обычно гарантируют лишь стационарность. На замкнутом выпуклом $X$: метод **проекции градиента** $x_{k+1} = \Pi_X(x_k - \alpha_k\nabla f(x_k))$; для компактного выпуклого $X$ — метод **условного градиента (Франк–Вульф)** через минимизацию линейной функции на $X$.

## Развёрнуто

### Постановка задачи
Ищем локальный минимум $f: \mathbb{R}^n \to \mathbb{R}$, $f \in C^1$. Необходимое условие: $\nabla f(x^*) = 0$. Если $f$ выпукла — локальный минимум совпадает с глобальным.

Градиент $\nabla f(x) = (\partial f/\partial x_1, \dots, \partial f/\partial x_n)^\top$ указывает направление наискорейшего возрастания (по производной по направлению $\partial_e f = \langle \nabla f, e \rangle$ максимум при $e = \nabla f / \|\nabla f\|$). Антиградиент — направление наискорейшего убывания.

### Общая схема градиентного спуска
$$x_{k+1} = x_k - \alpha_k \nabla f(x_k), \quad k = 0, 1, 2, \dots$$
Стоп-критерий: $\|\nabla f(x_k)\| < \varepsilon$ или $\|x_{k+1} - x_k\| < \varepsilon$.

**Варианты выбора шага $\alpha_k$:**
1. **Постоянный** $\alpha_k \equiv \alpha$. Для гладкой выпуклой задачи обычно берут $0<\alpha\le 1/L$ (или шире $0<\alpha<2/L$ для убывания на квадратичных/подходящих задачах), где $L$ — константа Липшица $\nabla f$. Просто, но требует знания $L$ и не гарантирует глобальный минимум без выпуклости.
2. **Дробный (Армихо, backtracking)**: стартуем с $\alpha = \alpha_0$, пока не выполнено условие достаточного убывания
$$f(x_k - \alpha \nabla f(x_k)) \le f(x_k) - c \alpha \|\nabla f(x_k)\|^2, \quad c \in (0, 1),$$
дробим $\alpha \leftarrow \beta \alpha$, $\beta \in (0,1)$.
3. **Наискорейший спуск (steepest descent)**: одномерная минимизация вдоль антиградиента
$$\alpha_k = \arg\min_{\alpha \ge 0} \varphi(\alpha), \quad \varphi(\alpha) = f(x_k - \alpha \nabla f(x_k)).$$
Из условия $\varphi'(\alpha_k) = 0$ получаем
$$\langle \nabla f(x_{k+1}), \nabla f(x_k) \rangle = 0,$$
то есть соседние направления ортогональны — траектория идёт «зигзагом», что замедляет сходимость на овражных функциях.

### Условия и скорость сходимости
Пусть $\nabla f$ $L$-липшицев: $\|\nabla f(x) - \nabla f(y)\| \le L \|x - y\|$.
- **Гладкая $f$, $\alpha_k = 1/L$**: $\min_{i \le k} \|\nabla f(x_i)\|^2 = O(1/k)$ — сходимость к стационарной точке.
- **Выпуклая гладкая**: $f(x_k) - f^* = O(1/k)$ (сублинейная).
- **Сильно выпуклая** (с константой $\mu > 0$, т.е. $f(y) \ge f(x) + \langle \nabla f(x), y-x\rangle + \tfrac{\mu}{2}\|y-x\|^2$): линейная сходимость
$$f(x_k) - f^* \le \left(1 - \tfrac{\mu}{L}\right)^k (f(x_0) - f^*).$$
Число обусловленности $\kappa = L/\mu$: чем больше — тем медленнее.

### Метод сопряжённых градиентов
Чтобы избежать зигзага, вместо $-\nabla f(x_k)$ берут направление
$$d_k = -\nabla f(x_k) + \beta_k d_{k-1},$$
где $\beta_k$ обеспечивает сопряжённость относительно гессиана. Для квадратичной функции $f(x) = \tfrac12 x^\top A x - b^\top x$ (с симметричной положительно определённой $A$) метод сходится не более чем за $n$ шагов. Формулы $\beta_k$ — Флетчера–Ривса: $\beta_k = \|\nabla f(x_k)\|^2 / \|\nabla f(x_{k-1})\|^2$, или Полака–Рибьера.

### Метод Ньютона (для сравнения)
$$x_{k+1} = x_k - [\nabla^2 f(x_k)]^{-1} \nabla f(x_k).$$
**Квадратичная** локальная сходимость, но требует гессиан $\nabla^2 f$ и его обращение ($O(n^3)$ за итерацию). Квази-ньютоновские методы (BFGS, L-BFGS) приближают $\nabla^2 f$ из истории градиентов.

### Условная оптимизация: $\min_{x \in X} f(x)$, $X \subset \mathbb{R}^n$ выпукло замкнуто

**Метод проекции градиента (projected gradient):**
$$x_{k+1} = \Pi_X(x_k - \alpha_k \nabla f(x_k)),$$
где $\Pi_X(y) = \arg\min_{x \in X} \|x - y\|$ — евклидова проекция. Эффективен, когда проекция вычисляется дёшево: бокс $[a,b]^n$ (покоординатное усечение), шар, симплекс, аффинное подпространство. Скорости сходимости — те же, что у обычного градиентного спуска, при выпуклой $f$.

**Метод условного градиента (Франк–Вульф):**
На каждом шаге решается *линейная* задача на $X$:
$$y_k = \arg\min_{y \in X} \langle \nabla f(x_k), y \rangle, \quad x_{k+1} = x_k + \alpha_k (y_k - x_k), \; \alpha_k \in [0,1].$$
Шаг $\alpha_k = 2/(k+2)$ или из одномерной минимизации. Не требует проекции — только линейный оракул, что выгодно на политопах, ядерной норме, симплексе. Обычно предполагают компактное выпуклое $X$ или хотя бы существование решения линейного оракула; скорость $O(1/k)$ требует выпуклой гладкой $f$ и ограниченного диаметра $X$. Итерации остаются в $X$ как выпуклые комбинации $y_k$.

### Стохастический градиентный спуск (SGD)
Если $f(x) = \tfrac1N \sum_{i=1}^N f_i(x)$ (ML, эмпирический риск), считают градиент по случайной подвыборке (mini-batch):
$$x_{k+1} = x_k - \alpha_k \nabla f_{i_k}(x_k).$$
Дёшево на итерацию, но шумно — нужен убывающий шаг $\alpha_k = O(1/k)$ или его аналоги (Adam, RMSProp). Скорость $O(1/k)$ для сильно выпуклых, $O(1/\sqrt{k})$ иначе.

### Пример
$f(x, y) = x^2 + 2y^2$, $\nabla f = (2x, 4y)^\top$. Старт $x_0 = (1, 1)$, $f(x_0) = 3$.

**Наискорейший спуск.** $\varphi(\alpha) = (1 - 2\alpha)^2 + 2(1 - 4\alpha)^2$.
$\varphi'(\alpha) = -4(1-2\alpha) - 16(1-4\alpha) = -20 + 72\alpha = 0 \Rightarrow \alpha_0 = 5/18$.
$x_1 = (1 - 2 \cdot 5/18,\; 1 - 4 \cdot 5/18) = (4/9,\; -1/9)$.
$f(x_1) = 16/81 + 2/81 = 18/81 = 2/9 \approx 0{,}222$.
Проверка ортогональности: $\nabla f(x_1) = (8/9,\; -4/9)$, $\nabla f(x_0) = (2, 4)$, скалярное произведение $= 16/9 - 16/9 = 0$ — соседние градиенты ортогональны.

На вытянутом эллипсе ($f = x^2 + 100 y^2$, $\kappa = 100$) метод заметно зигзагует — проявляется зависимость от обусловленности; сопряжённые градиенты или Ньютон справляются быстрее.

### Подводные камни
- **Овражные функции** ($\kappa \gg 1$): метод наискорейшего спуска сходится медленно и зигзагом — лучше сопряжённые градиенты, Ньютон или предобуславливание.
- **Невыпуклость**: гарантируется только сходимость к стационарной точке, не обязательно к минимуму.
- **Шаг $\alpha > 2/L$**: расходимость даже на квадратичной функции.
- **Метод проекции** бесполезен, если $\Pi_X$ не считается аналитически — тогда выручает Франк–Вульф (нужен лишь LP-оракул).
