# Градиентные методы поиска локального экстремума функции многих переменных: метод наискорейшего подъема (или спуска), метод проекции градиента, метод условного градиента.

**Задача:** $\min_x f(x)$ для $f: \mathbb{R}^n \to \mathbb{R}$.

**Градиент** $\nabla f(x)$ — направление наискорейшего возрастания. **Антиградиент** $-\nabla f$ — направление наискорейшего убывания.

**Базовая итерация:** $x_{k+1} = x_k - \alpha_k \nabla f(x_k)$. Стоп-критерий: $|\nabla f| < \varepsilon$.

### Выбор шага $\alpha_k$
- **Постоянный** $\alpha_k = \alpha$. Сходится при $\alpha < 2/L$ ($L$ — константа Липшица градиента). Нужно знать $L$.
- **Дробление (Armijo / backtracking):** стартуем с $\alpha_0$, проверяем условие достаточного убывания $f(x - \alpha \nabla f) \le f(x) - c\alpha|\nabla f|^2$, дробим $\alpha \leftarrow \beta \alpha$ при невыполнении. Не требует $L$.
- **Наискорейший спуск:** точная одномерная минимизация $\alpha_k = \arg\min_\alpha f(x_k - \alpha \nabla f(x_k))$. Соседние градиенты ортогональны → **зигзаг** на овражных функциях.

### Условия и скорости сходимости

**Липшицев градиент:** $|\nabla f(x) - \nabla f(y)| \le L|x - y|$. Эквивалентно $|\nabla^2 f| \le L$ — кривизна ограничена.

**Сильная выпуклость:** $f(y) \ge f(x) + \langle \nabla f(x), y-x \rangle + \frac{\mu}{2}|y-x|^2$. Гессиан $\succeq \mu E$.

**Число обусловленности** $\kappa = L/\mu$ — насколько вытянута функция.

|Класс|Скорость|Шагов до $\varepsilon$|
|---|---|---|
|Гладкая, невыпуклая|$\min \lVert\nabla f\rVert^2 = O(1/k)$|$O(1/\varepsilon^2)$|
|Выпуклая гладкая|$f - f^* = O(1/k)$|$O(1/\varepsilon)$|
|Сильно выпуклая|$f - f^* = O((1-1/\kappa)^k)$|$O(\kappa \log(1/\varepsilon))$|

### Метод Ньютона (для сравнения)
$x_{k+1} = x_k - [\nabla^2 f]^{-1} \nabla f$. **Квадратичная** сходимость, но требует гессиан и $O(n^3)$ за итерацию. Квази-ньютоновские (BFGS, L-BFGS) — приближают гессиан.

### Сопряжённые градиенты
Направление $d_k = -\nabla f + \beta_k d_{k-1}$. Избегают зигзага без использования гессиана. На квадратичной функции сходятся за $n$ шагов. Формулы $\beta_k$: Флетчер-Ривс, Полак-Рибьер.

### Условная оптимизация: $\min_{x \in X} f(x)$

**Проекция градиента:** $x_{k+1} = \Pi_X(x_k - \alpha_k \nabla f)$. После шага возвращаем точку обратно в $X$ через ближайшую точку. Эффективен, когда проекция дёшева: бокс (clip), шар (масштабирование), симплекс, аффинное.

**Условный градиент (Франк-Вульф):**

1. $y_k = \arg\min_{y \in X} \langle \nabla f(x_k), y \rangle$ — минимум **линейной аппроксимации** $f$ на $X$.
2. $x_{k+1} = x_k + \alpha_k (y_k - x_k)$, $\alpha_k = 2/(k+2)$.

Не нужна проекция, только LP-оракул. $x_{k+1}$ — выпуклая комбинация, автоматически в $X$. Скорость $O(1/k)$. Применим на политопах, симплексе, шаре по ядерной норме.

### Стохастический градиентный спуск (SGD)
Если $f = \frac{1}{N}\sum f_i$ (большая сумма, например, эмпирический риск в ML), считают градиент по случайной мини-батче. Дёшево на итерацию, но шумно. Скорость $O(1/k)$ для сильно выпуклых, $O(1/\sqrt{k})$ иначе. Расширения: Adam, RMSProp с адаптивным шагом.

### Подводные камни
- **Овражные функции** ($\kappa \gg 1$): наискорейший спуск зигзагует, лучше сопряжённые градиенты или Ньютон.
- **Невыпуклость:** только сходимость к стационарной точке, не глобальному минимуму.
- **$\alpha > 2/L$:** расходимость даже на квадратичной.
- **Проекция дорога** на политопах → лучше Франк-Вульф.

### Главное на экзамене
1. Базовая итерация и роль антиградиента.
2. Три способа выбора шага (постоянный / Armijo / наискорейший) + зигзаг.
3. Скорости сходимости для трёх классов функций.
4. Сравнение с Ньютоном (зачем нужны квази-ньютоновские).
5. Проекция градиента vs Франк-Вульф — когда что применимо.
