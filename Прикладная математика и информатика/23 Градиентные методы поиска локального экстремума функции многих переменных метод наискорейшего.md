# Градиентные методы поиска локального экстремума функции многих переменных: метод наискорейшего подъема (или спуска), метод проекции градиента, метод условного градиента.

**Задача:** $\min_x f(x)$ для $f: \mathbb{R}^n \to \mathbb{R}$. **Градиент** $\nabla f(x)$ — направление наискорейшего возрастания. **Антиградиент** $-\nabla f$ — направление наискорейшего убывания. **Базовая итерация:** $x_{k+1} = x_k - \alpha_k \nabla f(x_k)$. Стоп-критерий: $|\nabla f| < \varepsilon$.
## Выбор шага $\alpha_k$
- **Постоянный** $\alpha_k = \alpha$. Сходится при $\alpha < 2/L$, где $L$ — константа Липшица градиента. Нужно знать $L$.
- **Дробление (Armijo / backtracking):** стартуем с $\alpha_0$, проверяем условие достаточного убывания $f(x - \alpha \nabla f) \le f(x) - c\alpha|\nabla f|^2$, дробим $\alpha \leftarrow \beta \alpha$ при невыполнении. Не требует $L$.
- **Наискорейший спуск:** точная одномерная минимизация $\alpha_k = \arg\min_\alpha f(x_k - \alpha \nabla f(x_k))$. Соседние градиенты ортогональны → **зигзаг на овражных функциях** (большое число обусловленности $\kappa = L/\mu$).
## Сходимость
Зависит от класса функции:
- Гладкая невыпуклая: к стационарной точке, $O(1/\varepsilon^2)$ шагов.
- Выпуклая гладкая: $f - f^* = O(1/k)$.
- Сильно выпуклая: линейная скорость $O(\kappa \log(1/\varepsilon))$. Для сравнения: метод Ньютона $x_{k+1} = x_k - [\nabla^2 f]^{-1} \nabla f$ даёт квадратичную сходимость, но требует вычисления и обращения гессиана $O(n^3)$.
## Условная оптимизация: $\min_{x \in X} f(x)$
**Метод проекции градиента:** $$x_{k+1} = \Pi_X(x_k - \alpha_k \nabla f(x_k))$$ После шага антиградиента возвращаем точку обратно в $X$ через ближайшую точку множества. Эффективен, когда проекция дёшева: бокс (clip), шар (масштабирование), симплекс, аффинное подпространство.
**Метод условного градиента (Франк-Вульф):**
1. $y_k = \arg\min_{y \in X} \langle \nabla f(x_k), y \rangle$ — минимум **линейной аппроксимации** $f$ на $X$.
2. $x_{k+1} = x_k + \alpha_k (y_k - x_k)$, $\alpha_k = 2/(k+2)$. Не нужна проекция, только LP-оракул (минимизация линейной функции на $X$). $x_{k+1}$ — выпуклая комбинация точек $X$, автоматически в $X$. Скорость $O(1/k)$. Применим на политопах, симплексе, шаре по ядерной норме — там, где проекция дорогая, а LP-задача дешёвая.
