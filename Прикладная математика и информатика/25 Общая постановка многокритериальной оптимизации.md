# Общая постановка многокритериальной оптимизации. Парето-эффективная граница. Методы решения многокритериальной оптимизации.

## TL;DR
**Многокритериальная оптимизация (МКО)**: $F(x) = (f_1(x),\dots,f_m(x)) \to \min,\ x\in X$. Цели обычно противоречивы — единого «оптимума» нет. Решение $x^*$ **Парето-эффективно** (недоминируемо), если не существует $x\in X$, для которого $f_i(x)\le f_i(x^*)\ \forall i$ и хотя бы для одного — строго. Множество таких $F(x^*)$ — **Парето-фронт** (граница). Методы: свёртка критериев, $\varepsilon$-ограничений (главного критерия), идеальной точки, целевого программирования, последовательных уступок, интерактивные. Выбор зависит от роли ЛПР и наличия информации о предпочтениях.

## Развёрнуто

### Постановка
$$\min_{x\in X}\ F(x) = (f_1(x), f_2(x),\dots, f_m(x)),$$
$X\subseteq\mathbb R^n$ — допустимое множество, $f_i: X\to\mathbb R$ — критерии (целевые функции). Образ $Y = F(X)\subseteq\mathbb R^m$ — критериальное пространство.

### Доминирование и Парето-эффективность
$x'$ **доминирует** $x''$, если $f_i(x')\le f_i(x'')$ для всех $i$, и **строгое** неравенство хотя бы для одного $i$.

$x^*\in X$ — **Парето-оптимальное** (эффективное), если не существует доминирующего его $x\in X$.
$x^*\in X$ — **слабо Парето-оптимальное**, если не существует $x$ с $f_i(x) < f_i(x^*)\ \forall i$.

**Парето-фронт** $P = \{F(x^*): x^*\text{ — Парето-оптимально}\}\subset Y$ — образ Парето-эффективного множества в критериальном пространстве. Даже в выпуклых задачах сам фронт как множество точек не обязан быть выпуклым; выпуклость обычно относится к расширенному достижимому множеству $Y+\mathbb R_+^m$ и к поддержанным участкам границы. В невыпуклых задачах фронт может быть несвязным.

**Идеальная (утопическая) точка**: $f^*_i = \min_x f_i(x)$, $i=1,\dots,m$ (каждый критерий — оптимизирован отдельно). Обычно $f^*\notin Y$.
**Антиидеальная (надир, nadir)**: $f^{nad}_i = \max_{x\in P} f_i(x)$ — верхние границы критериев на Парето-фронте.

### Особенности МКО
- Однозначного «оптимума» нет; нужен выбор ЛПР (лица, принимающего решение).
- Критерии часто разноразмерны → необходима **нормировка** (масштабирование, например, к $[0,1]$).
- Некоторые критерии могут быть качественными — нужны экспертные шкалы.
- Сложность задачи возрастает с числом критериев (к 4–5 фронт сложно визуализировать).

### Методы решения

**1. Свёртка критериев (скаляризация).**
- **Линейная**: $\min \sum w_i f_i(x)$, $w_i\ge 0$, $\sum w_i = 1$; положительные веса дают Парето-эффективные поддержанные решения, нулевые веса могут дать только слабо эффективные. При стандартной выпуклости достижимого множества поддержанные точки описываются опорными гиперплоскостями. На невыпуклых/неподдержанных частях фронта линейная свёртка точки не находит.
- **Мультипликативная**: $\min \prod f_i(x)^{w_i}$ (для $f_i>0$).
- **Чебышёва**: $\min \max_i w_i(f_i(x) - f^*_i)$ — достигает любую точку Парето-фронта, в том числе невыпуклые.

**2. Метод $\varepsilon$-ограничений (главного критерия).** Один критерий выбирается главным:
$$\min f_k(x)\ \text{при}\ f_i(x)\le \varepsilon_i\ \forall i\ne k,\ x\in X.$$
Перебирая $\varepsilon_i$, строим Парето-фронт. Хорош для невыпуклых задач.

**3. Метод последовательных уступок.** Критерии упорядочены по важности. Сначала $\min f_1$, фиксируется $f_1^*$ + допустимая уступка $\Delta_1$, далее $\min f_2$ при $f_1\le f_1^*+\Delta_1$, и т.д.

**4. Метод идеальной точки.** Минимизируем расстояние до идеала:
$$\min_{x\in X}\|F(x) - f^*\|,\ \text{в норме } \ell_p,\ p\in\{1,2,\infty\}.$$
$\ell_\infty$ — взвешенный Чебышёв: достигает все Парето-точки (включая невыпуклые).

**5. Целевое программирование (Goal Programming).** ЛПР задаёт желаемые уровни $g_i$:
$$\min \sum (d^+_i + d^-_i)\ \text{при}\ f_i(x) - d^+_i + d^-_i = g_i,\ d^+, d^-\ge 0.$$
Минимизируются отклонения от целей.

**6. Интерактивные методы** (STEM, Зайонтса–Валлениуса, NIMBUS). ЛПР итеративно уточняет предпочтения после показа промежуточных решений. Меньше требований к априорной информации; используется в реальных задачах с многими критериями.

**7. Метаэвристики** для генерации Парето-фронта: NSGA-II, MOEA/D, SPEA2 — генетические алгоритмы для многокритериальных задач, особенно при невыпуклых, дискретных, чёрных ящиках.

### Пример (2 критерия)
$X = \{(x_1,x_2): 0\le x_i\le 1\}$, $f_1 = x_1 + x_2$, $f_2 = (x_1-1)^2 + (x_2-1)^2$.

$f_1$ минимально в $(0,0)$: $f_1 = 0, f_2 = 2$.
$f_2$ минимально в $(1,1)$: $f_1=2, f_2 = 0$.

Идеал $f^* = (0,0)$ — недостижим. Парето-эффективное множество в пространстве решений — диагональ $x_1 = x_2 = t$, $t\in[0,1]$; Парето-фронт в критериальном пространстве — кривая $(f_1,f_2)=(2t, 2(1-t)^2)$. Разные методы:
- Линейная свёртка $w_1 + w_2 = 1$: при $w_1=0{,}5$ оптимум — внутренняя точка фронта.
- $\varepsilon$: $\min f_2$ при $f_1\le 0{,}5$ → $t=0{,}25$.
- Чебышёва с $w=(0{,}5,0{,}5)$ — даёт точку, минимизирующую $\max(f_1, f_2)$.

### Рекомендации по выбору метода
- ЛПР задаёт веса априорно, задача выпуклая — линейная свёртка.
- Невыпуклый фронт — Чебышёв или $\varepsilon$-ограничения.
- Иерархия критериев — последовательные уступки.
- Желаемые уровни — goal programming.
- Большое число критериев / неясные предпочтения — интерактивные методы или метаэвристики (NSGA-II).

### Подводные камни
- Линейная свёртка не достигает невыпуклых частей фронта.
- Веса критериев должны быть нормированы по их масштабам — иначе результат произвольный.
- Не все Парето-эффективные решения «равноценны» — выбор требует ЛПР.
- На больших $m$ — «проклятие размерности» Парето-фронта (большая часть $X$ становится недоминируемой).
