# Общая постановка многокритериальной оптимизации. Парето-эффективная граница. Методы решения многокритериальной оптимизации.

## Постановка
$$\min_{x \in X} \quad F(x) = (f_1(x), f_2(x), \dots, f_m(x))$$ Несколько целевых функций одновременно. Цели обычно противоречивы — единого оптимума нет.
## Доминирование
$x'$ **доминирует** $x''$, если:
- $f_i(x') \le f_i(x'')$ для всех $i$;
- строгое неравенство хотя бы для одного $i$.
То есть «не хуже везде, лучше где-то».
## Парето-оптимальность
$x^*$ — **Парето-оптимально** (эффективно), если никакая другая точка не доминирует $x^*$. Смысл: чтобы улучшить один критерий, придётся ухудшить другой.
**Слабо Парето-оптимально:** не существует $x$ с $f_i(x) < f_i(x^*)$ для всех $i$ (строго лучше по всем).

## Парето-фронт
Множество всех Парето-оптимальных точек в пространстве критериев $F(X) \subseteq \mathbb{R}^m$.
Для выпуклых задач — выпуклая граница. Для невыпуклых — может быть несвязной.
## Идеальная и антиидеальная точки
**Идеальная (утопическая)** $f^*_i = \min_x f_i(x)$ — каждый критерий оптимизирован отдельно. Обычно недостижима.
**Антиидеальная (надир)** $f^{nad}_i = \max_{x \in P} f_i(x)$ — верхняя граница критериев на Парето-фронте.
## Особенности МКО

- Единого оптимума нет — нужно решение ЛПР (лица, принимающего решение).
- Критерии разноразмерны → нужна нормировка (к $[0, 1]$).

## Методы решения

**1. Линейная свёртка** $\min \sum w_i f_i(x)$, $w_i \ge 0$, $\sum w_i = 1$. Просто, быстро. Не достигает невыпуклых частей фронта — главное ограничение.
**2. Метод $\varepsilon$-ограничений (главного критерия)** $\min f_k$ при $f_i \le \varepsilon_i$, $\forall i \ne k$. Работает на невыпуклых фронтах.
**3. Метод последовательных уступок.** Иерархия критериев: оптимизируем $f_1$ → допустимая уступка $\Delta_1$ → оптимизируем $f_2$ при $f_1 \le f_1^* + \Delta_1$ → и т.д.
**4. Метод идеальной точки** $\min |F(x) - f^*|$. Норма Чебышёва ($\ell_\infty$) достигает все Парето-точки, включая невыпуклые.
**5. Целевое программирование (Goal Programming).** ЛПР задаёт желаемые уровни $g_i$. Минимизируем суммарные отклонения от целей.
Также используются **интерактивные методы** (ЛПР уточняет предпочтения в диалоге с алгоритмом) и **метаэвристики** для генерации Парето-фронта (NSGA-II, MOEA/D — генетические алгоритмы, подходят для невыпуклых и дискретных задач).
