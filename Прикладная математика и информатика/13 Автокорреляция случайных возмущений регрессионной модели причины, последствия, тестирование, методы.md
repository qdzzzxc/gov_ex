# Автокорреляция случайных возмущений регрессионной модели: причины, последствия, тестирование, методы устранения.

## TL;DR
**Автокорреляция** — $\mathrm{Cov}(\varepsilon_i,\varepsilon_j)\ne 0$ при $i\ne j$ (нарушение предпосылки 5 Гаусса–Маркова). Чаще всего AR(1): $\varepsilon_t = \rho\varepsilon_{t-1} + u_t$. **Причины**: инерция временных рядов, пропущенные регрессоры, неверная функциональная форма, шок. **Последствия**: МНК остаётся несмещённым, но **неэффективен**; стандартные ошибки **смещены** (обычно занижены при $\rho>0$), $t$ и $F$ ненадёжны. **Тесты**: Дарбин–Уотсон ($DW\approx 2(1-\hat\rho)$), Бройш–Годфри (LM), Льюнг–Бокс (для остатков). **Устранение**: GLS/FGLS, поправка Кохрейна–Оркатта, Прейса–Уинстена (Prais-Winsten), добавление лагов, HAC-стандартные ошибки (Newey–West).

## Развёрнуто

### Определение
В модели $y_t = x_t^\top\beta + \varepsilon_t$ предпосылка Гаусса–Маркова требует $\mathrm{Cov}(\varepsilon_i,\varepsilon_j)=0$. **Автокорреляция** (сериальная корреляция) — нарушение этого: $\mathrm{Cov}(\varepsilon_t,\varepsilon_{t-k}) = \gamma_k\ne 0$.

Типичная модель — AR(1):
$$\varepsilon_t = \rho\varepsilon_{t-1} + u_t,\quad u_t\sim \text{iid }(0,\sigma_u^2),\ |\rho|<1.$$
Тогда $\mathrm{Var}(\varepsilon_t) = \sigma_u^2/(1-\rho^2)$, $\mathrm{Cov}(\varepsilon_t,\varepsilon_{t-k}) = \rho^k\sigma_\varepsilon^2$.

### Причины
- **Инерция и циклы** во временных рядах (макроэкономика);
- **Пропущенные переменные**, имеющие свою временную динамику;
- **Неверная функциональная форма** (нелинейные эффекты, оставленные в остатках);
- **Лаговая структура** (если истинная модель $y_t = \alpha y_{t-1} + \dots$);
- **Сглаживание данных** (агрегирование, скользящие средние), пространственная зависимость в кросс-секциях.

### Последствия для МНК
- $\hat\beta$ остаётся несмещённой и состоятельной (если экзогенность сохраняется).
- $\hat\beta$ **неэффективна**: ковариационная матрица ошибки $\mathrm{Var}(\varepsilon)=\Omega \ne \sigma^2 I$, $\mathrm{Var}(\hat\beta) = (X^\top X)^{-1}X^\top\Omega X(X^\top X)^{-1}\ne \sigma^2(X^\top X)^{-1}$.
- Обычные стандартные ошибки **смещены** (как правило, занижены при положительной автокорреляции $\rho>0$) → завышенные $t$-статистики, неправильные доверительные интервалы и тесты.
- $R^2$ может быть искусственно высоким.

### Тестирование
**1. Тест Дарбина–Уотсона.** На AR(1) автокорреляцию.
$$DW = \frac{\sum_{t=2}^n(e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2} \approx 2(1 - \hat\rho).$$
$DW\approx 2$ — нет автокорреляции; $DW\to 0$ — сильная положительная; $DW\to 4$ — отрицательная. Сравнение с табличными $d_L, d_U$. Ограничения: только AR(1), классическая регрессия с константой, без лагов $y$.

**2. Тест Бройша–Годфри (LM-тест)**. Универсален, AR($p$):
- регрессируем $e_t$ на $X_t$ и $e_{t-1},\dots,e_{t-p}$;
- статистика $(n-p)R^2 \sim \chi^2_p$ при $H_0$: нет автокорреляции порядков $1..p$.
Подходит для моделей с лаговыми $y$.

**3. Тест Льюнга–Бокса (Q-тест).** $Q = n(n+2)\sum_{k=1}^p \hat r_k^2/(n-k)\sim\chi^2_p$, где $\hat r_k$ — выборочные автокорреляции остатков.

**4. Графически** — коррелограмма (ACF, PACF) остатков.

### Методы устранения
**1. Обобщённый МНК (Эйткена, GLS)**. Зная $\Omega$:
$$\hat\beta_{GLS} = (X^\top\Omega^{-1}X)^{-1}X^\top\Omega^{-1}Y.$$
Несмещённый и эффективный. На практике $\Omega$ оценивают (FGLS).

**2. Кохрейн–Оркатт (для AR(1))**. Итеративно: оцениваем $\hat\rho$ из остатков, затем модель в разностях $y_t - \hat\rho y_{t-1} = (1-\hat\rho)\beta_0 + \beta_1(x_t - \hat\rho x_{t-1}) + u_t$. Повторяем. Теряем первое наблюдение.

**3. Прейс–Уинстен (Prais-Winsten).** Модификация Кохрейна–Оркатта: первое наблюдение не теряется, а преобразуется как $y_1^*=\sqrt{1-\hat\rho^2}\,y_1$, $x_1^*=\sqrt{1-\hat\rho^2}\,x_1$.

**4. Добавление лагов** $y_{t-1}, x_{t-k}$ — часто автокорреляция исчезает, если включить пропущенные динамические члены.

**5. HAC-стандартные ошибки (Newey–West)**. Сохраняем МНК-оценки $\hat\beta$, но используем устойчивые к авто- и гетероскедастичности SE — корректные тесты при больших выборках без явной модели $\Omega$.

**6. Спецификация в духе ARMA** для динамики ошибок.

### Пример (схема расчётов)
Модель $y_t = \beta_0 + \beta_1 x_t + \varepsilon_t$, $T=50$. $DW = 0{,}9$ ($\Rightarrow \hat\rho\approx 0{,}55$) — сильная положительная автокорреляция. Применяем Кохрейна–Оркатта: $y_t^* = y_t - 0{,}55 y_{t-1}$, $x_t^* = x_t - 0{,}55 x_{t-1}$, новая регрессия даёт $DW^*\approx 1{,}9$ — устранили.

### Подводные камни
- DW неприменим в моделях с лаговой зависимой переменной — нужен $h$-статистика Дарбина или BG.
- При неверной спецификации (пропущенные переменные/нелинейность) «лечение» автокорреляции не решает корневой проблемы.
- Большие $\rho$ + единичный корень — нужна теория коинтеграции, а не GLS.
