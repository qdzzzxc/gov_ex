# Центральная предельная теорема. Закон больших чисел. Применение центральной предельной теоремы.

## TL;DR — ЗБЧ и ЦПТ

### Закон больших чисел (ЗБЧ)
Для i.i.d. $X_1, X_2, \dots$ с конечным $\mathbb E X_i = \mu$: $$\bar X_n = \frac{X_1 + \dots + X_n}{n} \to \mu.$$

**Слабый ЗБЧ (Хинчин):** сходимость по вероятности — $P(|\bar X_n - \mu| > \varepsilon) \to 0$.

**Усиленный ЗБЧ (Колмогоров):** сходимость почти наверное — $P(\lim \bar X_n = \mu) = 1$.

**Неравенство Чебышёва:** $P(|X - \mathbb E X| \ge \varepsilon) \le D X / \varepsilon^2$. Универсальный инструмент, через который доказывается слабый ЗБЧ.

### Центральная предельная теорема (Линдеберг-Леви)

Для i.i.d. $X_i$ с $\mathbb E X_i = \mu$ и **конечной** дисперсией $\sigma^2$:

$$Z_n = \frac{S_n - n\mu}{\sigma \sqrt n} = \frac{\bar X_n - \mu}{\sigma / \sqrt n} \xrightarrow{d} N(0, 1).$$

**Условия:**
1. Независимость.
2. Одинаковое распределение.
3. Конечная дисперсия — **главное** ограничение.
### Связь ЗБЧ и ЦПТ
- **ЗБЧ:** $\bar X_n \to \mu$ — стабилизация среднего.
- **ЦПТ:** уточняет скорость ($1/\sqrt n$) и распределение остатка (нормальное).

$$\bar X_n - \mu \sim \frac{\sigma}{\sqrt n} \cdot N(0, 1).$$
### Применения
- **Доверительные интервалы** для среднего: $\bar X \pm z \cdot \sigma/\sqrt n$.
- **Аппроксимация биномиального нормальным:** $\text{Bi}(n, p) \approx N(np, np(1-p))$ при больших $n$.
- **Z-тест, t-тест** — проверка статистических гипотез.
- **Метод Монте-Карло:** усреднение случайных реализаций для оценки интегралов; погрешность $\sim 1/\sqrt n$ независимо от размерности.
### Подводные камни
- ЦПТ требует **конечной дисперсии**. Распределения с тяжёлыми хвостами (Коши) не подчиняются — для них работают **устойчивые распределения** (Леви).
- Удивительный факт: среднее независимых Коши снова Коши — усреднение **не уменьшает разброс**.
- При малом $n$ или сильной асимметрии нормальная аппроксимация плохая.
- Условие i.i.d. можно ослабить (теоремы Ляпунова, Линдеберга для разнораспределённых случаев), но нужен контроль дисперсий.
### Главное на экзамене
1. **ЗБЧ:** $\bar X_n \to \mu$ для i.i.d. с конечным мат. ожиданием. Слабый (по вероятности), усиленный (почти наверное).
2. **ЦПТ:** нормированная сумма → $N(0, 1)$ при конечной дисперсии.
3. **Связь:** ЗБЧ — стабилизация, ЦПТ — скорость $1/\sqrt n$ и нормальный остаток.
4. **Условие конечной дисперсии** — критично. Коши не подчиняется.
5. **Применения:** доверительные интервалы, тесты гипотез, аппроксимация биномиального, Монте-Карло.
