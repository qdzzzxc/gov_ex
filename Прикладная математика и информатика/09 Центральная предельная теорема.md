# Центральная предельная теорема. Закон больших чисел. Применение центральной предельной теоремы.

## TL;DR
**Закон больших чисел (ЗБЧ)** — среднее арифметическое независимых одинаково распределённых СВ сходится (по вероятности — слабый ЗБЧ Хинчина; почти наверное — усиленный ЗБЧ Колмогорова) к мат. ожиданию. **Центральная предельная теорема (ЦПТ, Линдеберг–Леви)** уточняет распределение отклонений: для независимых одинаково распределённых $X_i$ с конечной дисперсией $\sigma^2$ нормированная сумма $(\sum X_i - n\mu)/(\sigma\sqrt n)$ сходится по распределению к $N(0,1)$. Применяется для построения доверительных интервалов, аппроксимации биномиального распределения, проверки гипотез, оценки погрешности усреднений.

## Развёрнуто

### Закон больших чисел
Пусть $X_1,X_2,\dots$ — последовательность независимых одинаково распределённых (i.i.d.) СВ с $\mathbb E X_i = \mu$, $\bar X_n = (X_1+\dots+X_n)/n$.

**Слабый ЗБЧ (Хинчин):** $\bar X_n \xrightarrow{P} \mu$, т.е. $\forall\varepsilon>0: P(|\bar X_n-\mu|>\varepsilon)\to 0$ (требует только конечного $\mu$).

**Усиленный ЗБЧ (Колмогоров):** $\bar X_n \xrightarrow{\text{п.н.}} \mu$ (почти наверное; для i.i.d. с конечным $\mu$).

**Неравенство Чебышёва** (инструмент): $P(|X-\mathbb E X|\ge \varepsilon)\le \mathrm{D}X/\varepsilon^2$. Из него — простое доказательство ЗБЧ при конечной $\sigma^2$.

### Центральная предельная теорема (Линдеберг–Леви)
$X_1,X_2,\dots$ — i.i.d., $\mathbb E X_i=\mu$, $\mathrm D X_i=\sigma^2\in(0,\infty)$. Тогда:
$$Z_n = \frac{S_n - n\mu}{\sigma\sqrt n} = \frac{\bar X_n - \mu}{\sigma/\sqrt n} \xrightarrow{d} N(0,1).$$
Эквивалентно: $\forall a<b\in\mathbb R$: $P(a\le Z_n\le b)\to \Phi(b)-\Phi(a)$, где $\Phi$ — функция распределения стандартного нормального.

**Идея доказательства** — через характеристические функции: $\phi_{Z_n}(t) = \phi_{(X_1-\mu)/(\sigma\sqrt n)}^n(t)$, разложение $\phi(t/\sqrt n)\approx 1 - t^2/(2n)$, и $(1-t^2/2n)^n\to e^{-t^2/2}$ — характеристическая функция $N(0,1)$.

### Обобщения
- **Теорема Ляпунова** — для разнораспределённых независимых при выполнении условия Ляпунова (контроль третьего момента).
- **Теорема Линдеберга** — общее условие Линдеберга, наиболее слабое для классической ЦПТ.
- **Скорость сходимости** (Берри–Эссеен): $\sup_x|P(Z_n\le x)-\Phi(x)|\le C\rho/(\sigma^3\sqrt n)$, $\rho = \mathbb E|X_1-\mu|^3$, $C\approx 0{,}47$.

### Связь с ЗБЧ
ЗБЧ говорит, что $\bar X_n\to\mu$. ЦПТ уточняет масштаб флуктуаций: $\bar X_n - \mu \sim \sigma/\sqrt n \cdot N(0,1)$ — стандартная скорость $1/\sqrt n$ и асимптотически нормальное распределение остатка.

### Применение
1. **Доверительные интервалы для среднего**: при больших $n$ —
$$\bar X \pm z_{1-\alpha/2}\frac{\sigma}{\sqrt n}.$$
Если $\sigma$ неизвестна — используют оценку $S$; точное $t$-распределение получается при нормальности генеральной совокупности, а в общем случае это асимптотическая замена по ЦПТ (при больших $n$ $t\to z$).

2. **Аппроксимация биномиального распределения**: $X\sim Bi(n,p)$, тогда $X\approx N(np, np(1-p))$ при $np(1-p)\ge 9$. С поправкой на непрерывность: $P(X\le k)\approx \Phi((k+0{,}5-np)/\sqrt{np(1-p)})$.

3. **Аппроксимация Пуассона нормальным** при $\lambda$ большом.

4. **Проверка статистических гипотез** (Z-тест, t-тест).

5. **Метод Монте-Карло**: при усреднении $n$ независимых реализаций погрешность падает как $1/\sqrt n$.

### Пример
Бросаем монету $n=100$ раз, $X$ — число орлов, $X\sim Bi(100, 0{,}5)$, $\mathbb E X=50$, $\sigma^2=25$.
$$P(X\ge 60) \approx 1 - \Phi((59{,}5-50)/5) = 1-\Phi(1{,}9)\approx 0{,}0287.$$
Точное значение $\approx 0{,}0284$ — отличное совпадение.

### Подводные камни
- ЦПТ требует **конечной** дисперсии. Распределения с тяжёлыми хвостами (Коши) не подчиняются ЦПТ — для них работают устойчивые (стабильные) законы.
- При $n$ малом или сильной асимметрии распределения нормальная аппроксимация может быть плохой.
- Условие i.i.d. можно ослабить, но нужен какой-то контроль зависимостей.
