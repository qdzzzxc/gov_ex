## Эконометрика
### Коэффициенты детерминации
$$R^2 = 1 - \frac{RSS}{TSS}$$ $$R^2_u = 1 - \frac{RSS}{\sum y_i^2} \quad (\text{без константы})$$ $$\bar R^2 = 1 - (1 - R^2) \cdot \frac{n - 1}{n - k}$$
### Информационные критерии
$$AIC = -2 \ln L + 2k, \quad BIC = -2 \ln L + k \ln n$$

### Тесты значимости
**t-тест (один коэффициент):** $$t = \frac{\hat\beta_j}{\text{SE}(\hat\beta_j)} \sim t_{n - k}$$
**F-тест (группа $q$ коэффициентов):** $$F = \frac{(RSS_R - RSS_U) / q}{RSS_U / (n - k)} \sim F_{q, n - k}$$
**Общий F-тест (регрессия в целом):** $$F = \frac{R^2 / (k - 1)}{(1 - R^2) / (n - k)} \sim F_{k - 1, n - k}$$
**Связь:** $F = t^2$ при $q = 1$.
### **Тест Дарбина-Уотсона** (для AR(1)): $$DW = \frac{\sum_{t=2}^n (e_t - e_{t-1})^2}{\sum_{t=1}^n e_t^2} \approx 2(1 - \hat\rho)$$
### AR(1)
$$\mathrm{E}[x_t] = \frac{c}{1-\varphi},\quad \mathrm{D}[x_t] = \frac{\sigma_\varepsilon^2}{1-\varphi^2},\quad \rho(1) = \varphi.$$
Двумерны случай:
$$\sigma_x^2 = \frac{\sigma_\varepsilon^2}{(1-r_i^2)(1-r_j^2)}.$$
### AR(2) с кратными корнями
$$\mathrm{E}[x_t] = \frac{c}{1-\varphi_1-\varphi_2}$$ Характеристическое уравнение $1 - \varphi_1 z - \varphi_2 z^2 = 0$ имеет двойной корень $\lambda$: $\varphi_1 = 2\lambda, \varphi_2 = -\lambda^2$. АКФ: $\rho(k) = (1+bk)\lambda^{|k|}$, монотонное затухание без колебаний.

формула теоретической дисперсии:
$$\sigma_x^2 = \frac{(1+\lambda^2)\sigma_\varepsilon^2}{(1-\lambda^2)^3}$$

## Тервер
### Математическое ожидание
$$\bar X = \frac{1}{n} \sum_{i=1}^{n} X_i$$
### Дисперсия
$$\mathrm{var}(X) = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar X)^2$$
### Ковариация
$$\mathrm{cov}(X, Y) = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar X)(Y_i - \bar Y) = E[XY] - E[X] \cdot E[Y]$$
## ML
### Линейная модель для 1 признака
$$\hat a_1 = \frac{\mathrm{cov}(X,Y)}{\mathrm{var}(X)},\quad \hat a_0 = \bar Y - \hat a_1 \bar X.$$
### Метрики
$$\mathrm{precision}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FP}_k},\quad \mathrm{recall}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FN}_k},\quad F_1 = \frac{2 P R}{P+R}.$$
### Энтропия
$$H(X) = -\sum_{i=1}^k p_i \log_2 p_i,\qquad p_i = \frac{n_i}{n}.$$
$$IG(X, A) = H(y) - \sum_{v} \frac{|X_v|}{|X|} H(y_v)$$
где:
- $H(y)$ — энтропия таргета в исходном узле;
- $X_v$ — подгруппа объектов со значением признака $A = v$;
- $H(y_v)$ — энтропия таргета внутри подгруппы;
- $|X_v|/|X|$ — доля подгруппы.
## ФинМат
### Доходность и риск портфеля
$$\mu_p = \mathbf{w}^\top \boldsymbol{\mu},\quad \mathrm{D}[\mu_p] = \sigma_p^2 = \mathbf{w}^\top \Sigma \mathbf{w}.$$
### **NPV** (Net Present Value):
$$\mathrm{NPV}(r) = \sum_{t=0}^{i} \frac{C_t}{(1+r)^t}.$$
### **IRR** — ставка, при которой $\mathrm{NPV}(r) = 0$.
### Дюрация (* - модифицированная)
$$D = \frac{\sum_k t_k \cdot PV(C_k)}{\sum_k PV(C_k)} = \frac{\sum_k t_k \cdot \frac{C_k}{(1+r)^{t_k}}}{P}$$
$$D^* = \frac{D}{1 + r}$$
Формула чувствительности: $$\frac{\Delta P}{P} \approx -D^* \Delta r + \frac{1}{2} C (\Delta r)^2$$
### Издержки
Предельные издержки — производная функции полных издержек: $MC(x) = TC'(x)$.
$$TC(x) = \int MC(x)\, dx + C$$
**Средние издержки** $AC(x) = TC(x)/x$ — издержки на единицу продукции.
TC определяются «скоростью» затрат = интеграл
### Формулы аннуитетов (постнумерандо)
**Приведённая стоимость (PV):** $$PV = C \cdot a_{n,y}, \quad a_{n,y} = \frac{1 - (1+y)^{-n}}{y}$$
**Будущая стоимость (FV):** $$FV = C \cdot s_{n,y}, \quad s_{n,y} = \frac{(1+y)^n - 1}{y}$$
**Бессрочный аннуитет** ($n \to \infty$, $y > 0$): $$PV = \frac{C}{y}$$
**Растущий бессрочный** (модель Гордона, $y > g$): $$PV = \frac{C}{y - g}$$**Пренумерандо** (платёж в начале периода): умножить любую PV-формулу на $(1+y)$.
### Иммунизация Реддингтона:
$$\begin{cases} N_A P_A + N_B P_B = PV_d, \\ N_A P_A D_A + N_B P_B D_B = PV_d \cdot D_d. \end{cases}$$
### **Модель Кокса–Росса–Рубинштейна**

$$u = e^{\sigma\sqrt{\Delta t}},\quad d = 1/u,\quad p^* = \dfrac{e^{r\Delta t} - d}{u - d},\quad \Delta t = T/N$$

Симметрия $ud = 1$ ⇒ дерево рекомбинирует, узлов $O(N^2)$. Калибровка: $\ln(S_T/S_0) \sim \mathcal{N}\left((r-\sigma^2/2)T,\ \sigma^2 T\right)$ — геометрическое броуновское движение. При $N\to\infty$ → Блэк–Шоулз.
