## Эконометрика
### AR(1)
$$\mathrm{E}[x_t] = \frac{c}{1-\varphi},\quad \mathrm{D}[x_t] = \frac{\sigma_\varepsilon^2}{1-\varphi^2},\quad \rho(1) = \varphi.$$
## AR(2) с кратными корнями
$$\mathrm{E}[x_t] = \frac{c}{1-\varphi_1-\varphi_2}$$ Характеристическое уравнение $1 - \varphi_1 z - \varphi_2 z^2 = 0$ имеет двойной корень $\lambda$: $\varphi_1 = 2\lambda, \varphi_2 = -\lambda^2$. АКФ: $\rho(k) = (1+bk)\lambda^{|k|}$, монотонное затухание без колебаний.

формула теоретической дисперсии:
$$\sigma_x^2 = \frac{(1+\lambda^2)\sigma_\varepsilon^2}{(1-\lambda^2)^3}$$

## Тервер
### Математическое ожидание
$$\bar X = \frac{1}{n} \sum_{i=1}^{n} X_i$$
### Дисперсия
$$\mathrm{var}(X) = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar X)^2$$
### Ковариация
$$\mathrm{cov}(X, Y) = \frac{1}{n-1} \sum_{i=1}^{n} (X_i - \bar X)(Y_i - \bar Y)$$
## ML
### Линейная модель для 1 признака
$$\hat a_1 = \frac{\mathrm{cov}(X,Y)}{\mathrm{var}(X)},\quad \hat a_0 = \bar Y - \hat a_1 \bar X.$$
### Метрики
$$\mathrm{precision}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FP}_k},\quad \mathrm{recall}_k = \frac{\mathrm{TP}_k}{\mathrm{TP}_k + \mathrm{FN}_k},\quad F_1 = \frac{2 P R}{P+R}.$$
### Энтропия
$$H(X) = -\sum_{i=1}^k p_i \log_2 p_i,\qquad p_i = \frac{n_i}{n}.$$
$$IG(X, A) = H(y) - \sum_{v} \frac{|X_v|}{|X|} H(y_v)$$
где:
- $H(y)$ — энтропия таргета в исходном узле;
- $X_v$ — подгруппа объектов со значением признака $A = v$;
- $H(y_v)$ — энтропия таргета внутри подгруппы;
- $|X_v|/|X|$ — доля подгруппы.

## Издержки
Предельные издержки — производная функции полных издержек: $MC(x) = TC'(x)$.
$$TC(x) = \int MC(x)\, dx + C$$
**Средние издержки** $AC(x) = TC(x)/x$ — издержки на единицу продукции.
